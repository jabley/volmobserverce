Main theme

Next, we need to define the list of selectors used in our project, and then set style properties for them.

The selectors can be divided into three groups.

  • Element selectors, with familiar names from the XDIME 2 namespaces
  • Class selectors that apply to one or more elements that are defined using 'dot' notation, for example .company_header
  • ID selectors preceded by a '#' character

Adding a selector 

As an example, we will add the body selector to our theme.

  1. Make sure the main.mthm theme policy is opened
  2. Pick the 'Default' variant from the list of variants and click on the Design tab at the bottom of the window to switch the Theme Editor to the design view
  3. Click the New button in the Selectors section
  4. In the New Style Rule window choose the 'body' element from the Element list and click the Add button
  5. The 'body' element appears in the Selector Group box
  6. Click the Finish button to add selector to the theme
  7. The 'body' element appears in the Selectors list

  8. Follow the same procedure to add all remaining selectors, identifiers and classes, using the table below as a guide.

Setting style properties 

You set properties in the Style Properties section of the editor. The properties are divided into categories, some of which you can expand if required.

  1. Click the body element and select the Font category
  2. Set the Family property to 'sans-serif' and the Size property to '8pt'

  3. Expand the Box category to set all the Box Margin values to '0' and all the Box Padding values paddings to '0'
  4. Select the Format category and set Color to '#fff'
  5. Select the Background category and set Color to '#202020'
  6. All style properties for the body element have been set
  7. Repeat these steps to assign style properties to all the remaining selectors.

Style Rules view 

The Style Rules view provides an alternative way to view all the rules for a particular theme variant. It contains a hierarchical list that displays the style rules, organized by selector sequence. Pick a selector to preview its properties.
