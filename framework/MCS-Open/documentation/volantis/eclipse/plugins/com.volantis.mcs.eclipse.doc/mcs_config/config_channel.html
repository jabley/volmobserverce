channel

Purpose

Defines a custom Java class that implements a channel adapter interface.

Contains

argument

Contained by

channels

Attributes

AttributeDescriptionTypeDefaultOptionsUse
classThe custom Java class name that implements the interfacexs:stringnone  optional 
nameThe name of the adapterxs:stringnone  required 

Example

<channels>
  <channel name="smtp"
    class="com.volantis.mps.channels.SMTPChannelAdapter">
    <argument name="host" value="SMTP-HOST"/>
    <argument name="auth" value="false"/>
    <argument name="user" value="SMTP-USERNAME"/>
    <argument name="password" value="SMTP-PASSWORD"/>
  </channel>
</channels>
