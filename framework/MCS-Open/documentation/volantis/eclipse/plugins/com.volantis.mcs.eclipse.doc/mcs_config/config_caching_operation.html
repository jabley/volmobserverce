caching-operation

Purpose

Container for pipeline cache definitions.

Contains

cache

Contained by

pipeline-configuration

Attribute

AttributeDescriptionTypeDefaultOptionsUse
expiry-modeDetermines whether dependencies will be taken into account when working out when the cache expires. A value of 'fixed-age' will not track dependencies. A value of 'auto' will turn tracking on. This global value may be overridden at pipeline:cache element level.xs:stringnone fixed-age, auto optional 

Example

<pipeline-configuration>
  <caching-operation>
    <cache
      name="xmlcache"
      strategy="least-recently-used"
      max-entries="10"/>
  </caching-operation>
  <connection timeout="-1"
    enable-caching="false"
    max-cache-entries="1000"/>
</pipeline-configuration>
