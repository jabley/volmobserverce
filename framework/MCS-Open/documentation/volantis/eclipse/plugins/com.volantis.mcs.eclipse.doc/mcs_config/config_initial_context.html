initial-context

Purpose

Defines the starting context for resolving JNDI names. The contained parameters element can be used if a particular directory service needs initialization values to be passed to it.

Contains

parameter

Contained by

jndi-configuration

Attribute

AttributeDescriptionTypeDefaultOptionsUse
nameThe name of the initial context that must be unique within the containing jndi-configuration.xs:stringnone  required 

Example

<jndi-configuration>
  <initial-context name="initialContext">
    <parameter name="name1" value="value1"/>
    <parameter name="name2" value="value2"/>
  </initial-context>
</jndi-configuration>
