Component attributes

The table shows the relationship between component attributes and the (fallback) components they can refer to. It lists component attribute names in two groups on the left; fallback attributes, then specific attributes that apply only to certain types. The components with these attributes (refer to these components) are tabulated across the top. Names are shortened for clarity.

The shaded background shows the possible fallbacks. All components can have a Text component fallback.

Rollover components may have 'Normal' and 'Over' images, but only a text fallback.

The only component type without any fallback attributes is the Script component.

In each case the attribute value must be an existing component name.
