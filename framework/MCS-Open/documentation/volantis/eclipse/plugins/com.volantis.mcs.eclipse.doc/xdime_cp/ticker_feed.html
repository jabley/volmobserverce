ticker:feed

Purpose

This element, if inserted as a child of the Carousel or Ticker Tape, will make these widgets show feed obtained by the Feed Poller from the BuzzCast Server. Clicking on an item in Carousel or Ticker Tape widget will show this item in the associated Item Display widget.

Contains

ticker:item-icon, ticker:item-plain-description, ticker:item-title

Contained by

widget:carousel, widget:ticker-tape

Attributes

AttributeDescriptionTypeDefaultOptionsUse
channelSpecifies the channel to display items from. If attribute is not specified, all items from all channels will be displayed.  optional 
item-displaySpecifies the id of the Item Display widget to display selected item. If attribute is not specified, no item will be displayed when selected.  optional 

Attribute group

Core attributes

Examples

Feed Carousel widget

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget"
  xmlns:ticker="http://www.volantis.com/xmlns/2006/09/ticker">
  <head>
    <title>ticker:feed</title>
  </head>
  <body>
    <ticker:feed-poller
      url="http://localhost:8080/buzzcast/vmds/?uid=tickeruser&amp;delivery-format=4"
      style="min-polling-interval:120; max-polling-interval:120; "/>
    <widget:carousel style="height:120px; width:300px; border: solid 2px green; ">
      <ticker:feed item-display="id"/>
    </widget:carousel>
  </body>
</html>

Feed Ticker Tape widget

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget"
  xmlns:ticker="http://www.volantis.com/xmlns/2006/09/ticker">

    <title>ticker:feed</title>
  </head>
  <body>
    <ticker:feed-poller
      url="http://localhost:8080/buzzcast/vmds/?uid=tickeruser&amp;delivery-format=4"
      style="min-polling-interval:120; max-polling-interval:120; "/>
    <widget:ticker-tape style="height:30px; width:300px; border: solid 2px green; ">
      <ticker:feed item-display="id"/>
    </widget:ticker-tape>
  </body>
</html>
