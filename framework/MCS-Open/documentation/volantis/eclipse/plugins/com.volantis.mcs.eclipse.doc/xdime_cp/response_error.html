response:error

Purpose

The error message displayed when the AJAX Table widget has no data to download.

Contained by

response:body

Attribute group

Core attributes

Example

<response:response xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs"
  xmlns:response="http://www.volantis.com/xmlns/2006/05/widget/response">
  <response:head>
    <response:link rel="mcs:theme" href="/main.mthm"/>
  </response:head>
  <response:body>
    <response:error>Information could not be retrieved.</response:error>
  </response:body>
</response:response>
