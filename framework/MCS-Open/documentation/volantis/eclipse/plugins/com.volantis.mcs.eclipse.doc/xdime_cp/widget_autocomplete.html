widget:autocomplete

Purpose

Defines an Autocompleter widget.

Contained by

xf:input

Attribute

AttributeDescriptionTypeDefaultOptionsUse
srcSpecifies the URL address of the autocompletion service responsible for returning a list of matching itemsxs:anyURI  required 

Attribute group

Core attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>widget:autocomplete</title>
    <xf:model>
      <xf:instance>
        <si:instance>
          <si:item name="f1"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <xf:group>
      <xf:input ref="f1">
        <xf:label>country</xf:label>
        <widget:autocomplete src="http://localhost:8080/mcs/projects/client-app/service/autocomplete"/>
      </xf:input>
    </xf:group>
  </body>
</html>
