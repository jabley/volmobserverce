xf:textarea

Purpose

A form control for larger amounts of text. Multi-line input, such as the body of an message, is an example.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>xf:textarea</title>
    <xf:model id="feedback">
      <xf:instance>
        <si:instance>
          <si:item name="comment"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <xf:textarea model="feedback" ref="comment" style="mcs-rows: 3; mcs-columns: 20">
      <xf:label>Feedback:</xf:label>
    </xf:textarea>
  </body>
</html>
