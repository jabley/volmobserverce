ticker:channels-count

Purpose

Returns the number of distinct channels in the set of items.

Contained by

body

Attributes

AttributeDescriptionTypeDefaultOptionsUse
followedSelects channels by the 'followed' status before counting. If it is set to 'yes', only channels with items for which a link have been followed are selected. If its value is set to 'no', only channels that include items for which no link has been followed are selected. If attribute is not specified, all channels are selected. yes, no optional 
readSelects channels by the 'read' status before counting. If the attribute value is set to 'yes', only channels with read items are selected. If its value is 'no', only channels with unread items are selected. If attribute is not specified, all read and unread channels are selected. yes, no optional 

Attribute group

Core attributes
