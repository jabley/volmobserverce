mcs:br

Purpose

Breaks the current line of text.

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs">
  <head>
    <title>mcs:br</title>
  </head>
  <body>
    <div>
      <p>
        First line.<mcs:br/>
        Second line.<mcs:br/><mcs:br/>
        Third line.
      </p>
    </div>
  </body>
</html>
