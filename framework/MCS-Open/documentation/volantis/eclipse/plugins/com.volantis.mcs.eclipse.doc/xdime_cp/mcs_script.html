mcs:script

Purpose

Refers to an external script or to a script component, or contains a script.

Contained by

head

Attributes

AttributeDescriptionTypeDefaultOptionsUse
idUnique identifier for the elementxs:ID  optional 
srcName of the script component or the location of an external script filexs:anyURI  optional 
srctypeContent type of an external script filexs:string  optional 
typeContent type of the element's contentxs:string  optional 

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs">
  <head>
    <title>mcs:script</title>
    <mcs:script type="text/javascript">alert('hello')</mcs:script>
  </head>
  <body>
    <p>Welcome</p>
  </body>
</html>
