Accessing web services

The web service driver provides access to WSDL-based web services via HTTP. You need to specify web service datasources and the driver will handle all the service requests and responses.

To illustrate the use of the driver we will create a temperature converter using the web service provided by W3 Schools http://www.w3schools.com/webservices/tempconvert.asmx?WSDL. An XSL transform will be used to convert the response returned by the service to XDIME presentation markup.
