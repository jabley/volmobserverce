Before we begin

Prerequisites

This tutorial follows on from Next Steps with MCS, and assumes that you have worked through those examples, especially those on XDIME 2. So, if you have not done so already, please read it before you continue. It reuses policies and assets located in the MCS/webapps/mcs/tutorial/ directory.

It will also be helpful if you have some knowledge or experience of XSL transformations, because we will not go into detail about their structure and use.

Note: The examples described in this tutorial can be found in the MCS/webapps/mcs/tutorial/ directory.

Required software

Before you start make sure you have MCS installed on your computer. Refer to the installation documentation.

Jive portal

By now you should be quite familiar with the Jive site. Policies are available for a PC and Smartphone versions of many site pages. However, as our focus is on integrating external content, we will work with PC layouts and themes. The emphasis here is on processing external resources. We do not discuss layouts, themes and other policies, assuming that you already understand the concepts and related tasks.
