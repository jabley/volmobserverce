XDIME page with template

We will now recreate the Jive welcome page using the XDIME template prepared in the previous module.

Applying templatesÂ 

  1. An XDIME page that uses a template starts with the html element containing the head section and the body element. The html element must contain the namespace declaration for the template prefix.
    <?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs"
  xmlns:template="http://www.volantis.com/xmlns/marlin-template">
  <head>
    <title>Welcome to the Jive Intranet</title>
    <!-- Links to the layout and theme for this page -->
    <link rel="mcs:layout" href="/jivehome.mlyt"/>
    <link rel="mcs:theme" href="/jive.mthm"/>
  </head>
  <body>
   ...
  </body>
</html>
  2. Within the body element, the template:apply element binds the page and its template together using the href attribute
    <template:apply href="templates/template.xdtpl">
  <template:binding name="title" value="Welcome to Jive Intranet"/>
  <template:binding name="content">
    <template:complexValue>
      <!-- Company announcements -->
      <div class="company_item" id="announcements">
        <h3 class="company_header">Announcements</h3>
        <dl class="sec_menu">
          <dt>
            <a href="#" class="title">Jive New Products!!</a>
          </dt>
          <dd>Check out the latest line of Jive basketball shoes set to come
            out in November.</dd>
        </dl>
      </div>
      <!-- Information from Human Resources -->
      <div class="company_item" id="human_resources">
        <h3 class="company_header">Jive Human Resources</h3>
        <dl class="sec_menu">
          <dt>
            <a href="#" class="title">Opportunities at Jive</a>
          </dt>
          <dd class="preview">Jive Sports is seeking people to fill new
            job openings.</dd>
        </dl>
      </div>
      ...
    </template:complexValue>
  </template:binding>
</template:apply>
  3. The template:binding elements specify values of two parameters defined in the template file
  4. Value of the 'title' parameter specifies the name of the page. It allows character data only, since its type attribute was set to 'simple'. See Defining a template for details.
  5. The content of the page is bound to the template using the template:complexValue element. The type of this parameter was set to 'complex', therefore it may contain character data and an XDIME markup. Refer to Defining a template for further information.
  6. The rest of the file contains the Jive welcome page specific content

Checklist

NamePurpose
a

Used to create an explicit link to another place in the same document or to another document. The current document is the source of the link and the value of the href attribute, defines the link target

body

Contains the document's content.

dd

Contains the definition of a term in a definition list.

div

A section used to add extra structure to documents. Style sheets can be used to control the presentation.

dl

A list containing individual terms and their definitions.

dt

The name of a term defined within a definition list.

h1, h2, h3, h4, h5, h6

Heading elements for sections within a document. The number associated with each element indicates its relative position in the hierarchy of headings, with 1 indicating the top level and 6 the lowest level.

head

Information such as the title and other metadata that is not document content. The contained title element is required.

html

The container for the XDIME 2 document.

link

Defines a single link. Multiple links and relationships may be used in the head section of a document.

template:apply

Applies a template to the bound parameter values. If the optional href attribute is set, the template:binding element must be used to specify a binding. If the attribute is omitted, multiple template:bindings and a nested template:definition must be used.

template:binding

Specifies a binding to a parameter value. If a value is fixed, then it should be set using the value attribute. If value is dynamic or larger than can be used as an attribute, then it can be set using a nested element appropriate to the parameter type.

template:complexValue

Specifies a complex value for a parameter. The element has an open content model.

title

The title element is used to identify the document. It can contain only text.

Core attributes

Attributes that are common to XDIME 2 elements.

Hypertext attributes

Attributes that define link targets.
