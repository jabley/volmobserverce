Selection controls

The selection controls allow the user to choose one or more options from the pre-defined list. The xf:select1 and xf:select elements provide single and multiple selection lists respectively.

xf:select1 

  1. The xf:select1 element specifies a single selection form control. The xf:item element defines a single item on the list of possible options. Each item must contain a value and an optional label. The value is the string that will be inserted into the element defined in the form model and referenced by the ref attribute. 
    <xf:select1 model="demoSelect" ref="drop_down"
  style="mcs-selection-list-style: menu">
  <xf:label>Label</xf:label>
  <xf:item>
    <xf:label>Item1</xf:label>
    <xf:value>Item1</xf:value>
  </xf:item>
  <xf:item>
    <xf:label>Item2</xf:label>
    <xf:value>Item2</xf:value>
  </xf:item>
  <xf:item>
    <xf:label>Item3</xf:label>
    <xf:value>Item3</xf:value>
  </xf:item>
</xf:select1>
  2. You use the mcs-selection-list-style theme property to control visual appearance of the selection controls. For instance, mcs-selection-list-style set to 'menu' will create a drop-down list.

  3. You may also present all the values at once as radio buttons by setting the mcs-selection-list-style property to 'controls' 
    <xf:select1 model="demoSelect" ref="select1"
  style="mcs-selection-list-style: controls;
  mcs-selection-list-option-layout: control-first">

  4. The mcs-selection-list-option-layout property controls the order of the captions and controls in the display if both occupy the same pane. The possible values are: 'caption-first' and 'control-first'.

xf:select 

  1. The xf:select element specifies a multiple selection form control. You define the list of options in the same way as in the case of xf:select1. 
    <xf:select model="demoSelect" ref="list"
  style="mcs-selection-list-style: menu;
  mcs-rows: 2">
  <xf:label>Label</xf:label>
  <xf:item>
    <xf:label>Item1</xf:label>
    <xf:value>Item1</xf:value>
  </xf:item>
  <xf:item>
    <xf:label>Item2</xf:label>
    <xf:value>Item2</xf:value>
  </xf:item>
  <xf:item>
    <xf:label>Item3</xf:label>
    <xf:value>Item3</xf:value>
  </xf:item>
</xf:select>
  2. You may specify the initial value of the xf:select control using the comma-separated list
    <xf:model id="demoSelect">
  <xf:instance>
    <si:instance>
      <si:item name="drop_down"/>
      <si:item name="select1"/>
      <si:item name="select">Item1,Item3</si:item>
      <si:item name="list"/>
    </si:instance>
  </xf:instance>
</xf:model>
  3. The mcs-selection-list-style property set to 'menu' creates a selection list. The mcs-rows property controls the size of the list.

  4. If mcs-selection-list-style is set to 'controls', then the list will be displayed as checkboxes

Complete XDIME 2 code

The following example illustrates the use of the selection elements.

  1. Create a new XDIME file called xforms_select.xdime
  2. Modify the file by including the following code 
    <?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>Selection Controls</title>
    <xf:model id="demoSelect">
      <xf:instance>
        <si:instance>
          <si:item name="drop_down"/>
          <si:item name="select1"/>
          <si:item name="select">Item1,Item3</si:item>
          <si:item name="list"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <h4>Drop Down List</h4>
    <p>
      <xf:select1 model="demoSelect" ref="drop_down"
        style="mcs-selection-list-style: menu">
        <xf:label>Label</xf:label>
        <xf:item>
          <xf:label>Item1</xf:label>
          <xf:value>Item1</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item2</xf:label>
          <xf:value>Item2</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item3</xf:label>
          <xf:value>Item3</xf:value>
        </xf:item>
      </xf:select1>
    </p>
    <h4>Radio Controls</h4>
    <p>
      <xf:select1 model="demoSelect" ref="select1"
        style="mcs-selection-list-style: controls;
        mcs-selection-list-option-layout: control-first">
        <xf:label>Label</xf:label>
        <xf:item>
          <xf:label>Item1</xf:label>
          <xf:value>Item1</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item2</xf:label>
          <xf:value>Item2</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item3</xf:label>
          <xf:value>Item3</xf:value>
        </xf:item>
      </xf:select1>
    </p>
    <h4>List</h4>
    <p>
      <xf:select model="demoSelect" ref="list"
        style="mcs-selection-list-style: menu;
        mcs-rows: 2">
        <xf:label>Label</xf:label>
        <xf:item>
          <xf:label>Item1</xf:label>
          <xf:value>Item1</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item2</xf:label>
          <xf:value>Item2</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item3</xf:label>
          <xf:value>Item3</xf:value>
        </xf:item>
      </xf:select>
    </p>
    <h4>Checkboxes</h4>
    <p>
      <xf:select model="demoSelect" ref="select"
        style="mcs-selection-list-style: controls;
        mcs-selection-list-option-layout: caption-first">
        <xf:label>Label</xf:label>
        <xf:item>
          <xf:label>Item1</xf:label>
          <xf:value>Item1</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item2</xf:label>
          <xf:value>Item2</xf:value>
        </xf:item>
        <xf:item>
          <xf:label>Item3</xf:label>
          <xf:value>Item3</xf:value>
        </xf:item>
      </xf:select>
    </p>
  </body>
</html>

Checklist

NamePurpose
si:instance

Container for data items used to supply initial values for forms controls, or to provide additional data to be submitted with the form.

si:item

Defines an item of instance data that is required if a form control needs to be initialized, or when data not associated with a form control needs to be provided during form submission.

xf:instance

Optional element that contains instance data associated with the model element. It is used to supply initial values for forms controls, or to provide additional data to be submitted with the form.

xf:label

Provides a descriptive label for forms controls.

xf:model

Represents a form definition, used as a container for elements associated with its submission.

xf:select

A form control that provides for selection of one or more choices from a list of options.

xf:select1

A form control that provides for selection of a single choice from an available list of options; for example, a group of radio buttons.

xf:value

Provides the option values associated with select controls.

Form attributes

Binding expressions provide the mapping between the data associated with a particular control and the data provided for processing when the form is submitted. The expression is the name of the parameter associated with the control's data when the form is submitted. These parameters appear on the submission URL if the form is submitted using the HTTP GET method, and are hidden if the POST method is used. The methods are specified as attributes on the xf:submission element.
