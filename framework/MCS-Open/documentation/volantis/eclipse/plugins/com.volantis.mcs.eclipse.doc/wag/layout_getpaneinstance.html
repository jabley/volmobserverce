layout:getPaneInstance

Purpose

Returns a reference to one instance of a pane inside an iterating format. The reference can be used wherever a pane name can be used.

Note: Only useful in XDIME rendering. For XDIME 2 see Container reference functions.

Signature

layout:getPaneInstance [paneStem, index]

Parameters

NameDescriptionTypeUse
paneStemThe name of the pane as it appears in the layout without any index dependent informationxs:stringrequired 
indexThe index of the instance of the pane required. There must be at least one of these values. There may be more if the reference requires it.xs:integerrequired 

Example

<pane name="{layout:getPaneInstance('title',2)}">                    
  <p>Practical Digital Image Processing</p>
</pane>
