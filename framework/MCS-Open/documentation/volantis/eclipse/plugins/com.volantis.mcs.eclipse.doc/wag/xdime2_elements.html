XDIME 2 elements

XDIME markup covers a full range of block and inline elements, many of which will be familiar from XHTML. In addition special elements provide for features such as selection, fragmentation and forms. For further guidance on the use of presentation markup you should read Web author's guide.

The XDIME content integration elements are listed in XDIME content integration elements in separate namespace groupings. You should also refer to the topics in About DCI.

For information about the structure of element references you should refer to About element references.
