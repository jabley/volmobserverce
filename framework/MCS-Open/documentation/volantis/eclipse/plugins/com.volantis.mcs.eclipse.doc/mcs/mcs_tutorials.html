Tutorials

To get you started there are several tutorials. Getting started is for complete beginners, covering how to install the MCS, set up the Eclipse development IDE, and using the MCS policy editors and XDIME 2 markup on a simple welcome page.

Next steps with MCS takes you through constructing a more complex mobile site, the Jive Sports intranet. Next, you can extend the Jive site by Adding dynamic content from HTTP resources and relational databases. In Using the widget library, you can study the declarative XML structures and create compelling interactive pages using AJAX methods. The Working with XForms tutorial illustrates the use of XForms. The XForms standard defines a device-neutral, platform-independent set of form controls allowing to separate the purpose of a form from its presentation.
