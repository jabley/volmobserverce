Installing the Eclipse plugins

MCS includes a number of Eclipse plugins for tools and documentation. These need to be copied from the installation to the Eclipse environment as follows.

To install the Eclipse plugins:

  • Copy the contents of the /MCS/eclipse/plugins directory, including any sub-directories, to the Eclipse plugins directory location
  • Also copy the contents of /MCS/eclipse/features/ to the features directory

An alternative way to install MCS plugins to Eclipse is to use Eclipse manager:

  1. Open Eclipse
  2. Go to Help | Software Updates | Find and Install to open the wizard window
  3. Select Search for new features to install
  4. Press the New Local Site button and select the /MCS/eclipse directory
  5. Press the Finish button
  6. Accept the terms in the license agreement
  7. Press Next, and then Finish to complete

Tip: Once you have installed the Eclipse plugins, you will also be able to access the documentation in the Eclipse Help system.

To uninstall the Eclipse plugins:

  1. Open Eclipse
  2. Go to Help | Software Updates | Manage Configuration to open the wizard window
  3. Select MCS Development Tools
  4. Press disable on the right subwindow to restart your Eclipse
  5. Go to Help | Software Updates | Manage Configuration
  6. Select Show Disabled Features
  7. Press Uninstall on the right subwindow
  8. All MCS files will be removed from Eclipse
