Google Docs connector elements

The Google Docs™ connector namespace prefix is gdocs: and the namespace declaration http://www.volantis.com/xmlns/2008/08/gdocs.

NamePurpose
gdocs:authenticate

Authenticates the user with Google Docs™. It requires an email address to identify the user, and the user's password.

gdocs:fetch

Retrieves the contents of a single document from Google Docs™.

gdocs:list-docs

Returns a list of the user's documents from Google Docs™.
