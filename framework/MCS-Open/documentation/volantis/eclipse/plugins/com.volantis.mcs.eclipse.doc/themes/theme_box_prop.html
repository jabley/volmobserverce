Box properties

Purpose

Box style properties that set box and table values such as display, visibility, margins and padding.

ControlDescriptionOptions
DisplayBox display valuesblock, compact, inline, list-item, marker, run-in, table, inline-table, table-caption, table-cell, table-column, table-column-group, table-footer-group, table-header-group, table-row, table-row-group 
Line HeightEither the minimal height of each inline box or the exact height of each box 
HeightContent height and height units 
Content width and width units 
VisibilityWhether the box is visible or hiddencollapse, hidden, visible 
Margin/PaddingControls for all values, or the four individual sides. All synchronizes the values.
