Theme overview attributes

Purpose

Variant selection criteria and content attributes on the Theme editor Overview page.

ControlDescriptionOptions
CriteriaType of selectionDefault, Generic, Targeted 
TargetsThe name of the targeted device or devices. Required if Criteria is 'Targeted'. Browse opens the Target Selection dialog.  
Null variantIf you set this control MCS behaves as if no policy could be found for the currently selected variants  
Import parent theme variantSpecifies whether ot not a theme variant can import its values from its parent in the device hierarchy
