Adding a theme

You can use the New Theme wizard to create theme variants. The wizard allows you to:

  • Define a theme name and its location in your project folders
  • Select a one or more devices to which the theme applies

Naming a theme

The first page allows you to locate and name the theme. Each theme is a file in your project. The focus will initially be on the Theme text box, and the policy tree view will be synchronized with your current project folder selection. After this page is complete, you can click Finish and complete the information in the editor at a later time.

  1. Choose File | New and choose the Theme wizard
  2. Enter a name in the text box
  3. Choose a folder in the policy tree view or enter the full path from the project folder in the first text box
  4. Click Next

Choosing a device category policy

On the next page you can optionally use the Device category policy control to choose a set of policy values, including custom values, that can be used to target devices.

  1. Browse the available custom device policy names to select a policy for this variant.
  2. Click Next

Selection criteria

On the next wizard page you can choose from 'Targeted', 'Default', 'Generic' and 'Null' selection values.

  1. Select a value of 'Targeted' in the Selection criteria dropdown to enable the Next button, and follow the steps in Choosing a device for information on targeting devices
  2. Click either Next or Finish

The new theme icon is displayed in the policy tree, and a Theme editor Overview page is opened and highlighted.

Defining theme attributes

In the Variant Content section, you can optionally define an external stylesheet as an alternative to specifying styles in the theme. If you enter a value here, any other settings are ignored.

You can also choose to import a parent theme variant from a device higher up the hierarchy.

  1. In the Variants list, select a variant
  2. If you want to import theme values, check the Import parent theme variant option.
Note: By default the Import parent theme variant option is unchecked.
