Separator properties

Purpose

If you specify that menus are oriented horizontally, MCS applies the separator properties between groups of menuitems.

ControlDescriptionOptions
CharactersThe literal character(s) to be used 
PositionWhere the separators are located in relation to groupsbefore, after, both 
RepeatNumber of times to repeat the separator  
ImageThe name of the image component or URI to be used as the separator. Browse opens the Image Component Selection dialog.none, [image_component] or URI 
Horizontal CharacterSpacing between groupsnone, space, non-breaking-space 
TypeWhether to use characters or an image component between groupsnone, image, characters
