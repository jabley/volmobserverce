Border properties

Purpose

Specifies the width, color, and style of the border area of a box and the spacing between adjacent table cell borders. In the case of color, style and width properties you can choose All to synchronize the values for all borders, or control the four individual borders separately.

ControlDescriptionOptions
CollapseSpecifies one of two models for setting borders on table cells. A setting of 'separate' causes the Spacing properties to be used.separate, collapse 
SpacingVertical and horizontal distances and units that separate adjacent cell borders in tables. The value applies to all cells in the table. If the Collapse property is set to 'collapse', then theSpacing values will have no effect. 
ColorColor of the border  
StyleStyle of the bordernone, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset 
WidthWidth of the border. Either a text value or a number. Entering a number enables a dropdown where you can select the unit value.thin, medium, thick
