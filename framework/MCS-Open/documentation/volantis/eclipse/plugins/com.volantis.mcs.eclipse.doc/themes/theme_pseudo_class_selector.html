Pseudo-class selectors

In the New Style Rule wizard you can click the Browse button next to the Pseudo-class control, and choose one or more selectors and add a parameter where appropriate. You can also order the selectors and remove them.

Pseudo-class  Description  Example 
first-child  The first p that is a child of div  div > p:first-child {border-width: thin;}  
nth-child  Alternate tr elements in a table  tr:nth-child (odd){color: black;}tr:nth-child (even){color: silver;}  
link  An a element that is a link  a:link {color: red;}  
visited  A defn element that is a visited link  defn:visited {color:blue;}  
active  An a element when the user holds the pointer down over it  a:active {font-weight: bold;}  
focus  An xf:input where the user is entering data  xf|input:focus {background: yellow;}  
hover  An a element when the user holds the pointer over it but does not activate the link  a:hover {color: black;}  
mcs-concealed  The concealed state is the state that an element is in when it is not displayed, i.e. has display: none  mcs-concealed:mcs-effect-style: puff;  
mcs-unfolded  Controls the unfolded state. It is set for label of submenu.  nl.special > label:mcs-unfolded {color: red}nl.special > label:mcs-unfolded::marker {content: url('unfolded.gif')}  
mcs-invalid  Specifies all style changes that are applied after validation has failed  xf|*:mcs-invalid {color:red}  
mcs-busy  Defines a busy state of the Status Update widget   ticker:update-status:mcs-busy {content: "Busy..."}  
mcs-failed  Specifies a failed state of the Status Update widget  ticker:update-status:mcs-failed {content: "The last update did not succeed: "; font: red}  
mcs-normal  The normal state of the Status Update widget  ticker:update-status:mcs-normal {content: "The last poll updated OK"}  
mcs-suspended  The suspended state of the Status Update widget   ticker:update-status:mcs-suspended {content: "Updates are suspended"}  
mcs-disabled  Allows styling on the elements in the disable state, such as widget:next, widget:play, etc. For example, widget:next is disabled if the end of the gallery was reached.  widget:play:mcs-disabled {display:none;}
