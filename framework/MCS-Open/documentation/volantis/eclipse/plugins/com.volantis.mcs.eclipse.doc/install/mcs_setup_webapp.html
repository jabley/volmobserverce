Setting up your web server

After you have installed MCS, you need to set up the MCS server, essentially the contents of the MCS/webapps/[context_root] directory, as a web application for your application server. This directory also includes an example welcome.jsp application for testing and verification purposes.

The web application servers which are compatible with MCS are listed in Supported platforms in MCS Release notes (PDF). The related topics describe examples for specific web application servers.

Note: You should also refer to the documentation for your application server.
