Installing on JBoss

You need to create a .war file to contain the contents of the [path]/MCS/webapps/[context_root] directory, and copy this file to $JBoss/server/default/deploy.

  1. Create a .war file in the [path]/MCS/webapps/ directory 
    jar -cvf [context_root].war *
  2. Copy this newly created file to the $JBoss/server/default/deploy directory
  3. Start the server

Make sure that your jboss-service.xml file contains the following settings. That is to ensure the environment is compatible with the Java Servlet specification.

<attribute name="Java2ClassLoadingCompliance">false</attribute>
<attribute name="UseJBossWebLoader">false</attribute>
