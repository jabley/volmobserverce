Dissecting pane attributes

Purpose

Attributes of dissecting pane formats.

ControlDescriptionOptions
Maximum content sizeSets the size in bytes for shards in HTML text content. Will be overridden by the device maxhtmlpage value if it is smaller.  
Shard link orderThe order of shard linksNext Previous, Previous Next 
Next/Previous Link TextLiteral text values for the links 
Next/Previous Link StyleclassClass names to be used for next and previous links. The name must be from the theme that is bound to the same canvas as the layout. 
Next/Previous Link ShortcutShortcuts for the next and previous links
