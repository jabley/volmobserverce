Wrapping a format

When you first create a layout variant you can begin your design by adding grids to build a structure.

However, you often find that you need to add additional structure after you have added formats. You can do this by wrapping a format in a grid. You can also wrap formats in iterators, fragments and forms.

To wrap a format:

  1. In the Outline view select the format you want to wrap
  2. Right-click and choose Wrap followed by the format you want to use

The wrapped format is placed inside the new one, or in the first cell in an N by M grid,
