Fragment attributes

Purpose

Attributes of fragment formats.

ControlDescriptionOptions
Generate peer fragment linksWhether MCS should generate peer links 
Link OrderOrder of the linksParent Peers, Peers Parent 
Link StyleclassThe name of a style class. The class must be from the theme that is bound to the same canvas as the layout. 
Next/Previous Link TextLink text to be used. A literal value or the name of a text component. If text component is selected, Browse is enabled and displays the Text Component Selection dialog.
