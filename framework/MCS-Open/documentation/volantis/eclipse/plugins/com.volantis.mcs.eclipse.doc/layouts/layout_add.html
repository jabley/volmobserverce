Adding a layout

You can use the New Policy wizard to create layout variants. The wizard allows you to define:

  • A layout name and its location in your project folders
  • Choose between canvas and montage formats, and set selection criteria
  • Select a one or more devices to which the layout applies

Then in the editor you can select name of a device category policy.

Naming a layout

The first page allows you to locate and name the layout. Each layout is a file in your project. The focus will initially be on the Layout text box, and the policy tree view will be synchronized with your current project folder selection. After this page is complete, you can click Finish and complete the information in the editor at a later time.

  1. Choose File | New and choose the Layout wizard
  2. Enter the layout name in the text box
  3. Choose a folder in the policy tree view, or enter the full path from the project folder in the first text box
  4. Click Next

Selection criteria

On the next wizard page you can choose between 'Default' and 'Targeted' selection values, and between the canvas and montage variant types.

  1. Select a value of 'Targeted' in the Selection criteria dropdown to enable the Next button, and follow the steps in Choosing a device for information on targeting devices
  2. Select a Variant type and click either Next or Finish

The new layout icon is displayed in the policy tree, and the Layout editor Overview tab is displayed.

Choosing a device category policy

In the editor you can optionally use the Device category policy control to choose a set of existing or custom policy values to use in targeting devices.

Content attributes

There are two optional values to set in the Variant Content section, either Default fragment in the case of a canvas, or Default segment in the case of a montage.
