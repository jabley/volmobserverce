device-name-by-tacfac

Purpose

Identifies the device from the eight digit combination of Type Allocation and Final Assembly codes (TAC FAC).

If the number is missing or invalid, '404 Not Found' is returned.

Method

GET

Signature

/device/name-by-tacfac/[number]

Parameter

NameDescriptionType
number

The TAC FAC number

xs:string

Examples

Input

/device/name-by-tacfac/35209941

Returns

<device-name
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">Alcatel-OT-332</device-name>
