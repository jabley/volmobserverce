policy-info

Purpose

Returns a policy name and description in a natural language. A single language may be specified in the Accept-Language header.

If the policy name is missing or invalid, '404 Not Found' is returned. If the repository contains no policies, a document with an empty policy-info element is returned.

Note: If the Accept-Language header is not present, the default locale of the web service will be used. If the device repository does not contain category names in the requested language, the repository default language will be used. The language attribute returned contains the required language and may differ from the response.

Method

GET

Signature

/policy/info/[policy-name]

Parameter

NameDescriptionType
policy-name

The category name

xs:string

Examples

Input with Accept-Language header 'de'

/policy/info/partno

Returns

<policy-info
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws"
   language="de">
   <descriptive-name>Teilenummer</descriptive-name>
   <description>Die vom Hersteller festgelegte Teilenummer, die dieser
    Einheit zugeordnet ist, falls bekannt.</description>
</policy-info>
