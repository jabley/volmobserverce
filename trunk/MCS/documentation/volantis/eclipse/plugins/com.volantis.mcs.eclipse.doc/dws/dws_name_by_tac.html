device-name-by-tac

Purpose

Identifies the device from the six digit Type Allocation Code (TAC).

If the number is missing or invalid, '404 Not Found' is returned.

Method

GET

Signature

/device/name-by-tac/[number]

Parameter

NameDescriptionType
number

The TAC number

xs:string

Examples

Input

/device/name-by-tac/352099

Returns

<device-name
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">Alcatel-OT-332</device-name>
