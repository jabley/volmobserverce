policy-value

Purpose

Returns a specified policy value for a device.

If the device name is missing or invalid, '404 Not Found' is returned.

Method

GET

Signature

/device/policy/value/[device-name]/[policy-name]

Parameters

NameDescriptionType
device_name

The device name

xs:string
policy_name

The name of the policy

xs:string

Examples

Input

/device/policy/value/Nokia-9110i/UAProf.InputCharSet

Returns

<policy-values
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws" device-name="Nokia-9110i">   
   <policy policy-name="UAProf.InputCharSet">
      <set>
         <string>utf8</string>
      </set>
   </policy> 
</policy-values>
