device-name-by-ua-profile-url

Purpose

Identifies the device from the UAProf URL in the body of the message.

If the URL is missing or invalid, '404 Not Found' is returned.

Method

POST

Signature

/device/name-by-ua-profile-url

Examples

Input

/device/name-by-ua-profile-url/http://nds.nokia.com/uaprof/N7250r100.xml

Returns

<device-name
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">Nokia 7250</device-name>
