device-name-by-headers

Purpose

Identifies the device from the headers.

Because there is a number of different headers, the calling application should send them all in the body of the message, after removing any that should not be passed on to the web service.

If the body of the message is empty, 'PC' is returned as device name.

Method

POST

Signature

/device/name-by-headers

Examples

Input

/device/name-by-headers/User-Agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1)

Returns

<device-name
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">PC</device-name>
