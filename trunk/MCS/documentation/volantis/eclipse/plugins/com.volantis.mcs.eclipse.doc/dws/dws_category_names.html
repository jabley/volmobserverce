category-names

Purpose

Lists the names of the device policy categories.

If the repository contains no categories, a document with an empty category-names element is returned.

Method

GET

Signature

/category/names

Examples

Input

/category/names

Returns

<category-names
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">
   <category-name>protocol</category-name>
   <category-name>dynvis</category-name>
   <category-name>input</category-name>
   <category-name>output</category-name>
   <category-name>rules</category-name>
   <category-name>message</category-name>
</category-names>
