category-info

Purpose

Returns the category name in a natural language. A single language may be specified in the Accept-Language header.

If the category name is missing or invalid, '404 Not Found' is returned. If the repository contains no categories, a document with an empty category-info element is returned.

Note: If the Accept-Language header is not present, the default locale of the web service will be used. If the device repository does not contain category names in the requested language, the repository default language will be used. The language attribute returned contains the required language and may differ from the response.

Method

GET

Signature

/category/info/[category-name]

Parameter

NameDescriptionType
category-name

The category name

xs:string

Examples

Input with Accept-Language header 'de'

/category/info/audio

Returns

<category-info
    xmlns="http://www.volantis.com/xmlns/2006/01/device-ws"
    language="de">
    <descriptive-name>Audiodaten</descriptive-name>
</category-info>
