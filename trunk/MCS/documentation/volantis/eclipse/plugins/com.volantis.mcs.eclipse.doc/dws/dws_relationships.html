device-relationships

Purpose

Returns the relationships between a named device and its immediate surrounding devices in the standard device hierarchy.

The returned relationships element contains a fallback with at most a single device-name element. The children may contain zero or more device-name elements. The device-name attribute on the relationships element contains the device-name parameter value.

If there are no fallback or child devices, the associated elements are empty.

Method

GET

Signature

/device/relationships/[device-name]

Parameter

NameDescriptionType
device-name

The name of the device

xs:string

Examples

Input

/device/relationships/Nokia-9110i

Returns

<relationships
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws"
   device-name="Nokia-9110i">
   <fallback>
      <device-name>Nokia-Communicator</device-name>
   </fallback>
   <children>
      <device-name>Nokia-9110i-WAP</device-name>
      <device-name>Nokia-9110i-WWW</device-name>
   </children>
</relationships>
