Temporal attributes

Attribute Description
Cells Whether the iterator has a fixed or variable number of cells. If variable, MCS uses the number of cells in the layout; otherwise MCS uses the value in Number.
Number If Cells is fixed, MCS ignores cells in the layout over this number
Time Values Length of time for the display of each iterator cell. Browse opens the Time Selection dialog. Values less than one minute are shown in milliseconds.
