Adding a layout

You can use the New Layout wizard to create device layouts. The wizard allows you to define:

  • A layout name and its location in your project folders
  • Select a one or more devices to which the layout applies

Naming a layout

The first page allows you to locate and name the layout. Each layout is a file in your project. The focus will initially be on the Layout text box, and the policy tree view will be synchronized with your current project folder selection.

  1. Choose the Layout wizard
  2. Click Next
  3. Enter the layout name in the text box
  4. Choose a folder in the policy tree view, or enter the full path from the project folder in the first text box
  5. Click Next and choose a device

The new layout icon is displayed in the policy tree, and a Layout editor tab is opened and highlighted.

Defining layout attributes

The editor has two pages, Overview and Design. To change pages, click one of the tabs at the bottom left of the editor.

On the Overview tab you'll see a list of the devices you selected for this layout. You can add new devices or remove them at any time using the Browse or Delete buttons.

  1. In the Device Layouts list, select the device name you want to edit
  2. Enter the attribute values
  3. Go to the Design tab to begin your design
