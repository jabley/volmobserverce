Temporal iterator attributes

Purpose

Temporal attributes specify the time intervals between the rendering of different parts of the content. You can also set the maximum number of cells to be displayed.

ControlDescriptionOptions
Cells

Whether the iterator has a fixed or variable number of cells. If variable, MCS uses the number of cells in the layout; otherwise MCS uses the value in Number.

Fixed, Variable 
Number

If Cells is fixed, MCS ignores cells in the layout over this number

 
Time Values

Length of time for the display of each iterator. Browse opens the Time Selection dialog. Values less than one minute are shown in milliseconds.
