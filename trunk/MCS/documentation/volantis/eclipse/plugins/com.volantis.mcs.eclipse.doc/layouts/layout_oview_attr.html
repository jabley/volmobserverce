Layout overview attributes

Purpose

Selection criteria and content attributes on the Layout editor Overview page.

ControlDescriptionOptions
Criteria

Type of selection

Default, Generic, Targeted 
Targets

The name of the targeted device or devices. Required if Criteria is 'Targeted'. Browse opens the Target Selection dialog.

  
Null variant

If you set this control MCS behaves as if no policy could be found for the currently selected variants

  
Default Fragment

Identifies the default fragment for a canvas layout

  
Default Segment

Identifies the default segment for a montage layout
