Styling a format

You can associate grid, spatial iterator, and fragment formats with a style class you have defined in the theme editor. For example, this makes it possible to control the color and border size of a grid. You can apply classes to a grid or an iterator as a whole, or to rows or columns in the grid or the generated iterator layout. In the case of fragments, you can style the link text that chains the fragments together.

The class name you use must be from the theme that is bound to the same canvas as the layout.

To style a format:

  1. Open or select the Format Attribute view
  2. Select one or more formats in the Outline view or the Design page
  3. In the Styleclass control either enter the name the class
