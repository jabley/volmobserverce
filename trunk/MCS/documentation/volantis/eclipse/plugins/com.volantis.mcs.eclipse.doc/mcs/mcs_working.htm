Working with MCS policies

In MCS you can define a series of policies to describe the relationship of all the resources on your web site.

To develop a policy in MCS you have to complete the following tasks:

  1. Create a workbench project using the New Project wizard
  2. Add a components and associate them with assets
  3. Use devices policies in the built-in repository or define custom device attributes.
  4. Design device layouts
  5. Add themes for the devices
