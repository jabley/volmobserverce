MCS views

MCS provides the following views:

  • Format Attributes
  • Device Repository

In additon MCS makes use of three workbench views:

  • Navigator
  • Outline
  • Tasks

You use the Navigator view to manage projects, folders and project resources. The Outline and Format Attributes views work in conjunction with the MCS Layout editor. The Device Repository view displays device policy attributes for the device repository associated with a project, and you can use the Outline view with a repository tree. MCS uses the Task view to report on the errors status of all the resources in your MCS project.
