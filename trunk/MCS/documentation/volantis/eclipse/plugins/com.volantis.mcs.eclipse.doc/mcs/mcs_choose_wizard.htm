Choosing a wizard

In MCS you can choose a wizard to create a new project or add a new resource an existing project.

  1. Choose Window | Perspective | Resource
  2. Choose File | New | Other
  3. Select MCS in the wizard tree and then choose the wizard you need

Note: Although you can continue with the wizard when the Next. button is enabled, you may want to stop between pages. To do so you can click Finish when the button is enabled and edit other values later.
