Using the MCS Source editor

If there are problems with opening an MCS resource file with a policy editor, it may be because the underlying XML file has been made invalid for some reason. You can use the MCS Source editor to repair the file.

To open the editor:

  1. Select the resource you want to edit in the policy tree
  2. Right-click and choose Open With | MCS Source Editor
  3. Click any error icon in the left margin to highlight the related error detail in the Task list
  4. Fix the error, and close the editor
  5. Open the resource file again by choosing Open With and selecting the appropriate policy editor
