Asset group attributes

Purpose

You can use asset groups as a convenient way of labelling a collection of variants. The asset group name of a variant effectively specifies the directory in which the variant can be found.

ControlDescriptionOptions
Base location

Specifies whether the variants in the group are located on a server, the device or the default (server)

Server, Device, Default 
Base URL

URL to use as a prefix for the variant URLs in this group. The prefix http: on the URL identifies it to MCS as a remote location.
