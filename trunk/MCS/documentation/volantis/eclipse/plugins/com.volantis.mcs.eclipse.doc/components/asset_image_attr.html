Image attributes

Purpose

Properties of image variants in the Variant Selection and Variant Metadata sections of the component editor.

ControlDescriptionOptions
Width Hint

Percentage of the pane to be used by the image. Required if Criteria is 'Generic'; otherwise optional.

  
Encoding

Encoding of the variant

BMP, GIF, JPEG, PJPEG, PNG 
Pixel Depth

Color depth in bits of an image pixel

  
Rendering

Rendering for an image

Color, Monochrome 
Width

Width in pixels of the image to be displayed

  
Height

Height in pixels of the image to be displayed

  
Conversion mode

Whether or not the image should be passed to the Image Conversion Service (ICS) for transformation. If you select 'Always convert', the image will be converted and you can set cropping values. 'Never convert'(default) disables the crop controls.

Always convert, Never convert 
May crop before

Inclusive left pixel value to include when cropping a convertible image

  
May crop after

Inclusive right pixel value to include when cropping a convertible image
