Working with components and assets

In your project you can use component policies to associate assets with components. However, not all components can be associated with assets. For example, you cannot add assets to either button or rollover images.

To develop a component policy in MCS you have to complete the following tasks:

  1. Add a component policy to your project using the New Component wizard
  2. Using a component editor, choose an asset or assets to associate with your component
  3. Add an asset group to locate the assets on a server or device
