Adding an asset group

In MCS, you can use the Asset Group wizard to define the location of sets of assets. The wizard allows you to define:

  • An asset group name and its location in your project folders
  • Attributes or references to asset files

Naming an asset group

On the first page you can locate and name the asset group. Each group defines a location on your web server or on a device. The focus will initially be on the Asset Group text box, and the policy tree view will be synchronized with your current project folder selection.

  1. Choose the New Asset Group wizard
  2. Click Next
  3. Enter the group name in the text box
  4. Choose a folder in the policy tree view or enter the full path from the project folder in the first text box
  5. Click Next

Defining group attributes

On the next page you can enter a location for the asset group and a prefix URL to use as a prefix for the asset names in this group.

  1. Choose either Server or Device from the Location drop-down.
  2. Enter a Prefix URL for the group.
  3. Click Finish to close the wizard.
