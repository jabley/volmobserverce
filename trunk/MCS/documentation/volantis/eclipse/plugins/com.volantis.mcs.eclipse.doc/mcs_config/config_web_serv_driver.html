web-services-driver

Purpose

Contains web service definitions for the pipeline web services driver.

Contains

wsdl-catalog

Contained by

pipeline-configuration

Example

<web-services-driver>
   <wsdl-catalog>
      <wsdl-entry uri="uri/for/wsdl/entry">
         <class-resource path="/class/resource/path"/>
         <servlet-resource path="/servlet/resource/path"/>
         <uri-resource uri="uri/resource"/>
      </wsdl-entry>
   </wsdl-catalog>
</web-services-driver>
