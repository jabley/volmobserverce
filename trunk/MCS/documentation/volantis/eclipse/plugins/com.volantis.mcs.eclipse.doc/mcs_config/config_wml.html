wml

Purpose

Defines WML configuration in cases where the XDIME nativemarkup element is being rendered for the WML protocol.

Contained by

protocols

Attribute

AttributeDescriptionTypeDefaultOptionsUse
preferred-output-format

WML protocols will output WMLC if the accept header indicates that it is supported, otherwise they will output WML. WML protocols will only output WML whatever the value of the accept header.

xs:NCNamenone WML, WMLC optional 

Example

<protocols>
   <wml preferred-output-format="wmlc"/>
</protocols>
