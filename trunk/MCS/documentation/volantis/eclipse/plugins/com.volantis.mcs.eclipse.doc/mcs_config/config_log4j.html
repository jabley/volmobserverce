log4j

Purpose

Contains a reference to the file MCS uses for logging configuration.

Note: From MCS 3.3, this element is obsolete. It formerly contained a reference to the MCS log file mcs-log4j.xml. This element will be removed from the MCS configuration element reference in a later release. If the mcs-log4j.xml location is specified here, a warning message will be logged and the location will not be used. See Configuring log files for details of how to set the location.

Contained by

mcs-config

Attribute

Name Description Type Default Options Use
xml-configuration-file The path to the file containing the logging configuration xs:string none optional
