webd:header

Purpose

Defines HTTP header elements that enable headers to be created or copied. Unlike parameters, there is no difference in how they are handled by the two different types of request. There are also element forms for the attribute values.

Contained by

webd:headers

Attributes

AttributeDescriptionTypeDefaultOptionsUse
fromName of header to be copied from.xs:stringnone  optional 
nameName, or new name, of the header.xs:stringnone  optional 
valueValue of the header.xs:stringnone  optional 

Example

The example shows the element forms for the attributes.

<webd:headers>
    <webd:header>
      <webd:name>User-Agent</name>
      <webd:value>Nokia5210/1.0</value>
    </webd:header>
</webd:headers>
