template:complexValue

Purpose

Specifies a complex value for a parameter. The element has an open content model.

Contained by

template:binding, template:parameter

Attribute

AttributeDescriptionTypeDefaultOptionsUse
evaluationModeThe method of evaluating the value.xs:NCNameimmediate immediate, deferred, repeated optional 

Example

<template:complexValue>
   <pipeline:transform href="title.xsl" compilable="true">
      <urid:fetch href="story_not_found.xml')}" />
   </pipeline:transform>	
</template:complexValue>
