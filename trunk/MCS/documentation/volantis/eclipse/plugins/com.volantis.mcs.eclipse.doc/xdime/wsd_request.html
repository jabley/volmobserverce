wsd:request

Purpose

Container for a web service request.

Contains

wsd:message, wsd:wsdl-operation

Example

<wsd:request>
   <wsd:wsdl-operation
      wsdl="http://localhost:8080/pipeline/test.wsdl"
      portType="FindIPSoap"
      operation="GetURLIP"/>
   <wsd:message>
      <!-- message here-->
   </wsd:message>
</wsd:request>
