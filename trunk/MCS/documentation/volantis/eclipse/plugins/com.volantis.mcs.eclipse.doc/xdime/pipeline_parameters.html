pipeline:parameters

Purpose

A container for a set of parameters passed to an XSL transform.

Contains

pipeline:parameter

Example

<pipeline:transform compilable="false">
   <pipeline:parameters>
      <pipeline:parameter name="title-param" value="Title"/>
      <pipeline:parameter name="artist-param" value="Artist"/>
   <pipeline:parameters>
   <pipeline:transformation href='simple-params.xsl'/>
      ...
   <pipeline:transformation>
</pipeline:transform>
