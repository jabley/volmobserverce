montage

Purpose

Defines a montage, used to collect canvases into a set of segments for display on a device.

Contained by

canvas, segment

Attribute groups

Page attributes, Core attributes

Example

<montage
   layoutName="/MontageNew.mlyt"
   pageTitle="Holiday"
   cacheScope="all"
   maxAge="180">
   <segment name="bar" src="MontageBanner.xml"/>
   <segment name="menu" src="MontageTravelMenu.xml"/>
   <segment name="content" src="MontageDestinations.xml"/>
</montage>
