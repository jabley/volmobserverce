dissectingpane

Purpose

Used to set dissecting pane link text values from a page.

If the fragment in the name is not in the layout, MCS ignores the element.

Contained by

layout

Attribute group

Fragment attributes

Example

<layout>
   <dissectingpane name="frag_basket4" 
      linkText="Go to checkout" 
      backLinkText="Add to basket"/>
      ...
</layout>
