wsd:message

Purpose

Container for the outgoing message. This element has an open content model.

Contained by

wsd:request

Example

<wsd:request>
   <wsd:wsdl-operation
      wsdl="http://localhost:8080/pipeline/test.wsdl"
      portType="FindIPSoap"
      operation="GetURLIP"/>
   <wsd:message>
      <!-- message here-->
   </wsd:message>
</wsd:request>
