ssiinclude

Purpose

Defines additional material to be included by the web server before returning the response to the client browser. One of the two attributes file and virtual may be used to specify the content.

Contained by

wsdirectives

Attributes

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 
fileA URL on the web serverxs:stringnone  optional 
virtualA relative file path on the web serverxs:stringnone  optional 

Example

<wsdirectives>
   <ssiconfig errmsg="The file 'Q4report.html'is not available."/>
   <ssiinclude virtual="/mycorp/sales 2004/Q4report.html"/>
</wsdirectives>
