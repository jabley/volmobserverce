menuitem

Purpose

Specifies one or more menu items in a menu.

Note: You should use CSS color properties and the 'hover' pseudo-class in new markup to replace the deprecated onColor and offColor attributes.
Note: If you don't specify the pane attribute (see Block attributes) explicitly, the output will appear in the pane associated with the containing menu. This is true unless the target for the menu is a spatial format iterator, in which case MCS will determine the instance automatically.

Attribute groups

Block attributes, Event attributes, Focus attributes, Core attributes

Example

<menu pane="menupane" title="Prizes">
   <label>Prizes</label>
   <menuitemgroup>
      <menuitem text="1st Prize - New BMW"
         href="firstprize.xml" offImage="bullet.mimg" onImage="bmw.mimg"/>
   </menuitemgroup>
   <menuitemgroup>        
      <menuitem text="2nd Prize - Mediterranean Cruise" href="secondprize.xml"
         offImage="bullet.mimg" onImage="cruiseship.mimg"/>
      <menuitem text="3nd Prize - Two Rugby World Cup tickets" href="thirdprize.xml"
         offImage="bullet.mimg" onImage="scrum.mimg"/>
   </menuitemgroup>
</menu>
