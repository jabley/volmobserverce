pipeline:convertAbsoluteToRelativeURL

Purpose

Removes base URL prefixes or translates URLs in content.

Attributes

AttributeDescriptionTypeDefaultOptionsUse
baseURLThe remote server prefix to match. If used by itself, MCS strips off the prefix in all content instances.xs:anyURInone  optional 
substitutionPathMust be used with the baseURL attribute. The remote server prefix to replace the value in baseURL.xs:anyURInone  optional 

Example

<pipeline:convertAbsoluteToRelativeURL
   baseURL="http://www.mysite.com/"
   substitutionPath="/examples/xml/"/>
   <urid:fetch href="http://www.mysite.com/index.html" parse="xml" />
</pipeline:convertAbsoluteToRelativeURL>
