wsdirectives

Purpose

Container for web server directives.

Contains

ssiconfig, ssiinclude

Attribute

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 

Example

<wsdirectives>
   <ssiconfig errmsg="The file 'Q4report.html'is not available."/>
   <ssiinclude virtual="/mycorp/sales 2004/Q4report.html"/>
</wsdirectives>
