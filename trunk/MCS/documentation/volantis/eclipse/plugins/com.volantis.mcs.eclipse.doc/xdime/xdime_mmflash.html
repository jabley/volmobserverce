mmflash

Purpose

Defines a Macromedia Flash dynamic visual.

Contained by

canvas

Attributes

Name Description Type Default Options Use
altImg The image to be displayed if the device does not play dynamic visuals xs:string none optional
altText Text to be used if the dynamic visual cannot be displayed xs:string none optional
assetURLsuffix A suffix to add parameters to the URL, for example a query string containing user information. MCS adds the specified suffix to the final asset URL without attempting to interpret it. xs:string none optional
expr An MCS expression xs:string none optional
loop Whether to loop the animation or not xs:boolean true optional
menu Whether to display a full or limited menu. xs:boolean none required
name The component name xs:string none optional
play Whether to play the animation immediately xs:boolean true optional
swLiveConnect Whether or not to Start Java when loading Flash for first time xs:boolean false optional
tabindex Defines the order of moving focus from element to element using the TAB key or equivalent. xs:int none optional
url URL to locate a file (as distinct from a component name) to be used xs:string none optional

Attribute groups

Block attributes, Event attributes, Core attributes

Example

<mmflash
  class="flash_movies"
  id="movie1"
  name="snakemovie"
  play="true"
  loop="true"
  menu="false"
  altImg="snakepic"
  altText="The Snake"/>
