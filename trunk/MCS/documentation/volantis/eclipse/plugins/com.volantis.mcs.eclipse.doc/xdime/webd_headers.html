webd:headers

Purpose

Container for multiple HTTP header elements.

Contains

webd:header

Contained by

webd:get, webd:post

Example

The example shows the element forms of the webd:header attributes.

<webd:headers>
    <webd:header>
      <webd:name>User-Agent</name>
      <webd:value>Nokia5210/1.0</value>
    </webd:header>
</webd:headers>
