xfoptgroup

Purpose

Used to group a number xfoption elements.

Note: Not all markup languages support option groups or nested groups, so captions may not display on some devices.

Contains

xfoptgroup, xfoption

Contained by

xfmuselect, xfsiselect

Attributes

AttributeDescriptionTypeDefaultOptionsUse
captionThe caption for the groupxs:stringnone  required 
classAssigns a class name to an elementxs:NMTOKENnone  optional 
exprAn MCS expressionxs:stringnone  optional 
idAssigns an identifier to an elementxs:IDnone  optional 
promptPrompt textxs:stringnone  optional 

Example

<xfform name="preferences"
   method="post" action="prefs.xml"> 
   <xf:content>
      <p>To download an upgrade to version 6.2, please
      choose from the following options.</p>
   </xf:content>  
   <xfsiselect name="prefs" caption="Preferences">      
      <xfoptiongroup caption="Platform"
         initial="windows">
         <xfoption caption="Windows" value="windows"/>
         <xfoption caption="Mac OS X" value="mac"/>
         <xfoption caption="Linux" value="linux"/>
         <xfoption caption="Eclipse" value="eclipse"/>
      </xfoptiongroup>
      <xfoptiongroup caption="Language"    
         initial="en">
         <xfoption caption="English" value="en"/>
         <xfoption caption="French" value="fr"/>
         <xfoption caption="German" value="de"/>
         <xfoption caption="Italian" value="it"/>
         <xfoption caption="Spanish" value="es"/>
      </xfoptiongroup>
         ...
   </xfsiselect>    
   <xfaction type="submit" caption="Submit"/>
   <xfaction type="reset" caption="Reset"/>
</xfform>
