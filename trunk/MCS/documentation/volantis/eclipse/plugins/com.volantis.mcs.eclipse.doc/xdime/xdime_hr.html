hr

Purpose

Block element used to create a horizontal line (or rule).

Attribute groups

Block attributes, Event attributes, Core attributes

Example

<body>
   <h1>About XDIME</h1>
      <hr/>
      <p>Browser features can vary considerably.</p>
      <blockquote>For example, some will automatically
indent blockquotes from the main text margins.</blockquote>
      ...
</body>
