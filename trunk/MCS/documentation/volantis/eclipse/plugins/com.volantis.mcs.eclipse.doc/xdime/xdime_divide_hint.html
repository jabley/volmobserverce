dividehint

Purpose

Defines locations where MCS should attempt to dissect panes. It has an effect on the following block of text or element, and is cleared when the element closes.

Note: Only paragraph elements are automatically dissected into shards.

Attribute

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 

Example

<pane name='dissecting'> 
   <p>First paragraph</p>
   <dividehint/>
   <p>Second paragraph</p> 
</pane>
