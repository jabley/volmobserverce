mdc:parseAsXML

Purpose

Contains the information needed by the SQL driver to parse the values in a column as XML.

Attributes

AttributeDescriptionTypeDefaultOptionsUse
defaultNsPrefixThe prefix of namespace that should be used as the default for the parsed content.xs:stringnone  optional 
defaultNsURIThe URI of namespace that should be used as the default for the parsed content.xs:stringnone  optional 
parseParse as a complete XML document or a document fragment. If set to 'fragment', each value in the column is parsed as an external parsed entity.xs:stringdocument  optional 

Example

<sqld:query
   datasource="PressReleases" 
   sql="select TITLE,ABSTRACT 
      from PRESS_RELEASES 
      where STORY = ?">
   <sqld:string value="%{request:getParameter('STORY')}"/>
   <sqld:column name="ABSTRACT">
      <mdc:parseAsXML
         defaultNsPrefix="pr"
         defaultNsURI="http://mycorp.com/schemas/pr"/> 
   </sqld:column>
    ...
</sqld:query>
