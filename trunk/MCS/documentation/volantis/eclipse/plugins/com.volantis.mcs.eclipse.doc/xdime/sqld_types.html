SQL driver datatype elements

Purpose

There are several elements for datatypes defined in the sqld: namespace. For convenience they are grouped here rather than under individual element names.

  • sqlr:boolean
  • sqlr:date
  • sqlr:decimal
  • sqlr:double
  • sqlr:float
  • sqlr:int
  • sqlr:long
  • sqlr:string
  • sqlr:time
  • sqlr:timestamp
