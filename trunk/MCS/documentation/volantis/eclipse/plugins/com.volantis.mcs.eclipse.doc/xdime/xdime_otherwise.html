otherwise

Purpose

Conditional markup that is only rendered if none of the when elements in the containing select element has been matched.

Contained by

select

Example

<select precept="matchevery">
   <when expr="device:getPolicyValue('preferredimagetype')='GIF'">
      <img src="/logogif.mimg"/>
   </when>
   <when expr="device:getPolicyValue('preferredimagetype')='PNG'">
      <img src="/logopng.mimg"/>
   </when>
   <otherwise>
      <p>MyCorp logo</p>
   </otherwise>
</select>
