servlet:includeJSP

Purpose

Used to include JSP and XML files. If the file is not found, an error will be generated.

Attribute

AttributeDescriptionTypeDefaultOptionsUse
hrefThe relative path to the resource to include. If it starts with a '/' then it is context relative; otherwise it is page relative and is resolved relative to the base path, which is itself context relative.xs:stringnone  required 

Example

 <servlet:includeJSP href="header.jsp">
   <servlet:param name="a" value="1"/>
   <servlet:param name="b" value="2"/>
</servlet:includeJSP>
