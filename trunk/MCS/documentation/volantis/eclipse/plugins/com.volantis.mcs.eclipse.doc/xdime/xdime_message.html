message

Purpose

Container for a single canvas used for a message in Message Preparation Server (MPS).

Contains

canvas

Attribute

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 

Example

<message>
   <canvas layoutName="/message.mlyt" title="The Moons of Jupiter">
      <p pane="moons.0">The Moons of Jupiter</p>
      <pane name="moons.1">
         <p>Io</p>
         <img src="astro/io"/>
      </pane>
      <pane name="moons.2">
         <p>Europa</p>
         <img src="astro/europa"/>
      </pane>
      <pane name="moons.3">
         <p>Io</p>
         <img src="astro/ganymede"/>
      </pane>      
		...	
   </canvas>
</message>
