pipeline:parameter

Purpose

A parameter passed to the XSL transformation referenced in the href attribute on the pipeline:transformation element.

Contained by

pipeline:parameter

Attributes

AttributeDescriptionTypeDefaultOptionsUse
nameThe name of the parameter.xs:stringnone  optional 
valueThe parameter value.xs:stringfalse  optional 

Example

The example passes two parameters that match equivalent xsl:param elements in the transform. Passing parameters in transforms contains more information.

<pipeline:transform compilable="false">
   <pipeline:parameters>
      <pipeline:parameter name="title-param" value="Title"/>
      <pipeline:parameter name="artist-param" value="Artist"/>
   <pipeline:parameters>
   <pipeline:transformation href='simple-params.xsl'/>
      ...
   <pipeline:transformation>
</pipeline:transform>
