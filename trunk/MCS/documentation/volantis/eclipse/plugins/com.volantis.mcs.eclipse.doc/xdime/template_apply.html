template:apply

Purpose

Applies a template to the bound parameter values. If the optional href attribute is set, the template:binding element must be used to specify a binding. If the the attribute is omitted, multiple template:bindings and a nested template:definition must be used.

Contains

template:binding, template:bindings, template:definition

Attribute

AttributeDescriptionTypeDefaultOptionsUse
hrefThe URI of the template that is to be applied.xs:anyURInone  optional 

Example

<template:apply href="global_template.xml" >
   <template:binding name="page_title">
      <template:complexValue>
	   <pipeline:try>
	      <pipeline:preferred>
            <pipeline:transform href="title.xsl" compilable="true">
               <urid:fetch href="title.xml" />
            </pipeline:transform>	
         </pipeline:preferred>
         <pipeline:alternative>
            <pipeline:content>MyCorp</pipeline:content>
         </pipeline:alternative>
      </pipeline:try>
      </template:complexValue>
   </template:binding>
      ... 
</template:apply>
