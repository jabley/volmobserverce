pipeline:transformations

Purpose

Container for multiple pipeline:transformation elements. It must only be present if the parent pipeline:transform element does not have an href attribute.

Contains

pipeline:transformation

Contained by

pipeline:transform

Example

<pipeline:transform>
   <pipeline:transformations>
	   <pipeline:transformation href="filter_men.xslt"/>
		<pipeline:transformation href="filter_women.xslt"/>
		<pipeline:transformation href="sort_by_surname.xslt"/>
	</pipeline:transformations>
</pipeline:transform>
