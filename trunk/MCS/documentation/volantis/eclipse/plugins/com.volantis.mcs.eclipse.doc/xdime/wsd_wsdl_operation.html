wsd:wsdl-operation

Purpose

Defines the web service operation to be requested.

Contained by

wsd:request

Attributes

AttributeDescriptionTypeDefaultOptionsUse
operationThe name of the operationxs:NCNamenone  optional 
portTypeThe name of the port type in the WSDL descriptionxs:NCNamenone  optional 
wsdlA reference to a wsdl-entry element in the mcs-config.xml filexs:anyURInone  optional 

Example

<wsd:request>
   <wsd:wsdl-operation
      wsdl="http://localhost:8080/pipeline/test.wsdl"
      portType="FindIPSoap"
      operation="GetURLIP"/>
   <wsd:message>
      <!-- message here-->
   </wsd:message>
</wsd:request>
