winaudio

Purpose

Used to specify a subset of Windows Audio parameters in an embedded clip. Refer to the Windows Media Player documentation for further information on parameter options.

Note: This element is deprecated. It will be retained for compatibility and removed or replaced in a later release

Contained by

pane

Attributes

Name Description Type Default Options Use
showAudioControls Whether or not the mute and volume controls are visible xs:boolean none optional
showControls Whether or not any controls are visible xs:boolean none optional
showDisplay Whether additional information is displayed xs:boolean none optional
showPositionControls Whether the slider is visible or not xs:boolean none optional
showTracker Whether the slider bar is visible or not xs:boolean none optional

Attribute groups

Block attributes, Clip attributes, Event attributes, Hypertext attributes, Core attributes

Example

<winaudio
   name="lionroars"
   showControls="true"
   altText="Roaring Lion"/>
