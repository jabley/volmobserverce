xfcontent

Purpose

Used to add XDIME content to a form.

Contained by

xfform

Attribute group

Block attributes

Example

<xfform name="preferences"
   method="post" action="prefs.xml"> 
   <xf:content>
      <p>To download an upgrade to version 6.2, please
      choose from the following options.</p>
   </xf:content>  
   <xfsiselect name="prefs" caption="Preferences">           
   ...
   </xfsiselect>    
   <xfaction type="submit" caption="Submit"/>
   <xfaction type="reset" caption="Reset"/>
</xfform>
