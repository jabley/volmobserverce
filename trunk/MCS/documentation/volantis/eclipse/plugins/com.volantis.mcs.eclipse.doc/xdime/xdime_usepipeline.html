usePipeline

Purpose

Used to integrate JSPs with the pipeline process. It has no effect on XML content and can be used with no impact.

Attribute

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 

Example

<canvas>
   ...
   <usePipeline>
      <pipeline:content>
	      <webd:get url="http://localhost:8082/webdav/dci.xml">
            <webd:content type="text/xml" action="ignore"/>	 
		   </webd:get>
      </pipeline:content>
<usePipeline>
