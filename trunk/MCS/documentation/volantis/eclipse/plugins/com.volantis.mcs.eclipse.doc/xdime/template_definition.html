template:definition

Purpose

Container for template elements.

Contains

template:body, template:declarations, template:documentation

Example

The example shows the general structure of the template content model.

<template:definition>
   <template:declarations>
      <template:parameter name="page_title" type="complex"
         use="required" />
      <template:parameter name="page_content" type="complex"
         use="optional" />
      <template:parameter name="page_layout" type="simple"
         expressionVariable="PAGE_LAYOUT" use="required" />
      <template:parameter name="cache_key" type="simple" 
         expressionVariable="CACHE_KEY" use="required" />
   </template:declarations>
   ...
</template:definition>
