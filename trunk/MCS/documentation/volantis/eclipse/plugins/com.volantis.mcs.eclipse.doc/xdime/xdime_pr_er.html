XDIME presentation elements

The majority of XDIME presentation elements do not require a namespace prefix. They are listed in a single alphabetical order. A few elements in the Inline Plugin Module namespace with the prefix vtim: also belong to the presentation element set. These are listed separately.

For information about the structure of element references you should refer to About element references.

Additional XDIME content integration elements are listed in XDIME content integration elements in namespace groupings.

For further guidance on the use of XDIME presentation markup you should read Web author's guide. For complete details of the XDIME content model, refer to the XML schema contained in the marlin-cdm.xsd file.
