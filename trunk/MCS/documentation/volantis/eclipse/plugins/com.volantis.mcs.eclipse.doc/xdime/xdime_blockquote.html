blockquote

Purpose

Designates a block of text quoted in the current document.

Attribute

AttributeDescriptionTypeDefaultOptionsUse
citeThe URL of the source of the quoted materialxs:stringnone  optional 

Attribute groups

Block attributes, Event attributes, Core attributes

Example

<blockquote>Some browsers will automatically
indent blockquotes from the main text margins.</blockquote>
