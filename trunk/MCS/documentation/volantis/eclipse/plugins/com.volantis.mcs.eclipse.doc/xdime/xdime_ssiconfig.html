ssiconfig

Purpose

Contains the error message to be used if the ssiinclude element fails.

Contained by

wsdirectives

Attributes

AttributeDescriptionTypeDefaultOptionsUse
errmssgThe error messagexs:stringnone  optional 
exprAn MCS expressionxs:stringnone  optional 

Example

<wsdirectives>
   <ssiconfig errmsg="The file 'Q4report.html'is not available."/>
   <ssiinclude virtual="/mycorp/sales 2004/Q4report.html"/>
</wsdirectives>
