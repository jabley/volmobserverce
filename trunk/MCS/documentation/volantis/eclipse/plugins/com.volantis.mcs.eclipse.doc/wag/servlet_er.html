Servlet elements

The XDIME servlet elements have the namespace prefix servlet: and the namespace declaration is http://www.volantis.com/xmlns/marlin-servlet.

NamePurpose
servlet:includeJSP

Used to include JSP and XML files. If the file is not found, an error will be generated.

servlet:includeServlet

Used to include XML and JSP servlets. If the servlet is not found, an error will be generated.

servlet:param

Used for passing parameters to a server resource. They may be passed either by adding them to the query string in the path attribute on the including element, or by using nested servlet:param elements, or a combination of both. The parameters specified in nested elements take precedence over those specified in the path. An advantage of nested parameters, is that you can specify their values as expressions, such as templating expressions.
