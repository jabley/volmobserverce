SQL driver elements

The XDIME SQL result namespace prefix is sqlr: and the namespace declaration is http://www.volantis.com/xmlns/marlin-sql-result.

NamePurpose
sqlr:result

Contains the result of a SQL query or modification. For a query it contains multiple row elements. For a modification it contains a count of the rows that have been updated.

sqlr:row

The row container for a SQL result.

sqld:query

Container for a SQL query. The query can be a normal query, or be configured to take parameters. If a parameter is used, the query element must contain a nested string parameter.

sqlr:updateCount

Contains a count of the rows that have been updated with a sqld:modify.

SQL result datatype elements

There are several elements for datatypes defined in the sqlr: namespace. For convenience they are grouped here rather than under individual element names. The elements are empty, with the exception of the sqlr:exml element, where the content model is xs:any.
