Web author's guide

This guide provides background information that will help you to:

  • Develop MCS policies for devices using Eclipse
  • Create device independent content using XDIME 2 markup
  • Work with Dynamic Content Integration elements to access external resources
  • Use the Client Framework and the Widget Library

You'll find references from these topics to those in MCS Help sections, and to the element reference topics.

Several topics include diagrams showing the structure of XDIME schemas.
