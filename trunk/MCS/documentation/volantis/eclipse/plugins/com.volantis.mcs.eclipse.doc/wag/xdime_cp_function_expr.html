Container reference functions

XDIME 2 provides function expressions to that you can use to apply styles to content in iterator layouts. These functions are not defined to have a namespace. See XDIME 2 in layouts for examples using these functions.

NamePurpose
counter

Defines content instances in specific iterator panes in conjunction with the mcs-container-instance function.

mcs-container-instance

Selects a specific instance of a layout container for XDIME 2 markup.
