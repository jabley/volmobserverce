Working with XDIME 2

XDIME 2 provides developers with a simple, standards-compliant authoring environment.

The vocabulary is easy to learn and use. It relies on key web authoring specifications, and the way in which these standards are combined and used is based on work currently under way in the W3C Device Independence Working group.

  • XHTML Version 2
  • CSS Version 2.1 enhanced with some features of the forthcoming CSS Version 3
  • XForms Version 1.0
  • DISelect Version 1.0

If you are not familiar with these recommendations, you may want to review them before you develop any XDIME 2 content, and refer to the XDIME 2 elements together with any guidelines provided by your carrier partners.
