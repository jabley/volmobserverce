Web service driver elements

The XDIME web service driver namespace prefix is wsd: and the namespace declaration is http://www.volantis.com/xmlns/marlin-web-service-driver.

NamePurpose
wsd:message

Container for the outgoing message. This element has an open content model.

wsd:request

Container for a web service request.

wsd:wsdl-operation

Defines the web service operation to be requested.
