Installing on Caucho Resin

You need to create a directory that the machine running Resin can access, to contain the MCS installation MCS/webapps/[context_root] directory. It can be local to the server machine or accessed via a drive mapping. For example it might be a directory under the webapps directory of the Resin installation.

  1. Create a directory named [context_root] under the Resin webapps directory
  2. Copy the entire contents of the [path]/MCS/webapps/[context_root] directory to the Resin webapps/ directory, for example $RESIN_HOME/webapps/[context_root]

Note: If you are using Resin on a non-windows machine, copy the -xercesImpl.jar and xmlParserAPIs.jar files from the MCS/lib directory to the $RESIN_HOME/lib directory. Make sure that the generic xerces.jar file is not in the $RESIN_HOME/lib directory and is copied elsewhere. This is to ensure that the correct version of Xerces parser is used.
