Adding type allocation codes

On the Overview page of the Device Repository editor you can add GSM Association Type Allocation Codes (TAC )for a device.

  1. Choose a device from the Devices tree
  2. In the Type Allocation Codes section click Add. A new item is added to the list.
  3. Enter a six or eight digit type approval code

Note: Each TAC must be unique in the repository, and must be a 6 or 8 digit number. If you enter an invalid code, you will see an error message.

To remove a code, select it and click Remove.

To revert to the supplied repository codes for the device click Restore Defaults.
