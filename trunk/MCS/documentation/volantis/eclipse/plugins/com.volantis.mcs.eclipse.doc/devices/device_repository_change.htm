Changing a device repository

When you create a new web project, you need to associate the project with a device repository. However, you can later change to another repository if necessary.

To change to another repository:

  1. Right click on a project folder in the policy tree
  2. Choose Properties to open the Properties dialog
  3. Select MCS Properties Page
  4. In the Standard device repository file control, click the Browse button and navigate to a device file on your system
  5. Click OK to close the dialog
