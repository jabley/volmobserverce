Editing policy values

You can edit the policy values of devices using the Device Repository editor, including policies that you have added under the Custom category. MCS displays the controls in alphabetical order.

  1. Go to the Policies page in the editor
  2. In the Outline view of the device tree select a device
  3. Select a category to display the policy values in the policies section. If the control is enabled, go to step 5.
  4. Next to a policy choose Override from the drop down to enable editing
  5. Edit the policy value

To use the fallback device value choose Fallback.

To revert to the standard device policy value choose Restore Default.
