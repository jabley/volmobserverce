Choosing a device

You can associate devices with layouts and themes by selecting one or more devices from the device repository.

A tree view of the available devices is included as a page in the New Layout and New Theme wizards. You can also view and select from the tree in the Layout and Theme editors. When you add a device layout, you additionally specify either a canvas or montage layout.

If you have already created a layout or theme and you open a device view, any device names that you have already selected are filtered from the view.

  1. In the editor click the Browse button under the device list
  2. If you are choosing a device for a layout, select either Canvas or Montage in Device Layout Type
  3. In the tree view, expand the tree, and check the checkbox next to the device names to select them
  4. Click Finish to close the wizard

Tip: To deselect all the files in a folder and its sub folders, uncheck the box next to the folder name or click Deselect all.
