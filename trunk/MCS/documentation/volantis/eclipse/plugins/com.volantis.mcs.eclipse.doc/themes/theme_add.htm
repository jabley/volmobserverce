Adding a theme

You can use the New Theme wizard to create device themes. The wizard allows you to:

  • Define a theme name and its location in your project folders
  • Select a device to which the theme applies

Naming a theme

The first page allows you to locate and name the theme. Each theme is a file in your project. The focus will initially be on the Theme text box, and the policy tree view will be synchronized with your current project folder selection.

  1. Choose the New Theme wizard.
  2. Enter a name in the Theme text box.
  3. Choose a folder in the policy tree view or enter the full path from the project folder in the first text box.
  4. Click Next and choose a device.

The new theme icon is displayed in the policy tree, and a Theme editor tab is opened and highlighted.

Defining theme attributes

The Theme editor has two pages, Overview and Design. To change pages, click one of the tabs at the bottom left of the editor.

On the Overview page you'll see a list of the devices you selected for this theme. You can add new devices or remove them at any time using the Add or Remove buttons.

  1. In the device list, select a device name.
  2. If you want to use an existing style sheet instead of editing style rules, enter the URL for the external Cascading Style Sheet (CSS) file.
