Working with themes

You can define theme policies for individual devices and groups of devices. A theme is made up of a set of style rules, and each rule is identified by a selector, and has a set of properties.

To develop a theme in MCS you have to compete the following tasks:

  1. Add a theme policy to your project using the New Theme wizard
  2. Using the Theme editor, choose a device or devices to associate with your theme
  3. Add selectors to the theme and assign style properties to them
