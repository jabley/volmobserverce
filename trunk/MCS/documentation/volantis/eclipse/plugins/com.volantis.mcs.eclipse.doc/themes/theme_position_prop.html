Position properties

Purpose

Defines a layout positioning scheme of generated boxes. These affect the positioning properties of any floating elements.

ControlDescriptionOptions
Position

Absolute or relative position of a box in relation to the normal text flow

absolute, fixed, relative, static 
Top/Right/Bottom/Left

Offsets and units applied to a box depending on the Position value

 
Clear

Whether the current element allows preceding elements to float along its sides

both, left, right 
Float

Whether a box should float to the left, right, or not at all

left, right 
Vertical Align

Relative vertical positioning in a line

baseline, bottom, middle, super, sub, text-bottom, text-top, top 
Caption Side

Specifies the position of the caption in relation to a table

bottom, top
