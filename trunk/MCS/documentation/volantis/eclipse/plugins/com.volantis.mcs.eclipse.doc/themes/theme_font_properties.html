Font properties

Property Description
System Font
 Sets all font properties to the values in the selected system font on the user's machine
Family Lists multiple font families. Browse opens Font Family Selection dialog. Names containing white space must be enclosed in single or double quotes.
Size Absolute size using browser table or relative size in relation to parent
Size Adjust
 An aspect value that preserves x-height of the first choice font in the substitute font
Stretch Selects a normal, condensed, or extended face
Style Style of font
Font Variant Sets the font to normal or small-caps
Font Weight Weight of font. The bolder and lighter values are relative to the inherited font weight. Other values are absolute font weights.
