Layout properties

Purpose

Controls several layout properties of different objects, including the display of fragments and links.

Note: If you are using XDIME-CP, you specify a layout for the whole XDIME page and a region element to contain the XDIME-CP markup. You associate XDIME-CP elements with this layout using the Target Layout and Target Container, both of which must be present. The value of 'current' means that the element will use the current layout or container.

ControlDescriptionOptions
Target Container

The container in the Target Layout where the content should be placed. An expression may be used to identify a specific container instance.

current, [container_name], [expression] 
Target Layout

Defines the target MCS layout for an element and its content

current, [layout_name] 
Fragment List Orientation

Orientation of the list of links to fragments in the root fragment

current 
Fragment List Position

Position of the list of links to fragments in the root fragment

bottom, top 
Link Media

Medium the target of the link is intended for

normal, tV, web 
Link Style

How links should be displayed

default, button, numeric-shortcut
