Box properties

Purpose

Box style properties that set box and table values such as display, visibility, margins and padding.

ControlDescriptionOptions
Display

Box display values

block, compact, inline, list-item, marker, run-in, table, inline-table, table-caption, table-cell, table-column, table-column-group, table-footer-group, table-header-group, table-row, table-row-group 
Line Height

Either the minimal height of each inline box or the exact height of each box

 
Height

Content height and height units

 

Content width and width units

 
Visibility

Whether the box is visible or hidden

visible, hidden, collapse 
Margin/Padding

Controls for all values, or the four individual sides. All synchronizes the values.
