Separator properties

Purpose

If you specify that menus are oriented horizontally, MCS applies the separator properties between groups of menuitems.

ControlDescriptionOptions
Characters

The literal character(s) to be used

 
Position

Where the separators are located in relation to groups

before, after, both 
Repeat

Number of times to repeat the separator

  
Image

The name of the image component or URI to be used as the separator. Browse opens the Image Component Selection dialog.

none, [image_component] or URI 
Horizontal Character

Spacing between groups

none, space, non-breaking-space 
Type

Whether to use characters or an image component between groups

none, image, characters
