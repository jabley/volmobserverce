mcs:unit

Purpose

Container for an XDIME 2 content fragment, which need not be well formed, for example a reusable header or footer.

Examples

If the code in the first example is saved as footer.xdime, then the urid:fetch reference in the second example will add the footer content to the main body of the page.

<mcs:unit>
   <p><a href="../legal.html">Legal notice</a> | 
   Copyright 2007 Example software Gmbh. All rights reserved.</p>
<mcs:unit>
<body>
   ...
   <div id="footer">
      <urid:fetch href="footer.xdime"/>  
   </div>
<body>
