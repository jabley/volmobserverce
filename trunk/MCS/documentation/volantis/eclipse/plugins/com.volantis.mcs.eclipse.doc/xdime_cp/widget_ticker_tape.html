widget:ticker-tape

Purpose

Defines a Ticker Tape widget which provides a horizontal scrolling display area.

Contains

ticker:feed, widget:refresh

Contained by

body

Attribute group

Core attributes

Example

Create a Ticker Tape that updates its contents dynamically every 15 seconds.

<widget:ticker-tape>
   <widget:refresh src="service/ticker?id=myWeatherTicker" interval="15"/>
   <span>Ticker Tape can be configured to periodically refresh its content by fetching data,
      such as weather conditions, from a web service...</span>
</widget:ticker-tape>
