widget:block-content

Purpose

A structural element enclosing content, that can be displayed by widget:block. It supports transition effects.

Contained by

widget:block

Attribute group

Core attributes

Example

The initially hidden 'Hello world' text can be shown or hidden using corresponding buttons.

<widget:block id="block">
   <widget:block-content>Hello world</widget:block-content>
</widget:block>

<widget:button action="block#show">Show</widget:button>
<widget:button action="block#hide">Hide</widget:button>
