gallery:pages-count

Purpose

Displays the total number of pages in a selected gallery.

Contained by

gallery:gallery

Attribute group

Core attributes

Example

<gallery:gallery class="gallery" slideshow="slideshow" slideshow-popup="gallery-popup">
   <table>
      <tr>
         <td>
            <widget:previous>&lt;&lt;</widget:previous>
         </td>
         <td>
            <gallery:item-display/>
         </td>
         <td>
            <gallery:item-display/>
         </td>
         <td>
            <gallery:item-display/>
         </td>
         <td>
            <widget:next>&gt;&gt;</widget:next>
         </td>
      </tr>
   </table>
   <div>Displaying items from <gallery:start-item-number/> to
      <gallery:end-item-number/> of <gallery:items-count/></div>
</gallery:gallery>
