ticker:item-description

Purpose

Provides description of the item that is being displayed by the Item Display widget.

Contained by

ticker:item-display

Attribute group

Core attributes

Example

<ticker:item-display>
   <strong>Title: </strong><ticker:item-title/>
   <em>Details:</em>
   <p><ticker:item-description/></p>
      Used channel: <ticker:item-channel/>
   <widget:dismiss>Close</widget:dismiss>
</ticker:item-display>
