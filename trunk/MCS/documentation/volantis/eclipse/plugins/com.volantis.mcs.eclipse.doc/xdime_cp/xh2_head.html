head

Purpose

Information such as the title and other metadata that is not document content. The contained title element is required.

Contains

mcs:script, title, xf:model, access, link, meta, style

Contained by

html

Attribute groups

Core attributes, Selection attributes

Example

<html>
   <head>
      <title>About XDIME 2</title>
      <meta property="mcs:cache-scope">optimistic</meta>
      <meta property="mcs:max-age">180</meta>
      ...       
   </head>
</html>
