xf:textarea

Purpose

A form control for larger amounts of text. Multi-line input, such as the body of an message, is an example.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<xf:textarea ref="comments">
   <xf:label>Please add any comments here</xf:label>
</xf:textarea>
