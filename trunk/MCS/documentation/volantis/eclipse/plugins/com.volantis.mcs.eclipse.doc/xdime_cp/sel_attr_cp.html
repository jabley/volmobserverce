Selection attributes

Purpose

Attributes used to select alternative content.

If the expr and id attributes are used in an element other than the sel: namespace, they must have a sel: prefix. If the element namespace is sel: no prefix is required.

Attributes

Attribute Description Type Default Options Use
sel:expr

Determines whether or not the content is selected for processing. If the expression evaluates to 'true', the content is selected, otherwise it is not processed. If you omit the attribute the content is selected for processing by default.

 xs:string     optional 
sel:selid

The identifier for a host language element. During processing, this attribute is replaced with an id attribute with the same value.

 xs:string     optional
