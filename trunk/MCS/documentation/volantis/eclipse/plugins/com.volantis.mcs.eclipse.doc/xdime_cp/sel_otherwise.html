sel:otherwise

Purpose

Defines a set of material to be selected for processing if none of the sel:when elements within the containing sel:select element are selected.

Contains

sel:if, sel:select

Contained by

sel:select

Example

<sel:select>
   <sel:when
      expr="device:getAncestorRelationship('PC') != 'ancestor'">
      <h3 class="company_header">Jive at the Olympics</h3>
         <h4>New York - July 20: Jive selected as supplier to
         Team USA Basketball Olympic squad</h4>
    </sel:when>
    <sel:otherwise>
       <h3 class="company_header">New York - July 20: Jive
          selected as supplier to Team USA Basketball Olympic
          squad </h3>
    </sel:otherwise>
</sel:select>
