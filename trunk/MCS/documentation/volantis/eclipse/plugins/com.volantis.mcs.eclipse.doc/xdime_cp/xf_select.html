xf:select

Purpose

A form control that provides for selection of one or more choices from a list of options.

Contains

xf:item, xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<xf:select ref="rss">
<xf:label>Choose an RSS feed</xf:label>
   <xf:item id="news">
      <xf:label>World headlines</xf:label>
      <xf:value>yes</xf:value>
   </xf:item>
   <xf:item id="sport">
      <xf:label>Latest sports</xf:label>
      <xf:value>yes</xf:value>
   </xf:item>
   <xf:item id="leisure">
      <xf:label>Entertainment</xf:label>
      <xf:value>no</xf:value>
   </xf:item>
   <xf:item id="tech">
      <xf:label>Science and technology</xf:label>
      <xf:value>yes</xf:value>
   </xf:item>
</xf:select>
