xf:secret

Purpose

Used for input of confidential or sensitive information such as passwords.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<xf:input ref="username">
   <xf:label>Username</xf:label>
</xf:input>
<xf:secret ref="password">
   <xf:label>Password</xf:label>
</xf:secret>
