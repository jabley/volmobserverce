gallery:slideshow

Purpose

The main element of a Slideshow widget that displays either the content identified in the items attribute, or in the same attribute on an associated gallery:gallery element.

Contains

gallery:item-display, gallery:item-number, gallery:items-count, widget:next, widget:pause, widget:play, widget:previous, widget:stop

Contained by

widget:popup, body

Attribute

AttributeDescriptionTypeDefaultOptionsUse
itemsIdentifies the set of items to be displayed in a slideshow. If this attribute is not specified, the default set will be used.xs:string  optional 

Attribute group

Core attributes

Example

<widget:popup id="gallery-popup" class="popup">
   <gallery:slideshow id="slideshow" class="slideshow">
      <gallery:item-display/>
      <table>
         <tr>
            <td>
               <widget:previous>&lt;&lt;</widget:previous>
            </td>
            <td>
               <widget:pause>Pause</widget:pause>
            </td>
            <td>
               <widget:play>Play</widget:play>
            </td>
            <td>
               <widget:next>&gt;&gt;</widget:next>
            </td>
            <td>
               <widget:dismiss>Close</widget:dismiss>
            </td>
         </tr>
      </table>
      <div class="slideshow-info">Displaying item <gallery:item-number/> of
         <gallery:items-count/></div>
   </gallery:slideshow>
</widget:popup>
