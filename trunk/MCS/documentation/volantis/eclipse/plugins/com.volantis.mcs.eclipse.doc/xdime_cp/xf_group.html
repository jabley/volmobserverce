xf:group

Purpose

Combines a set of form controls into a user interface component. By applying theme properties on this element, you can control where form fragmentation occurs, and define or override at runtime the link texts specified on form fragment layouts.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<xf:group class="personal-details">
   <xf:group class="name">
      <xf:label>Name</xf:label>
      ...
   </xf:group>
   <xf:group class="address">
      <xf:label>Address</xf:label>
      ...
   </xf:group>
</xf:group>
<xf:group class="order">
    ...
</xf:group>
