widget:tab

Purpose

Defines a single tab for the Tabs widget. May contain the optional widget:load element for AJAX-loaded tabs.

Contains

widget:load

Contained by

widget:tabs

Attribute group

Core attributes

Example

<widget:tabs>
   <widget:tab>
      <p>Hello world</p>
   </widget:tab>
   <widget:tab>
      <widget:load src="ajax-tab-response.xdime"/>
      <p>This content will be changed by the AJAX request</p>
   </widget:tab>
</widget:tabs>
