widget:button

Purpose

A general purpose element, used by widgets which need a button-like control.

The :mcs-disabled pseudo-class may be used to specify styles for a button in the disabled state.

Contained by

widget:wizard

Attribute

AttributeDescriptionTypeDefaultOptionsUse
actionSpecifies an action to be invoked when the button is clickedxs:string clear-content, clear-value, disable, dismiss, enable, execute, first-page, force-sync, hide, hide-content, invoke, last-page, launch, load, next-month, next-page, next-row, next-year, press, previous-month, previous-page, previous-row, previous-year, reset, show, set-today show-content, split, start, stop optional 

Attribute group

Core attributes

Actions

ActionDescription
disableDisables the button
enableEnables the button
pressPresses the button. This action is enabled only when the button itself is enabled.

Property

PropertyDescriptionTypeAccess
is-enabledSpecifies whether or not the button is enabled. When a button is associated with an action, this property is a mirror of the is-enabled property of the corresponding action. In such cases this property is not writable.xs:booleanread/write 

Event

EventDescription
pressedThe event is sent when the button is pressed

Example

In the following example the widget:button element is used to launch the Forms Wizard widget with the id attribute set to 'myWizard'.

<widget:button id="start-button" class="button" action="myWizard#launch">
   Start the wizard
</widget:button>
