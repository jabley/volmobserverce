tr:skip-time

Purpose

The skip time, given as a number of minutes since midnight in client's time zone. The possible values are 0-1439. Up to two skip times can be defined for a single Feed Poller widget.

Contained by

tr:set-skip-times

Attributes

AttributeDescriptionTypeDefaultOptionsUse
fromSets the beginning of a idle period. It is given as a number of minutes since midnight in the time zone of the client device.xs:intiger  required 
toSets the end of the idle period. It is given as a number of minutes since midnight in the time zone the client device.xs:intiger  required 

Attribute group

Core attributes

Example

In this response example, following commands will be executed:

  • insert new item with specified ID, channel, title and description
  • remove item with specified ID
  • set (redirect) URL to the new BuzzCast Server
  • set new skip times to be between 11:00-12:00, and 22:00-2:30
  • set new polling interval to 10 minutes
<response:response
   xmlns="http://www.w3.org/2002/06/xhtml2"
   xmlns:wr="http://www.volantis.com/xmlns/2006/05/widget/response"
   xmlns:tr="http://www.volantis.com/xmlns/2006/09/ticker/response"/>

   <response:head
      <response:link rel="mcs:theme" href="/myTheme.mthm"/>
   </response:head>

   <response:body>
      <tr:feed-poller>
         <tr:add-item item-id="763h" channel="F1">
            <tr:title>Important News</tr:title>
            <tr:icon><object src="/icon.mimg"/></tr:icon>
            <tr:description>Breaking News
               <a href="http://www.importantnews.com">Read more...</a></tr:description>
         </tr:add-item>
         <tr:remove-item item-id="4532"/>
         <tr:set-url>http://www.new-service.com/vmds</tr:set-url>
         <tr:set-skip-times>
            <tr:skip-time from="1140" to="1200"/>
            <tr:skip-time from="1320" to="150"/>
         </tr:set-skip-times>
         <tr:set-polling-interval>600</tr:set-polling-interval>
      </tr:feed-poller>
   </response:body>
</response:response>
