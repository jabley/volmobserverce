BuzzCast elements

XDIME 2 makes use of several elements from the BuzzCast and Widget vocabularies, used to animate the user interface on supporting mobile devices. The namespace prefix used for the BuzzCast elements is ticker: and the namespace declaration is http://www.volantis.com/xmlns/2006/09/ticker.

NamePurpose
ticker:channels-count

Returns the number of distinct channels in the set of items.

ticker:feed

This element, if inserted as a child of the Carousel or Ticker Tape, will make these widgets show feed obtained by the Feed Poller from the BuzzCast Server. Clicking on an item in Carousel or Ticker Tape widget will show this item in the associated Item Display widget.

ticker:feed-poller

The main element for the Feed Poller widget.

ticker:item-channel

Specifies channel of the item being displayed by the Item Display widget.

ticker:item-description

Provides description of the item that is being displayed by the Item Display widget.

ticker:item-display

The main element for the Item Display widget.

ticker:item-icon

A placeholder for an item's icon.

ticker:item-title

Provides a title of the item being displayed by the Item Display widget.

ticker:items-count

Returns the number of items held by the BuzzCast client.

ticker:update-status

Displays status of the Feed Poller widget.
