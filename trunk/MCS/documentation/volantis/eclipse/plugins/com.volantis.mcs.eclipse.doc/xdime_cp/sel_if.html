sel:if

Purpose

Provides an alternative to using the sel:expr attribute alone when a selection condition applies to several elements. You can also use it control processing a document fragment that does not have a single root element.

Note: There is no sel:else element. Instead, the sel:select element and its children provide a way to choose between alternative content.

Contains

sel:if, sel:select

Contained by

sel:if

Attribute

AttributeDescriptionTypeDefaultOptionsUse
expr

Determines whether or not the content is selected for processing. If the expression evaluates to 'true', the content is selected, otherwise it is not processed. If you omit the attribute the content is selected for processing by default.

xs:string  optional 

Example

<sel:if sel:expr="dc:cssmq-width('px') &gt; 200">
   <sel:select>
      <object src="image1"/>
      <p>Many people had to be evacuated.</p>
   </sel:select>
</sel:if>
