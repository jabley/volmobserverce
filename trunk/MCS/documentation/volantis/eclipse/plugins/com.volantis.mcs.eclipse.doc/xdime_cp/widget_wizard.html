widget:wizard

Purpose

Defines a Forms Wizard widget.

Contains

widget:button, XForms elements

Contained by

body

Attribute

AttributeDescriptionTypeDefaultOptionsUse
cancel-dialogProvides a reference to the Popup widget that shows the warning message when user wants to cancel the wizardxs:string  optional 

Attribute group

Core attributes

Action

ActionDescription
launchCauses the wizard to start

Event

EventDescription
completedThe event send when the wizard is completed and the form content is returned to the server

Example

<head>
   <xf:model>
      <xf:instance>
         <si:instance>
            <si:item name="login"/>
            <si:item name="password"/>
         </si:instance>
      </xf:instance>
      <xf:submission id="myForm" action="wizard.xdime"/>
   </xf:model>
   <widget:script id="dismiss-yes-script"> $W('myPopup').dismiss('yes'); </widget:script>
   <widget:script id="dismiss-no-script"> $W('myPopup').dismiss('no'); </widget:script>
</head>
<body>
   <widget:wizard class="widget" id="myWizard" cancel-dialog="myPopup">
      <xf:group>
         <div>Please enter your login:</div>
         <table>
            <tr>
               <td>Login</td>
               <td>
                  <xf:input ref="login"/>
               </td>
            </tr>
         </table>
      </xf:group>
      <xf:group>
         <div>Please enter your password:</div>
         <table>
            <tr>
               <td>Password</td>
               <td>
                  <xf:input ref="password"/>
               </td>
            </tr>
            <tr>
               <td/>
               <td>
                  <xf:submit submission="myForm" id="submit-button">
                     <xf:label>OK</xf:label>
                  </xf:submit>
               </td>
            </tr>
         </table>
      </xf:group>
      <widget:popup id="myPopup">
         <div>Do you really want to cancel? All the data you entered will be lost.</div>
         <div class="buttons">
            <widget:button id="popup-dismiss-ok" class="button"
               action="dismiss-yes-script#invoke">Yes</widget:button>
            <widget:button id="popup-dismiss-no" class="button" action="dismiss-no-script#invoke"
               >No</widget:button>
         </div>
      </widget:popup>
   </widget:wizard>
   <widget:button id="start-button" class="button" action="myWizard#launch">Start the
   wizard</widget:button>
</body>
