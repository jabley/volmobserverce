object

Purpose

A reference to a media resource. To provide alternative text, you can place the text in the object content.

If the srctype attribute contains device independent types, then the src attribute must be an MCS policy name with the extensions .mauc, .mdyv or .mimg.

If the srctype attribute contains device dependent types, then they must all be of the same family, that is all video, all audio, all image, or all application/x-shockwave-flash.

You should always specify values for the width and height attributes to ensure that the object element will work for all devices.

Example

<object src="summit.png">
   <caption>Leaders at the recent NATO summit</caption>
   <param name="mcs-aspect-ratio-width" value="320"/>
   <param name="mcs-aspect-ratio-height" value="180"/>
</object>
