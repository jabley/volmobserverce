si:instance

Purpose

Container for data items used to supply initial values for forms controls, or to provide additional data to be submitted with the form.

Contains

si:item

Contained by

xf:instance

Example

<head>
   <title>Subscriber update</title>
   <xf:model id="address">
      <xf:instance>
         <si:instance>
            ...
            <si:item name="postcode">13004 Marseille</si:item>
            <si:item name="hidden">subscriber_id</si:item>
         </si:instance>
      </xf:instance>
       ...
   </xf:model>
</head>
