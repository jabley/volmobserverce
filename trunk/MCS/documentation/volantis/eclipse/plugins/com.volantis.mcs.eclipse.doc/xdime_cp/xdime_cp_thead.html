thead

Purpose

Container for a table header.

Note: This element is not supported except in the context of the widget:table element.

Contains

tr

Contained by

widget:table

Attribute group

Core attributes

Example

<widget:table>
   <thead>
      <tr>
         <th>Code</th>
         <th>Country name</th>
      </tr>
   </thead>
   <widget:tbody id="tbody" cached-pages-count="10" style="mcs-table-rows-per-page:10">
      <widget:load src="service/table" when="onload"/>
      <tr>
         <td>--</td>
         <td>Loading, please wait...</td>
      </tr>
   </widget:tbody>
</widget:table>
