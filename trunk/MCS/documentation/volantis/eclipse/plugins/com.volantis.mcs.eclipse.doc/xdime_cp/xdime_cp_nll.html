nl

Purpose

Defines a navigation list. The label element is required.

Contains

label, li

Attribute groups

Core attributes, Hypertext attributes, Selection attributes

Example

<nl>
   <label>Contents</label>
      <li>Concepts</li>
      <li>
      <nl>
      <label>Tasks</label>
         <li>Install</li>
         <li>Configure</li>
         <li>Test</li>
      </nl>
      </li>
      <li>Reference</li>
      <li>Help</li>
      ...
</nl>
