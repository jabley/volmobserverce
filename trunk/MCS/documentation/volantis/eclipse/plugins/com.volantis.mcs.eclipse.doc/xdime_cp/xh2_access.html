access

Purpose

Binds a numeric access key to an a element.

Contained by

head

Attributes

AttributeDescriptionTypeDefaultOptionsUse
keyThe unique key valuexs:integernone  required 
targetidThe identifier of the target elementxs:IDnone  required 

Attribute group

Core attributes

Example

<head>
    <title>Sample Access Key Binding</title>
    <access key="1" targetid="home"/>
    <access key="2" targetid="news"/>
    <access key="3" targetid="hr"/>
</head>
<body>
    <div>
       <a href="home.xdime" id="home">Jive Sports</a>
       <a href="company_news.xdime" id="news">Company News</a>
       <a href="hr.xdime" id="hr">Human Resources</a>
    </div>
</body>
