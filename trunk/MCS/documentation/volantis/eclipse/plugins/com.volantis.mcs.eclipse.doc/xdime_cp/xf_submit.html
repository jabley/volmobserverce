xf:submit

Purpose

A form control that initiates submission of the form data for processing.

Contains

xf:label, xf:setvalue

Attributes

AttributeDescriptionTypeDefaultOptionsUse
refThis attribute is part of the XForms schema, but in the context of a submit action it has no meaning. You therefore need not set it.xs:NCNamenone  optional 
submissionThe identifier of the submission element that defines how the form is submittedxs:IDREFnone  required 

Attribute groups

Core attributes, Selection attributes

Example

<html>
   <head>
      <xf:model>
         <xf:submission id="do-login" action="doLogin.jsp" method="get"/>
      </xf:model>
   </head>
   <body>
      <xf:input ref="username">
         <xf:label>Username:</xf:label>
      </xf:input>
      <xf:secret ref="password">
         <xf:label>Password:</xf:label>
      </xf:secret>
      <xf:submit submission="do-login">
         <xf:label>Login</xf:label>
      </xf:submit>            
   </body>
</html>
