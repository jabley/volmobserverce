filter

Purpose

Defines the pipeline configuration for rendering scripts in HTTP sources.

Contained by

module

Attributes

Name Description Type Default Options Use
class The name of custom class that filters a type of script xs:string none required
content-type The MIME type of the script xs:string none required

Example

<script>
   <module id="js_module">
      <filter content-type="application/x-javascript"
         class="com.myweb.javascriptHandler"/>
      </filter>
      ...
   </module>
   ...
</script>
