markup-extension

Purpose

Defines a single custom pipeline markup extension that can contain either a process or a rule element.

Contains

process, rule

Contained by

markup-extensions

Attributes

NameDescriptionTypeDefaultOptionsUse
local-nameThe name of the custom element without any namespace prefixxs:stringnone  required 
namespace-uriThe URI of the namespace of the custom elementxs:anyURInone  required 

Example

<markup-extensions>
   <markup-extension
      local-name="set-context"
      namespace-uri="http://mycorp.com/pipeline_ext">
      <process class-name="com.mycorp.xmlpipeline.ext.setcontext"/>
   </markup-extension>
<markup-extensions>
