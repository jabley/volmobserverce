web-driver

Purpose

Contains the proxy server and script reference definitions for the pipeline web driver.

Contains

proxy, script

Contained by

pipeline-configuration

Attribute

NameDescriptionTypeDefaultOptionsUse
connection-timeoutDefault round trip timeout for all web driver operations for an HTTP connection, in milliseconds. If not specified, MCS falls back to the value in the timeout attribute on the connection element. A value of -1 means that the connection will never timeout. xs:intnone  optional 

Example

<web-driver connection-timeout="10000">
   <proxy id="myproxy"  
          host="myhost"                  
          port="8086"/>         
</web-driver>
