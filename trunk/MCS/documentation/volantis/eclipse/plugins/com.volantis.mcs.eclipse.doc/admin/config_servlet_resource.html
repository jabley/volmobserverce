servlet-resource

Purpose

Identifies a resource that can be found within a servlet context.

Contained by

wsdl-entry

Attribute

Name Description Type Default Options Use
path The path to the servlet xs:anyURI none required

Example

<web-services-driver>
   <wsdl-catalog>
      <wsdl-entry uri="uri/for/wsdl/entry">
         <class-resource path="/class/resource/path"/>
         <servlet-resource path="/servlet/resource/path"/>
         <uri-resource uri="uri/resource"/>
      </wsdl-entry>
   </wsdl-catalog>
</web-services-driver>
