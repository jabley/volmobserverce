internal-generation

Purpose

If this element is present in releases prior to MCS 3.5, MCS will cache internal style sheets in memory using the values in the contained cache element.

Note: In MCS 3.5 and later the settings for this element no longer have any effect, though it is retained for backward compatibility.

Contains

cache

Contained by

style-sheets

Example

<style-sheets>
   <external-generation cache="enabled"
      flush-on-restart="true"
      base-directory="[path]/mcs/css"
      base-url="[context_root]/MCSCSS"/>
   <internal-generation>    
      <cache name="styleCache"      
         strategy="least-recently-used"
         max-entries="25"/>  
   </internal-generation>   
   <page-level-generation max-age="120000"/>
</style-sheets>
