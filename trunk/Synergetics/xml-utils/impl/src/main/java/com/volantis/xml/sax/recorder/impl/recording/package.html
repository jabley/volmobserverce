Provides classes that represent a recording.

Overview

A recording is created within the {@link SAXRecorderImpl} and when it has finished it creates an instance of {@link SAXRecordingImpl} that can then be played.

Efficient implementation of {@link SAXRecording}.

In this case efficiency means that it stores the events in a fixed number of arrays, rather than in lots of little objects. The reason for this is because it is much easier to GC a few large objects than lots of little objects.

See {@link SAXRecorderImpl} for details as to how the events are recorded.
