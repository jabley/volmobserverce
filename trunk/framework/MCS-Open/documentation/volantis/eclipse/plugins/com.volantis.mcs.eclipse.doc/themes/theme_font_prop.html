Font properties

Purpose

Sets a system font and defines the font family or font family set, and font style settings such as size, color, weight, stretch, and the variant.

ControlDescriptionOptions
System FontSets all font properties to the values in the selected system font on the user's machinecaption, icon, menu, message-box, small-caption, status-bar 
FamilyLists multiple font families. Browse opens Font Family Selection dialog. Names containing white space must be enclosed in single or double quotes.  
SizeAbsolute size and units using browser table, or relative size in relation to parentlarge, larger, small, smaller, x-large, xx-large, x-small, xx-small 
Size AdjustAn aspect value that preserves the x-height of the first choice font in the substitute fontleft, center, right 
StretchSelects a normal, condensed, or extended facecondensed, expanded, extra-expanded, narrower, normal, semi-condensed, semi-expanded, ultra-wider 
StyleStyle of fontitalic, normal, oblique 
VariantFont variantnormal, small-caps 
WeightWeight of font. The bolder and lighter values are relative to the inherited font weight. Other values are absolute font weights.bold, bolder, lighter
