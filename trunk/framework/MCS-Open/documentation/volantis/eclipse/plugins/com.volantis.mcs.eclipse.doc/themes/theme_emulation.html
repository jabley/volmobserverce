Theme emulation

Because HTML 3.2 and WML do not support CSS styling, MCS maps some CSS property values to attributes on individual elements from the theme policy information for HTML and WML protocol families.

The table defines the style mapping.

Property  HTML 3.2  WML 
font-size  Yes  Yes 
font-weight   bold  bold 
font-style  italic  italic 
text-decoration  line-through, underline  underline 
text-align  Yes  Yes 
vertical-align  Yes  Yes 
background-color  Yes  No 
table properties  border, cell-spacing, height, width  No
