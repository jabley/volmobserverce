mime-packaging

Purpose

Enables multipart MIME packaging.

Contained by

page-packaging

Attribute

AttributeDescriptionTypeDefaultOptionsUse
enabledIf 'true', MCS will package content in the form of multipart MIME documents using the policy values in the Protocol category in the device repository.xs:booleanfalse  optional 

Example

<page-packaging>
  <mime-packaging enabled="false"/>
</page-packaging>
