xf:label

Purpose

Provides a descriptive label for forms controls.

Contained by

xf:group, xf:input, xf:item, xf:secret, xf:select, xf:select1, xf:submit, xf:textarea

Attribute group

Core attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>xf:label</title>
    <xf:model id="login">
      <xf:instance>
        <si:instance>
          <si:item name="firstName"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <xf:input model="login" ref="firstName">
      <xf:label>Your Name:</xf:label>
    </xf:input>
  </body>
</html>
