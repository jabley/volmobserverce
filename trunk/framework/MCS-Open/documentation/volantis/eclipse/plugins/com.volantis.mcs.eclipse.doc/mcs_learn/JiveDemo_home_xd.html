Jive home page

In this module you will add to the body of the Jive Home Page. The PC page presents all of the material as a single page. The Smartphone presents it as three linked fragments.

Content of the left column of the PC page is reached on the Smartphone version with the 'Company Information' link. Material in the right is accessed from the 'News Feeds' link.

Home Page body

  1. Open the file jivehome.xdime in your XML editor
  2. Start the company announcements section of the page. This section is targeted at the second fragment in the Smartphone version. The style classes company_item and company_header control the styling. Notice that the Background Image property of the selector company_header is set to the image component /images/orange_header.mimg and the Repeat property is set to 'repeat-x'.
    <div class="company_item" id="announcements">
  <h3 class="company_header">Announcements</h3>
  <dl class="sec_menu">
    <dt>
      <a href="#" class="title">Jive New Products!!</a>
    </dt>
    <dd>Check out the latest line of Jive basketball shoes set to come out in November.</dd>
  </dl>
</div>
  3. Continue with the Human Resources and Note From the CEO sections, referring to jive.mthm for information on the style properties 
    <div class="company_item" id="human_resources">
  <h3 class="company_header">Jive Human Resources</h3>
  <dl class="sec_menu">
    <dt>
      <a href="#" class="title">Opportunities at Jive</a>
    </dt>
    <dd class="preview">Jive Sports is seeking people to fill new</dd>
  </dl>
</div>

Complete XDIME 2 code

Checklist

NamePurpose
a

Used to create an explicit link to another place in the same document or to another document. The current document is the source of the link and the value of the href attribute, defines the link target

blockquote

Defines a block of text quoted in the current document.

dd

Contains the definition of a term in a definition list.

div

A section used to add extra structure to documents. Style sheets can be used to control the presentation.

dl

A list containing individual terms and their definitions.

dt

The name of a term defined within a definition list.

h1, h2, h3, h4, h5, h6

Heading elements for sections within a document. The number associated with each element indicates its relative position in the hierarchy of headings, with 1 indicating the top level and 6 the lowest level.

Core attributes

Attributes that are common to XDIME 2 elements.

Hypertext attributes

Attributes that define link targets.
