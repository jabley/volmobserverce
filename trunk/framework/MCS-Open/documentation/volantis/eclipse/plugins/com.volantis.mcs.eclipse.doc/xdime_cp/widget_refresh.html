widget:refresh

Purpose

Allows a widget to update its contents.

Contained by

widget:carousel, widget:digital-clock, widget:progress, widget:ticker-tape

Attributes

AttributeDescriptionTypeDefaultOptionsUse
intervalThe minimum interval between content refreshes in seconds. Initially the timer starts from the moment the widget is created; subsequently it starts after the widget's content has been updated.xs:intiger  required 
srcThe URL from which the update can be retrieved, relative to the containing pagexs:anyURI  required 

Attribute group

Core attributes

Action

ActionDescription
executeExecutes AJAX request

Property

PropertyDescriptionTypeAccess
srcThe URL from which the data can be retrievedxs:anyURlread/write 

Events

EventDescription
failedThe event is sent when AJAX request fails
succeededThe event is sent when AJAX request succeeds

Example

A Ticker Tape widget that updates its contents dynamically every 15 seconds.

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget">
  <head>
    <title>widget:refresh</title>
  </head>
  <body>
    <widget:ticker-tape id="myWeatherTicker">
      <widget:refresh
        src="http://localhost:8080/mcs/projects/client-app/service/ticker?id=myWeatherTicker"
        interval="15"/>
      <span>Ticker Tape can be configured to periodically refresh its content by fetching data, such
        as weather conditions, from a web service...</span>
    </widget:ticker-tape>
  </body>
</html>
