DCI services and drivers

MCS provides two types of driver that allow you to access datasources.

  • HTTP web driver for accessing web pages
  • Web services driver for other web resources

With the HTTP web driver you specify the datasource in the request. When you use the web service driver, you define the datasource in the mcs-config.xml file. Then you reference this configuration definition from the pipeline XML.
