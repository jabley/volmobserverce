Montage and segment attributes

Purpose

Montage and segment layout attributes define values for the output from multiple canvasses on a device.

ControlDescriptionOptions
Border ColorColor of the montage grid and/or the individual segment borders  
Frame spacingSpacing between the segments  
Frame BorderWhether or not the montage grid and/or the individual segments have borders  
ScrollingWhether the segment content scrolls or not 
Margin HeightSegment margin height in pixels 
Margin WidthSegment margin width in pixels  
ResizeWhether segment can be resized
