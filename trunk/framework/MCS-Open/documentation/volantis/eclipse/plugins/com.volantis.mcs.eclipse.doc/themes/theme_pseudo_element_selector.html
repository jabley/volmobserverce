Pseudo-element selectors

In the New Style Rule wizard you can click the Browse button next to the Pseudo-element control, and choose one or more selectors. You can also order the selectors and remove them.

Pseudo-element  Description  Example 
first-line  The first line of the first paragraph that is a child of h1 element  h1 > p::first-line {font-variant: small-caps;}  
first-letter  The initial drop-cap letter of a paragraph   h1 > p::first-letter {font-weight: bold; font-size: 32pt;}  
mcs-next  An XForms group element that generates a link to the following form fragment, with the text of the group label  xf|group::after::mcs-next {content:normal}  
mcs-previous  An XForms group element that generates a link to the preceding form fragment, with the text 'Back'  xf|group::before::mcs-previous {content:"Back"}  
mcs-reset  An XForms group element that generates a reset link on a form fragment, with the text 'Clear'  xf|group::after::mcs-reset {content:"Clear"}  
mcs-shortcut  A menu element followed by the shortcut 'Alt + E'  menu::mcs-shortcut {content: 'Alt + E'}  
after  A label followed by ': '  label::after {content: ": "}  
before  The text 'Note' before the content of a paragraph  p.note::before {content: "Note: "}  
marker  Allows styling on the marker of li elements of a list, so the style of the list item content can be separated from the style of the marker  li::marker {color: green;}  
mcs-cancel  An XForms group element that generates a cancel link on a form fragment, with the text 'Cancel'  xf|group::after::mcs-cancel {content:"Cancel"}  
mcs-complete  An XForms group element that generates a 'Complete' button instead of the 'Next' button in the last step of the form wizard  xf|group::after::mcs-complete {content:"Complete"}  
mcs-label  A bold label on the active tab of a Tabs widget  widget:tab::mcs-label:active {font-weight: bold;}  
mcs-item  A separator between items in a BuzzCast feed. The content property provides the separator.  ticker:feed::mcs-item::mcs-between {content: " *** ";}  
mcs-between  Combined with mcs-item this pseudo-element inserts a separator between feed items.  ticker:feed::mcs-item::mcs-between {content: " *** ";}
