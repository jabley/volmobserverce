application-plugins

Purpose

Defines applications related to MCS.

Contains

mps

Contained by

mcs-config

Example

<application-plugins>
  <mps internal-base-url="http://localhost:8080/[context_root]"
    message-recipient-info="com.volantis.mps.recipient.DefaultRecipientResolver">
    <channels>
    ...
    </channels>
  </mps>
</application-plugins>
