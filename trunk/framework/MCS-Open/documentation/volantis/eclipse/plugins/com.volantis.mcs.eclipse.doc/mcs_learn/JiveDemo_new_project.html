Creating a project

In this module you will learn how to create a new MCS project. It will hold all the resources, policies, and XDIME 2 files required by the web application.

Creating a project 

  1. Choose the File | New | Project
  2. Pick MCS Project from the list of available wizards and then click the Next button

  3. Name your new project 'jive' and click the Next button

  4. Now you need to provide the location of the device repository. Click the Browse button and navigate to the repository file you want to use in this project. Device repository files have the extension .mdpr. When you are done, click the Finish button.

Congratulations, you've just created your first MCS project!

Adding a new folder

By default all the policies are stored in the WebContent/mcs-policies directory created by the MCS Project wizard. Our layouts and theme policies will be placed in that location. However, we will create an additional folder in the mcs-policies directory, called 'images', to store all image component policies.

  1. Choose the File | New | Folder from the Eclipse menu
  2. Select the jive/WebContent/mcs-policies as a parent folder
  3. Call the new folder 'images' and then click the Finish button
