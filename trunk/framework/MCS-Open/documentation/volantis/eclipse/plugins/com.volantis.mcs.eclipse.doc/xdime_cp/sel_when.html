sel:when

Purpose

One of a number of possible matches that could be selected. The content is selected for processing if the expr value is matched and the precept on the containing sel:select element places the sel:when element in scope.

Contains

sel:if, sel:select

Contained by

sel:select

Attribute

AttributeDescriptionTypeDefaultOptionsUse
exprDetermines whether or not the content is selected for processing. If the expression evaluates to 'true', the content is selected, otherwise it is not processed. If you omit the attribute the content is selected for processing by default.xs:string  required 

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:sel="http://www.w3.org/2004/06/diselect">
  <head>
    <title>sel:select</title>
  </head>
  <body>
    <sel:select>
      <sel:when expr="device:getPolicyValue('supports.client.framework')='true'">
        <p>This device supports Client Framework.</p>
      </sel:when>
      <sel:otherwise>
        <p>This device does not support Client Framework.</p>
      </sel:otherwise>
    </sel:select>
  </body>
</html>
