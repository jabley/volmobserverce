default

Purpose

Required container for default project information.

Contains

assets, generated-resources, jdbc-policies, xml-policies

Contained by

projects

Attribute

AttributeDescriptionTypeDefaultOptionsUse
preloadWhether or not the project information is preloaded into the caches when MCS starts.xs:booleannone  optional 

Example

<projects>
  <default preload="true">
    <jdbc-policies name="myproject"/>
    <assets base-url="/[context_root]/">
      <audio-assets prefix-url="/audio/"/>
      <dynamic-visual-assets prefix-url="/dynvis/"/>
      <image-assets prefix-url="/images/"/>
      <script-assets prefix-url="/scripts/"/>
      <text-assets prefix-url="/text/"/>
    </assets>
    <generated-resources base-dir="myproject"/>
  </default>
</projects>
