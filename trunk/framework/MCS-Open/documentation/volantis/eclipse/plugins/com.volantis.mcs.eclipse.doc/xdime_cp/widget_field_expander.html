widget:field-expander

Purpose

Defines a Field Expander widget that includes initially hidden form fields.

Contains

xf:group, xf:input, xf:secret, xf:select, xf:select1, xf:textarea

Contained by

body

Attribute group

Core attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>widget:field-expander</title>
    <xf:model>
      <xf:instance>
        <si:instance>
          <si:item name="login"/>
          <si:item name="password"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <div>
      <widget:field-expander id="myFieldExpander">
        <xf:input ref="name">
          <xf:label>Name </xf:label>
        </xf:input>
        <xf:group>
          <xf:input ref="login">
            <xf:label>Login</xf:label>
          </xf:input>
          <xf:input ref="password">
            <xf:label>Password</xf:label>
          </xf:input>
        </xf:group>
      </widget:field-expander>
    </div>
  </body>
</html>
