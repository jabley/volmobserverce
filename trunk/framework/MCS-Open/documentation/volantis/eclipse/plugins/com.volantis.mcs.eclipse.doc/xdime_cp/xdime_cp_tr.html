tr

Purpose

Denotes a row within a table.

Contains

td, th

Contained by

table

Attribute groups

Core attributes, Selection attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2">
  <head>
    <title>table</title>
  </head>
  <body>
    <table>
      <caption>Layout attributes</caption>
      <tr>
        <th>Attribute</th>
        <th>Description</th>
      </tr>
      <tr>
        <td>layoutName</td>
        <td>Name of the layout to use</td>
      </tr>
    </table>
  </body>
</html>
