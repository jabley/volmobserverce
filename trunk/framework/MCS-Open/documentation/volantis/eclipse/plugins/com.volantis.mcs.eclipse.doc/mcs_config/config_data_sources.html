data-sources

Purpose

Container for multiple named datasource definitions.

Contains

named-data-source

Contained by

mcs-config

Example

<data-sources>
  <named-data-source name="MCS"
    connect-on-start-up="false">
    <mcs-database
      host="HOST_NAME"
      port="1526"
      source="DBNAME"
      vendor="oracle"
      password="pwd"
      user="user" />
  </named-data-source>
</data-sources>
