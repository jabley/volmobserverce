jndi-configuration

Purpose

Contains the definition of a configuration of one or more JNDI datasources.

Contains

initial-context

Contained by

mcs-config

Example

<jndi-configuration>
  <initial-context name="initialContext">
    <parameter name="name1" value="value1"/>
    <parameter name="name2" value="value2"/>
  </initial-context>
</jndi-configuration>
