preload

Purpose

Specifies the policies to preload from the remote repository.

Contained by

remote-policies

Attribute

AttributeDescriptionTypeDefaultOptionsUse
urlThe URL for the set of policy definitions that will be preloaded into the remote policy cachexs:anyURInone  required 

Example

<remote-policies policy-base-url="http://myserver.com/mypolicies/">
