Layout properties

Purpose

Controls several layout properties of different objects, including the display of fragments and links.

ControlDescriptionOptions
Target ContainerThe container in the Target Layout where the content should be placed. An expression may be used to identify a specific container instance.current, [container_name], [expression] 
Target LayoutDefines the target MCS layout for an element and its contentcurrent, [layout_name] 
Fragment List OrientationOrientation of the list of links to fragments in the root fragmentcurrent 
Fragment List PositionPosition of the list of links to fragments in the root fragmentbottom, top 
Link MediaMedium the target of the link is intended fornormal, tV, web 
Link StyleHow links should be displayeddefault, button, numeric-shortcut
