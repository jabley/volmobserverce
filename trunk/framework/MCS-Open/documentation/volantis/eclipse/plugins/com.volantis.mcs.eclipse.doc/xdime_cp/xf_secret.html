xf:secret

Purpose

Used for input of confidential or sensitive information such as passwords.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>xf:secret</title>
    <xf:model id="login">
      <xf:instance>
        <si:instance>
          <si:item name="password"/>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <xf:secret model="login" ref="password">
      <xf:label>Password:</xf:label>
    </xf:secret>
  </body>
</html>
