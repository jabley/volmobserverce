widget:option

Purpose

A single item on a list.

Contained by

widget:select

Attributes

AttributeDescriptionTypeDefaultOptionsUse
selectedSpecifies a default itemxs:string selected optional 
valueAssigns value to an itemxs:string  optional 

Attribute group

Core attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget">
  <head>
    <title>widget:option</title>
  </head>
  <body>
    <div>
      <span>Select:</span>
      <widget:select>
        <widget:option value="Berlin">Berlin</widget:option>
        <widget:option value="Budapest">Budapest</widget:option>
        <widget:option value="Prague">Prague</widget:option>
        <widget:option value="Vienna">Vienna</widget:option>
      </widget:select>
    </div>
  </body>
</html>
