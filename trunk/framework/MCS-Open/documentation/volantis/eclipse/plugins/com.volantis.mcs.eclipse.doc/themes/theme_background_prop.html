Background properties

Purpose

Specifies the background of an element; colors and images. You can also position the background, repeat it, and declare whether it should be fixed or scrolled

ControlDescriptionOptions
Attachmentwhether the background image is fixed or notfixed, scroll 
ColorThe color of the backgroundcolor value, transparent 
ImageThe name of the image component or URI to be used as the separator. Browse opens the Image Component Selection dialog.none, [image_component] or URI 
RepeatWhether background is repeated or not, and if so whether it is horizontal(repeat-x), vertical (repeat-y) or tiled (repeat).repeat, repeat-x, repeat-y, no-repeat 
HorizontalInitial horizontal position for a repeatpercentage, length, left, center, right 
VerticalInitial vertical position for a repeatpercentage, length, top, middle, bottom
