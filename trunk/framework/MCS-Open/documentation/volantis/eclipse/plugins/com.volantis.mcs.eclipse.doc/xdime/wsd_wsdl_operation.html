wsd:wsdl-operation

Purpose

Defines the web service operation to be requested.

Contained by

wsd:request

Attributes

AttributeDescriptionTypeDefaultOptionsUse
operationThe name of the operationxs:NCNamenone  optional 
portTypeThe name of the port type in the WSDL descriptionxs:NCNamenone  optional 
wsdlA reference to a wsdl-entry element in the mcs-config.xml filexs:anyURInone  optional 

Examples

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:pipeline="http://www.volantis.com/xmlns/marlin-pipeline"
  xmlns:wsd="http://www.volantis.com/xmlns/marlin-web-service-driver">
  <head>
    <title>Web service driver</title>
  </head>
  <body>
    <p>Celsius2Fahrenheit</p>
    <pipeline:transform href="test.xsl">
      <wsd:request>
        <wsd:wsdl-operation wsdl="http://www.w3schools.com/webservices/tempconvert.asmx?WSDL"
          portType="TempConvertSoap" operation="CelsiusToFahrenheit"/>
        <wsd:message>
          <parameters>
            <tem:Celsius xmlns:tem="http://tempuri.org/">20</tem:Celsius>
          </parameters>
        </wsd:message>
      </wsd:request>
    </pipeline:transform>
  </body>
  </html>

The wsr:response element contains the web service response.

<wsr:response>  
  <wsr:message>
    <!-- message here-->
  </wsr:message>
</wsr:response>

The response must be transformed to XDIME presentation markup.

<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">
  <xsl:template match="/">
    <div xmlns="http://www.w3.org/2002/06/xhtml2"
      xmlns:wsr="http://www.volantis.com/xmlns/marlin-web-service-response">
      <p> 20 C = <xsl:value-of select="wsr:response/wsr:message"/> F </p>
    </div>
  </xsl:template>
</xsl:stylesheet>
