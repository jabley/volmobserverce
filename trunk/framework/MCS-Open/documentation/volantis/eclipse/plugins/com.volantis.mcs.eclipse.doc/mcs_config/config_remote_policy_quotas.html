remote-policy-quotas

Purpose

Container for the policy quota elements for one or more remote servers.

Note: In versions previous to MCS 3.5.1 several elements containing individual remote policy quota values were defined in the remote-policies section of the mcs-config.xml file. There is now a single cache quota for each server and these elements are now obsolete.

Contains

remote-policy-quota

Contained by

remote-policies

Example

<remote-policies>
  <remote-policy-quotas>
    <remote-policy-quota
      URL="http://myserver.com/remoteComponents/images/jpg
      percentage="50"/>
    <remote-policy-quota
      URL="http://anotherserver.com/newImage
      percentage="50"/>
  </remote-policy-quotas>   
</remote-policies>
