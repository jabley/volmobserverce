widget:timer

Purpose

Defines a timer.

Contains

widget:clock-content, widget:load

Attributes

AttributeDescriptionTypeDefaultOptionsUse
start-timeInitial time given in millisecondsxs:integer  optional 
stop-timeEnding time given in millisecondsxs:integer  optional 

Attribute group

Core attributes

Actions

ActionDescription
resetResets a timer
startStarts counting time
stopStops the clock

Properties

PropertyDescriptionTypeAccess
start-timeInitial time given in millisecondsxs:integerread/write 
stop-timeEnding time given in millisecondsxs:integerread/write 

Event

EventDescription
finishedThe event is sent when the time limit is reached

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget">
  <head>
    <title>widget:timer</title>
    <style type="text/css" media="handheld,all">
      #myTimer {
        mcs-datetime-format: '%d%1%H%1%i%1%s';
      }
    </style>
  </head>
  <body>
    <widget:timer id="myTimer" start-time="180000" stop-time="0"/> 
    <div>
      <widget:button action="myTimer#start">Start</widget:button>
      <widget:button action="myTimer#stop">Stop</widget:button>
      <widget:button action="myTimer#reset">Reset</widget:button>
    </div>
    <div>
      <span>Start time:</span>
      <widget:input property="myTimer#start-time"/>
      <span>Stop time:</span>
      <widget:input property="myTimer#stop-time"/>
    </div>
  </body>
</html>
