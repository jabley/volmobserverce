response:response

Purpose

Defines the outer container for a widget response.

Contains

response:body, response:head

Contained by

html

Attribute group

Core attributes

Example

Replace existing content of a Carousel widget with another set of news information. Styles for each list item are specified in the /main.mthm theme file.

<response:response xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs"
  xmlns:response="http://www.volantis.com/xmlns/2006/05/widget/response">
  <response:head>
    <response:link rel="mcs:theme" href="/main.mthm"/>
  </response:head>
  <response:body>
    <response:carousel id="myWeatherCarousel">
      <li><a href="http://www.budapest.hu">Budapest</a>: 22°C, rain and wind</li>
      <li><a href="http://www.prague-city.cz">Prague</a>: 8°C, rain</li>
    </response:carousel>
  </response:body>
</response:response>
