debug

Purpose

Enables generation of XDIME comments in rendered pages, and output of rendered pages to the log file.

Contained by

mcs-config

Attributes

AttributeDescriptionTypeDefaultOptionsUse
commentsIf set to 'true', MCS will attempt to generate comments in the output pages.xs:booleanfalse  optional 
logPageOutputIf set to 'true', MCS sends page output to the log filexs:booleanfalse  optional 

Example

<debug
  comments="false"
  logPageOutput="false"/>
