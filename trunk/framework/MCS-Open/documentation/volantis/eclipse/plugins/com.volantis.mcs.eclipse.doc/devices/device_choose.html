Choosing a device

You can associate devices with variants, layouts and themes by selecting one or more devices from the device repository, and also by specifying custom device policy values that you have applied to devices.

Different views of the available devices or policies are included on the final page in most component wizards, and in the Layout and Theme wizards.

You can access the same views from the related editors when you set Criteria to 'Targeted' and browse from the Target control.

If you have already created a layout or theme variant, and you open a device view, any device names that you have already selected are filtered from the view.

The Target Selection interface contains two or three tabs. The Device List tab initially lists all the devices in the repository in alphanumeric order, and you can enter filter values to display a subset. The Device Tree tab shows the standard device hierarchy.

The Device category policy dropdown in the Attributes section for this policy shows all the device policy sets with single-select and boolean values, including custom device policies. If you have made a selection, the Device Category tab lists the policy values. Any selected targets are displayed in a consolidated list on all three tabs.

Note: The Device category policy setting applies to the entire component, layout or theme policy, and therefore to all the variants in it. If you change this setting after targeting some devices, you should review all the targeting values; otherwise they may not work as expected.

In the wizard:

  1. Choose 'Targeted' in the Selected criteria dropdown
  2. Click Next to go to the selection page

In the editor:

  1. Select a variant to enable the attribute controls
  2. Choose 'Targeted' in the Criteria dropdown
  3. Click Browse next to the Target control to open the selection dialog

In the selection interface:

  1. Filter the Device List by name to display a standard device subset, or choose them from the Device Tree.
  2. To choose devices from a custom set, go to the Device Category tab and check the policy values to use

Tip: The filter text that you enter can contain wildcard characters. You use * for any string, for example nok* will return all patterns containing 'nokia'. You can also use ? for any single character, for example nokia?7??? will find Nokia series 7000 device names.
