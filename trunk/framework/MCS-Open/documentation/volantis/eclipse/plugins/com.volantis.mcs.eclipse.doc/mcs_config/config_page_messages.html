page-messages

Purpose

Text added as comments to pages generated by MCS, provided that the output protocol supports comments.

Contained by

mcs-config

Attribute

AttributeDescriptionTypeDefaultOptionsUse
headingThe text of the messagexs:stringnone  optional 

Example

  heading="This is a configurable comment generated
  by MCS from the page-messages element"/>
