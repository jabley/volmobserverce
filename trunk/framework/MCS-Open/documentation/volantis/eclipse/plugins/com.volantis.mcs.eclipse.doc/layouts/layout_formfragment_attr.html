Form fragment attributes

Purpose

Attributes of fragment formats.

ControlDescriptionOptions
Generate reset buttonWhether or not MCS should generate a reset button 
Next/Previous Link TextLink text to be used. A literal value or the name of a text component. If text component is selected, Browse is enabled and displays the Text Component Selection dialog. 
Next/Previous Link StyleclassClass names to be used for next and previous links. A class must be from the theme that is bound to the same canvas as the layout. 
Next/Previous Link PositionThe position of next and previous links in relation to the form field
