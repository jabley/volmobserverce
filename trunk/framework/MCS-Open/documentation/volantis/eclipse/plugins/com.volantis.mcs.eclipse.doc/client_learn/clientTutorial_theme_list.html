Additional themes

Before moving on, you should create the remaining theme policies. Please treat this section as an exercise, and refer to the MCS Help pages if you need supporting information.

Style property categories 

  1. The index.mthm theme requires styles for the following selectors
    Property  Control  Value 
    ul.index  
    Format  Color  #fff 
    Box Padding   Left  18px 
    Box Margin  All  0px 
    ul.index > li > a  
    Format  Color  #fff 
    Font  Weight  bolder 
    Text  Decoration  none 
  2. The style properties for the following selectors must be set in the progressbar.mthm theme
    Property  Control  Value 
    widget|progress  
    Format  Color  #23516a 
    Box  Height  1em 
    Box Padding  All  2px 
    Text  Align  left 
    div#progressValueDisplay  
    Format  Color  #fff 
    Font  Weight  bold 
    Box Margin  Top  4px 
    Text  Align  center 
  3. Below are the style properties for the selectors that you need to set in validator.mthm
    Property  Control  Value 
    #name  
    Effects  Validation error action  message  
    Form  Input format  Xxxx*x 
    #pin  
    Effects  Validation error action  message  
    Form  Input format  ###### 
    #formMsgSuccess  
    Font  Weight  bold 
    .error  
    Format  Color  #f00 
  4. The wizard.mthm theme has to define styles for the following selectors
    Property  Control  Value 
    xforms|*::mcs-next  
    Content  Content  Next 
    xforms|*::mcs-cancel  
    Content  Content  Cancel 
    xforms|*::mcs-previous  
    Content  Content  Back
