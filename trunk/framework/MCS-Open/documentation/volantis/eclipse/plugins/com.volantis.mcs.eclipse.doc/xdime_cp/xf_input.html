xf:input

Purpose

A form control for text input.

Contains

xf:label

Attribute groups

Core attributes, Selection attributes, Form attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:si="http://www.volantis.com/xmlns/2006/01/xdime2/si"
  xmlns:xf="http://www.w3.org/2002/xforms">
  <head>
    <title>xf:input</title>
    <xf:model id="login">
      <xf:instance>
        <si:instance>
          <si:item name="firstName">joe</si:item>
        </si:instance>
      </xf:instance>
    </xf:model>
  </head>
  <body>
    <xf:input model="login" ref="firstName">
      <xf:label>Your Name:</xf:label>
    </xf:input>
  </body>
</html>
