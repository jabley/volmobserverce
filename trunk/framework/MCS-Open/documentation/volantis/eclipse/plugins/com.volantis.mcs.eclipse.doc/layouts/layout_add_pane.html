Adding iterators

To handle the display of repeating data structures you can use spatial or temporal iterators, and their associated panes.

To create a table layout, you should use a spatial iterator, to repeat data over time you should use a temporal iterator.

To add a spatial or temporal iterator:

  1. Choose a device in the Outline view
  2. Right-click on an empty format and choose Add | Iterator | Spatial or Temporal
  3. Right-click the iterator and choose Add | Pane
  4. Set the attributes for the pane

Tip: When you create a spatial iterator, you should always check the Align content format attribute to ensure that content is aligned correctly in the resulting tables. You should also set column (or row) iterations to 'Exactly 1'
