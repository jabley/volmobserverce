Hypertext attributes

Purpose

Attributes that define link targets.

Attribute

AttributeDescriptionTypeDefaultOptionsUse
hrefDefines the link target. It can be a URL or a component reference.xs:anyURI  optional 

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2">
  <head>
    <title>anchor</title>
  </head>
  <body>
    <div>
      <a href="http://www.volantis.com" title="Volantis Systems">click me!</a>
    </div>
  </body>
</html>
