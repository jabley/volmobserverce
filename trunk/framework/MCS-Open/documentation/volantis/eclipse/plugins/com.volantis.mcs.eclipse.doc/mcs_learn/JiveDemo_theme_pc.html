Adding a PC theme

This module will guide you through the process of creating a theme for PC variant. First, we need to define the list of selectors used in our project, and then set style properties for them. In the next module we will do the same for Smartphone variant.

We can broadly divide the selectors into three groups.

  • element selectors, with familiar names from the XDIME 2 namespaces
  • Class selectors that apply to one or more element that are defined using 'dot' notation, for example .company_header
  • ID selectors, preceded by a '#' character, which in this case are used to target content at specific layout panes

Adding a selector 

The Theme editor has two pages, an Overview page listing the theme variants and their general attributes, and a Design page where you can specify your style selectors and define style properties.

  1. Make sure that jive.mthm theme policy is open
  2. Pick the PC variant and click the Design tab
  3. Click the New button in the Selectors section
  4. In the New Style Rule window choose the 'body' element from the drop down list and click the Add button. The 'body' element appears in the Selector Group window.

  5. Click Finish to add selector to the theme. The 'body' element appears in the Selectors box.

Setting style properties 

You set properties in the Style Properties section of the editor. The properties are divided into categories, some of which you can expand if required.

  1. Choose the 'body' element in the Selectors section
  2. Select the Font category and set the Family property to 'Arial,sans-serif'. If you click the Browse button, MCS displays the Font Family Selection dialog, where you can build a list to use.

  3. In the same category, set the Size property to the 'small'

Style Rules view 

The Style Rules view provides an convenient way to view all the rules for a particular theme variant. It contains a table that displays the style rules, organized by selector sequence.

  1. Choose Window | Show View | Other, and select the Style Rules view under MCS Views
  2. Pick a theme variant on the Overview page to populate the view
  3. Expand a selector and choose a property. MCS displays the property in the Design page.
