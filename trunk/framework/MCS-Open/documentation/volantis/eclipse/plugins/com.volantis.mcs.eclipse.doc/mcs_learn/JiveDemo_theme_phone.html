Smartphone theme

The theme design for the Smartphone variant is so close to the PC design that it will be faster to clone it than to add a variant and enter the property data.

Copying a theme

  1. In the Variant section, select the PC variant, right click, and choose Copy from the context menu
  2. Paste the copy in the same section
  3. Set the Selection criteria dropdown to 'Targeted'
  4. Delete 'PC' from the Targets list
  5. Browse to target the Smartphone device family
  6. Use the Style Rules view to inspect the properties and make any changes to the values using the table below
