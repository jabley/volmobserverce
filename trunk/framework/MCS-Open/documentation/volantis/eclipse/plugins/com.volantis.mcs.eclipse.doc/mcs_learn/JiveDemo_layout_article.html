Article page layout

The Jive project uses two different layouts, first for the home page, second for the article pages. Article pages require only a simple modification.

This time you will clone the home page and edit the layout design, by replacing the announcement and news section with an article page.

Creating a article page layout

  1. Open the PC layout and save it as jivearticle.mylt in the same folder
  2. Select the PC variant on the Overview page, go to either the Outline view or the Design page, and select the 'footer' pane.
  3. Right click and choose Copy
  4. Now select the outer grid in row four that contains the announcements and news, right click and choose Paste. The row is replace by a copy of the pane.
  5. Select the pane you have just pasted and change the name to 'article'.
