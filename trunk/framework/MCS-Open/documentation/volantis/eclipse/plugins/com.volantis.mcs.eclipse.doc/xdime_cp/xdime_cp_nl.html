nl

Purpose

Defines a navigation list. The label element is required.

Contains

label, li

Attribute groups

Core attributes, Selection attributes

Example

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2">
  <head>
    <title>nl</title>
  </head>
  <body>
    <nl>
      <label>Contents</label>
      <li>Concepts</li>
      <li>
        <nl>
          <label>Tasks</label>
          <li>Install</li>
          <li>Configure</li>
          <li>Test</li>
        </nl>
      </li>
      <li>Reference</li>
      <li>Help</li>
    </nl>
  </body>
</html>
