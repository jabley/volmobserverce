tr:set-url

Purpose

Changes the URL of the BuzzCast Server. Further Feed Poller requests will be executed using changed URL.

Note: If the BuzzCast Server requests a change of the URL, the Feed Poller immediately makes a request for the changed URL. In case three subsequent requests using new URL fails, the original URL will be restored.

Contained by

tr:feed-poller

Attribute group

Core attributes

Example

In this response example, following commands will be executed:

  • insert new item with specified ID, channel, title and description
  • remove item with specified ID
  • set (redirect) URL to the new BuzzCast Server
  • set new skip times to be between 11:00-12:00, and 22:00-2:30
  • set new polling interval to 10 minutes
