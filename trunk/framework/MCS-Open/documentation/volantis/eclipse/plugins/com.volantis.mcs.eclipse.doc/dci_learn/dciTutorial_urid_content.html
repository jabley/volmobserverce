Defining common content

In this module, we will create two files. The first one will contain the Jive page header, while the second one the footer. The next topic will demonstrate how to bind those files to an XDIME page.

Tip: Both files described here contain valid XDIME fragments. However, it is possible to use XSL transforms to convert structured data to XDIME at runtime. The examples in the following modules will provide details.

Common content

  1. The XML file containing an XDIME fragment needs to start with an XML declaration
    <?xml version="1.0" encoding="UTF-8"?>
  2. The XML declaration must be followed by the XDIME code with the corresponding namespace declarations. The example below shows part of the Jive page header. It specifies the Jive logo, the menus and so on. 
    <div xmlns="http://www.w3.org/2002/06/xhtml2">
  <!-- The banner -->
  <div id="logo">
    <a class="homelogo" href="#">
      <object class="homelogo" src="/images/jive_logo.mimg"/>
    </a>
  </div>
  ...
  <!-- The current stock price -->
  <div id="stockprice"> Latest from NYSE for Jive (VBE): 21.50 (+ 2.25) </div>
</div>

Header

  1. Create a jiveheader.xdinc file in the Volantis/MCS/webapps/mcs/tutorial/ directory
  2. Complete the header file to include the following code
    <?xml version="1.0" encoding="UTF-8"?>
<div xmlns="http://www.w3.org/2002/06/xhtml2">
  <!-- The banner -->
  <div id="logo">
    <a class="homelogo" href="#">
      <object class="homelogo" src="/images/jive_logo.mimg"/>
    </a>
  </div>
  <div id="photos">
    <object src="/images/photos.mimg"/>
  </div>
  <!-- The main menu -->
  <div id="menu_company_news" class="main_menu">
    <a class="pnav" href="#">
      <object src="/images/menu_company_news.mimg"/>
    </a>
  </div>
  <div id="menu_human_resources" class="main_menu">
    <a class="pnav" href="#">
      <object src="/images/menu_human_resources.mimg"/>
    </a>
  </div>
  <div id="menu_employee_store" class="main_menu">
    <a class="pnav" href="#">
      <object src="/images/menu_employee_store.mimg"/>
    </a>
  </div>
  <!-- The current stock price -->
  <div id="stockprice"> Latest from NYSE for Jive (VBE): 21.50 (+ 2.25) </div>
</div>

Footer

  1. Create a jivefooter.xdinc file in the same directory
  2. Modify this newly created file by including the following code
    <?xml version="1.0" encoding="UTF-8"?>
<div id="footer" xmlns="http://www.w3.org/2002/06/xhtml2">
  <p>Copyright <a href="#">Jive Sports</a></p>
</div>

Checklist

Here are some element references used in the includes. Follow the links to see full details of their structure.

NamePurpose
a

Used to create an explicit link to another place in the same document or to another document. The current document is the source of the link and the value of the href attribute, defines the link target

div

A section used to add extra structure to documents. Style sheets can be used to control the presentation.

object

A reference to a media resource. Object allows a variety of media resources to be placed on the rendered page. The resource may be specified as an image policy or a device depended mime type. Object also allows the control of media transcoding to ensure the resource will render on the requesting agent. See References to media for details.

p

Block element that defines a paragraph.

Core attributes

Attributes that are common to XDIME 2 elements.

Hypertext attributes

Attributes that define link targets.
