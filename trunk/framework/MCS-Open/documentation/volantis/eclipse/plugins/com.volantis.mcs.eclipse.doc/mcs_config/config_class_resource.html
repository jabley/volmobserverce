class-resource

Purpose

Identifies a resource that can be found on the class path.

Contained by

wsdl-entry

Attribute

AttributeDescriptionTypeDefaultOptionsUse
pathThe class path xs:anyURInone  required 

Example

<web-services-driver>
  <wsdl-catalog>
    <wsdl-entry uri="uri/for/wsdl/entry">
      <class-resource path="/class/resource/path"/>
      <servlet-resource path="/servlet/resource/path"/>
      <uri-resource uri="uri/resource"/>
    </wsdl-entry>
  </wsdl-catalog>
</web-services-driver>
