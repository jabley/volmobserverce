Configuring transcoder access

MCS and other applications can be configured to communicate with a variety of transcoders through a media access proxy service, which provides a unique, short, URL to a mobile client and enables that URL to access a device specific transcoded asset.

The optional media-access-proxy element in the mcs-config.xml file contains the URL prefix for this media access proxy service.

<media-access-proxy url-prefix="http://map/transcode/">
