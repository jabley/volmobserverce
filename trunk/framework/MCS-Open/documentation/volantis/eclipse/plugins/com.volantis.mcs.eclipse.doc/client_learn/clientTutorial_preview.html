Previewing the site

The Firefox web browser allows you emulate different mobile devices. This maybe helpful while developing web applications in the MCS. To preview the emulated version of your web application you need to have Firefox browser with User Agent plugin installed.

Installing Firefox and User Agent Switcher

  1. Install the Firefox web browser. Refer to the Firefox documentation for details.
  2. Start the Firefox application
  3. Choose the Tools | Extensions from the Firefox menu to open Extensions window
  4. Click the Get More Extensions link. The Firefox will open the web service that allows you to download and install extensions
  5. On the list of available extensions find the 'User Agent Switcher' plug-in
  6. To avoid issues previewing the mobile site please install also the 'XHTML Mobile Profile' and 'wmlbrowser' plug-ins
  7. Download those plug-ins and restart the Firefox browser

Emulating mobile devices

You can emulate the Smartphone devices using Firefox browser.

  1. Choose the Tools | User Agent Switcher | Options | Options... from the Firefox menu
  2. Choose the User Agents menu
  3. Click the Add button to add a new device
  4. Type the name of device in the Description text box. In our case 'SonyEricssonP910i'.
  5. Type the User Agent string for the Smartphone devices: SonyEricssonP910i/R3A SEMC-Browser/Symbian/3.0 Profile/MIDP-2.0 Configuration/CLDC-1.0
  6. Click the OK button
  7. The 'SonyEricssonP910i' should appear on the list of available devices
  8. Switch the user agent to that device by picking it from the Tools | User Agent Switcher | SonyEricssonP910i menu
  9. Open the MCS web application to preview the emulated version of the page
  10. To return to the normal preview choose the Tools | User Agent Switcher | Default from the Firefox menu
