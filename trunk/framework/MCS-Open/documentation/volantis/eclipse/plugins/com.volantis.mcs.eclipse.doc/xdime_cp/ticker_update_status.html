ticker:update-status

Purpose

Displays status of the Feed Poller widget.

Contained by

body

Attribute group

Core attributes

Examples

<?xml version="1.0" encoding="UTF-8"?>
<html xmlns="http://www.w3.org/2002/06/xhtml2"
  xmlns:widget="http://www.volantis.com/xmlns/2006/05/widget"
  xmlns:ticker="http://www.volantis.com/xmlns/2006/09/ticker">
  <head>
    <title>ticker:update-status</title>
  </head>
  <body>
    <ticker:feed-poller
      url="http://localhost:8080/buzzcast/vmds/?uid=tickeruser&amp;delivery-format=4"
      style="min-polling-interval:120; max-polling-interval:120; "/>
    <widget:ticker-tape style="height:30px; width:300px; border: solid 2px green; ">
      <ticker:feed item-display="id"/>
    </widget:ticker-tape>
    <div>Status: <ticker:update-status/></div>
  </body>
</html>

The styles may be defined in the following way.

#status-icon {position: fixed; top: 0px; left: 0px }
#status-description:mcs-busy {content: "Busy..."}
#status-description:mcs-failed {content: "The last update did not succeed: "; font: red}
#status-description:mcs-normal {content: "The last poll updated OK"}
#status-description:mcs-suspended {content: "Updates are suspended"}
