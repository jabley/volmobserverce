List of Third Party Packages and Associated Licenses for MCS

This directory contains license documents for various third-party software software components and packages that are used within this release of MCS and associated products. These packages have associated licenses that oblige anyone distributing the packages to include the licenses and, in some cases, copyright notices. The purpose of this collection of license documents is to fulfill this obligation.

This README lists these third-party packages and specifies which license documents are associated with particular packages.
