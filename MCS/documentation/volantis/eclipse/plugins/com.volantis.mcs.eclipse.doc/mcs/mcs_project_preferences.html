Project preferences

If you are using a shared Oracle policy database for collaborative projects, you need to specify the location of the Oracle database driver library.

Note: For this setting to take effect, you must restart MCS.

Regardless of the database in use, you can balance the speed of XML validation against overall performance of the policy editor interface. The Faster validation checkbox, which is checked by default, results in faster loading or refreshing of large policies in your local repository from the database. Because the faster setting can make full use of your CPU, it may at times be preferable to uncheck this control.

  1. Choose Window | Preferences and select MCS in the Preferences dialog
  2. Enter the driver name in the Oracle database driver location control or Browse to locate it on the file system
  3. Optionally change the Faster validation setting
  4. Click Apply
