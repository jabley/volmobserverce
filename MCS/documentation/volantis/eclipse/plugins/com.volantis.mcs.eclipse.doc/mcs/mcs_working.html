Working with MCS policies

In MCS you can define a series of policies to describe the relationship of all the resources on your web site.

To develop a policy in MCS you have to complete the following tasks:

  1. Create a workbench project using the MCS perspective and the New Project wizard
  2. Add components and associate them with variants
  3. Use device policies in the built-in repository or define custom device attributes
  4. Design layout variants for selected devices
  5. Add theme variants for the devices
