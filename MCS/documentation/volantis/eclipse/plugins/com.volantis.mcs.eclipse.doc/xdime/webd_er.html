Web driver elements

The XDIME web driver namespace prefix is webd: and the namespace declaration is http://www.volantis.com/xmlns/marlin-web-driver.

NamePurpose
webd:content

Used to handle different types of content resulting from a request.

webd:cookie

Contains elements that allow cookies to be created or copied. The domain, maxage, path and secure attributes are evaluated by the driver to see if the cookie should be passed on. There are also element forms for the attribute values.

webd:cookies

Container for multiple cookies.

webd:domain

Element form of the domain for which a cookie is valid. May be used instead of the domain attribute on a cookie element.

webd:get

Represents the HTTP GET method.

webd:header

Defines HTTP header elements that enable headers to be created or copied. Unlike parameters, there is no difference in how they are handled by the two different types of request. There are also element forms for the attribute values.

webd:headers

Container for multiple HTTP header elements.

webd:maxage

Element form of the maximum age of a cookie in seconds. May be used instead of the maxage attribute on a cookie element.

webd:name

Element form of the name contained in cookies, headers, and parameters. May be used instead of the name attribute on the parent element when creating it.

webd:parameter

Defines an HTTP parameter. There are also element forms for some attribute values.

webd:parameters

Container for multiple parameter elements.

webd:path

Element form of the path for which a cookie is valid. May be used instead of the path attribute on a cookie element.

webd:post

Represents the HTTP POST method.

webd:script

Specifies a script module to handle web page scripts, with a reference to a mcs-config.xml file.

webd:secure

Element form that indicates whether the cookie should only be sent over HTTPS. May be used instead of the secure attribute on a cookie element.

webd:target

Element form of the target of a parameter, which may be used instead of the target attribute on the webd:parameter element.

webd:value

Element form of the value contained in cookies, headers, and parameters. May be used instead of the value attribute on the parent element when creating it.
