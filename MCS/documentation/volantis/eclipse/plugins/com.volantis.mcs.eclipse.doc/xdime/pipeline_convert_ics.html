pipeline:convertImageURLToTranscoder

Purpose

Rewrites all image URLs in a page to be routed through the Image Conversion Service (ICS). The operation will calculate the most appropriate ICS rule to use, based on the calling device characteristics.

Attributes

AttributeDescriptionTypeDefaultOptionsUse
maxfilesizeThe maximum image sizexs:integernone  optional 
relativeWidthThe percentage width of the image on the devicexs:integernone  optional 
serverThe address of the image conversion serverxs:anyURInone  required 

Example

<pipeline:convertImageURLToTranscoder 
   server="http://ics.mysite.com:8080" 
   maxfilesize="9000"
   relativeWidth="65">
   <urid:fetch href="http://www.mysite.com/index.html" parse="xml" />
</pipeline:convertImageURLToTranscoder>
