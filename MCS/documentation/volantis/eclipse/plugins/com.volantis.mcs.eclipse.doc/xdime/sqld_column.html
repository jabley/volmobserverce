sqld:column

Purpose

Defines a column in a database table.

Contains

mdc:parseAsXML

Contained by

sqld:query

Attribute

AttributeDescriptionTypeDefaultOptionsUse
nameThe name of the column.xs:stringnone  required 

Example

<sqld:query
   datasource="PressReleases" 
   sql="select TITLE,ABSTRACT 
      from PRESS_RELEASES 
      where STORY = ?">
   <sqld:string value="%{request:getParameter('STORY')}"/>
   <sqld:column name="ABSTRACT">
      <mdc:parseAsXML
         defaultNsPrefix="pr"
         defaultNsURI="http://mycorp.com/schemas/pr"/> 
   </sqld:column>
    ...
</sqld:query>
