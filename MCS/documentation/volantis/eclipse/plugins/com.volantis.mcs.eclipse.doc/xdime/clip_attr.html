Clip attributes

Purpose

Attributes that are common to audio and video elements. The effect of the attributes and the possible attribute values may vary with the player.

Attributes

Name Description Type Default Options Use
altText Text to be used if the clip cannot be played xs:string none optional
assetURLsuffix A suffix to add parameters to the URL, for example a query string containing user information. MCS adds the specified suffix to the final asset URL without attempting to interpret it. xs:string none optional
autoStart Whether the clip plays immediately, or is started manually xs:boolean none optional
height The height of the control in pixels xs:positiveInteger none optional
loop Whether the clip should loop continuously or not xs:string none optional
name The name of the clip xs:string none optional
tabindex Defines the order of moving focus from element to element using the tab key or equivalent xs:int none optional
url The URL of the clip xs:string none optional
width The width of the control in pixels xs:positiveInteger none optional
