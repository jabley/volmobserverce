quicktime

Purpose

Used to specify a subset of QuickTime parameters in an embedded clip. Refer to the QuickTime documentation for further information on parameter options.

Contained by

pane

Attributes

Name Description Type Default Options Use
controller Whether or not the control bar is displayed xs:boolean none optional
endTime The end timecode of a streaming movie xs:string none optional
hidden Whether the clip is visible or not. Useful with audio-only movies that are designed to play in the background xs:boolean none optional
kioskMode Enables or disables saving the clip using the popup menu or drag and drop xs:boolean false optional
playEveryFrame If set to 'true', every frame is forced to play, and audio is turned off xs:boolean false optional
scale Enables or disables scaling of a compressed clip xs:string none optional
srcDontUseBrowser If true, browser caching is bypassed, and playback is passed directly to the plugin xs:boolean false optional
startTime The start timecode of a streaming movie xs:string none optional
volume Whether or not the device volume setting is overridden xs:positiveInteger optional

Attribute groups

Block attributes, Clip attributes, Event attributes, Hypertext attributes, Core attributes

Example

<quicktime name="cheetahruns"
   loop="palindrome"
   kioskMode="true"
   altText="Running Cheetah"/>
