xfboolean

Purpose

Defines a control to set binary values, for example a checkbox on a PC browser or a pair of radio buttons.

Note: The trueValue and falseValue attributes can be either a single text string or a set of comma separated values. If a single text string is used, the attribute forms the label for the control. If the content is a set of comma separated values, the first is used as a label, and all values act as a grammar for VoiceXML response.

Example
