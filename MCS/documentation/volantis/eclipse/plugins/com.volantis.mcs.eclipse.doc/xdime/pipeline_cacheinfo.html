pipeline:cacheInfo

Purpose

A wrapper for a set of cache information. It must only be present if the parent pipeline:cache element does not have a key attribute, otherwise it is an error.

Contains

pipeline:cacheControl, pipeline:cacheKey

Contained by

pipeline:cache

Example

<pipeline:cacheInfo>
   <pipeline:cacheKey value="men"/>
   <pipeline:cacheKey value="women"/>
   <pipeline:cacheControl timeToLive="600"/>
</pipeline:cacheInfo>
