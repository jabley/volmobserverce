template:value

Purpose

Identifies the position to insert parameter content in the document. This may be used at any point within the body of the template. If the parameter has not been declared, or is not of simple type then an error is raised.

Contained by

template:body

Attribute

AttributeDescriptionTypeDefaultOptionsUse
refReturns the simple value bound to the referenced parameter name. Cannot be used in an attribute expression.xs:NCNamenone  required 

Example

<template:body>
   ...
   <template:value ref="page_content"/>
   ...
<template:body>
