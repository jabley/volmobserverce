realaudio

Purpose

Used to specify a subset of Real Audio parameters in an embedded clip. Refer to the Real Player documentation for further information on parameter options.

Contained by

pane

Attributes

Name Description Type Default Options Use
controls Specifies which player controls are displayed xs:string none optional
numloop The number of times the clip should loop xs:positiveInteger none optional
shuffle Applies to multiple clips in a RAM or SMIL file. If set to 'true', clips will be played in a random order. xs:boolean none optional
startJVM If 'true', starts the JVM to run the clip. Otherwise prevents it from running if it has not already started. xs:boolean none optional

Attribute groups

Block attributes, Clip attributes, Event attributes, Hypertext attributes, Core attributes

Example

<realaudio
   name="lionroars"
   autoStart="true"
   controls="All" 
   loop="true"
   numLoop="2" 
   startJVM="false"
   altText="Roaring Lion"/>
