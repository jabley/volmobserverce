config

Purpose

Container for email configuration.

Contains

from, period, smtp, subject, to

Contained by

e-mail

Example

<devices preload="false">
   ...
   <logging>
      <log-file>device_log</log-file>
      <e-mail>
         <e-mail-sending>enable</email-sending>
         <config><smtp><host>smtp.mycorp.com</host></smtp>
            <from>
               <address>admin@mycorp.com</address>
               <name>mcsAdmin</name>
            </from>
            <to>
               <address>device.team@volantis.com</address>
               <name>Devices</name>
            </to>
         </config>
      </e-mail>
   </logging>  
</devices>
