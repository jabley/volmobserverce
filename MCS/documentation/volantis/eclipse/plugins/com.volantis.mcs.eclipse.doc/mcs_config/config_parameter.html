parameter

Purpose

Container for name-value pairs that may be required for initializing a JNDI or JDBC datasource.

Contained by

initial-context, jdbc-driver

Attributes

Name Description Type Default Options Use
name The name of the parameter. xs:string none required
value The value of the parameter. xs:string none required

Example

<jndi-configuration>
   <initial-context name="initialContext">
      <parameter name="name1" value="value1"/>
      <parameter name="name2" value="value2"/>
   </initial-context>
</jndi-configuration>
