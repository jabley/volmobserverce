page-level-generation

Purpose

Controls the cache age for in memory stylesheets.

Contained by

style-sheets

Attribute

NameDescriptionTypeDefaultOptionsUse
max-ageThe maximum period in seconds that an entry may remain in the cachexs:nonNegativeInteger or 'unlimited'unlimited  optional 

Example

<style-sheets>
   <external-generation
      base-directory="[path]/mcs/css"
      base-url="[context_root]/MCSCSS"/>
   <page-level-generation max-age="120000"/>
</style-sheets>
