http-proxy

Purpose

Optional element that defines the settings for a proxy server, and sets the system wide Java properties controlling proxies. If it is not specified then MCS will not attempt to configure the underlying system parameters.

Contained by

mcs-config

Attributes

NameDescriptionTypeDefaultOptionsUse
hostThe name of the hostxs:stringnone  optional 
portThe name of the portxs:unsignedShortnone  optional 

Example

<http-proxy
   host="proxy"
   port="8000"/>
