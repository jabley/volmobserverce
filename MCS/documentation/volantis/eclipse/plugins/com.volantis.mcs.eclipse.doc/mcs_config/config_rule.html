rule

Purpose

Defines the rule to be associated with custom pipeline markup.

Contained by

markup-extension

Attribute

NameDescriptionTypeDefaultOptionsUse
class-nameThe name of the class that implements the rule processingxs:stringnone  required 

Example

<markup-extensions>
   <markup-extension
      local-name="set-context"
      namespace-uri="http://mycorp.com/pipeline_ext">
      <rule class-name="com.mycorp.xmlpipeline.ext.setcontext"/>
   </markup-extension>
<markup-extensions>
