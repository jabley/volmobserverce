Working with components

In your project you can use component policies to associate variants with components. However, not all components can be associated with variants. For example, you cannot add variants to rollover images.

To develop a component policy in MCS you have to complete the following tasks:

  1. Add a component policy to your project using the New Component wizard
  2. Using a component editor, choose a variant or variants to associate with your component
  3. Add an asset group to locate the variants on a server or device
