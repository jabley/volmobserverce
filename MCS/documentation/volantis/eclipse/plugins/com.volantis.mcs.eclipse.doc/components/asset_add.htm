Adding an asset

You can add assets to image components in the New Component wizard, or with the Image component editor. With other component types, you can only add assets by using one of the component editors.

To open an editor:

  1. Right-click on a component in the policy tree and and choose Open with
  2. Now choose the MCS editor for the component

You can also double-click on the component in the policy tree.

The contents of the Asset Attributes section of an editor depends on the type of component you select. For a complete list of asset attribute values see Asset attributes.

To add an asset:

  1. Click Add to add a new asset. An icon is added to the list, and the edit controls are enabled.
  2. Now you can enter a value and other asset attributes. As you add information in the controls, it is appended to the text in the asset list. In some cases you can use a Browse button or a drop-down to make a selection.
  3. If there is a Value type drop-down, you can select an type from those available.

Note: When you browse to add an image asset, MCS can determine certain attributes automatically, for example the filename that is inserted in the Value attribute. However, MCS does not record the full path to the asset you have selected.

You can add only one convertible image assest to an image component. If you attempt to add more than one, you'll see an error meassage.

With all assets, you need to use asset group URLs to locate the files on a server or device, and use the Asset Group control to associate this URL with the asset.
