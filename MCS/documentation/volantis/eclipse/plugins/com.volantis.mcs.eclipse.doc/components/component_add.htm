Adding a component

In MCS, you can use the New Component wizard to create project components. The wizard allows you to define:

  • A component name and its location in your project folders
  • Attributes or references to existing project components
  1. Choose a New Component wizard
  2. Select MCS in the tree and then choose one of the component types
  3. Click Next

Naming a component

On the first page, you can locate and name the component. Each component is a file in your project. The policy tree view will be synchronized with your current project folder selection.

  1. Choose the policy source directory in the tree view
  2. Enter the component name
  3. Click Next

Defining component attributes

On the next page you can enter values for the component attributes. These are references to other existing components. The number of references will vary with the type of component you have created. For example an image component has two fallback references for image and text, while a dynamic visual component four fallbacks.

  1. Use the Browse button to display the Component Selection dialog and locate a component in your project. This view is filtered for the component type you need to select.
  2. Click Finish to close the wizard. The component icon is added to the project folder and the component editor is displayed in a new tab.

Selecting component assets

If you are adding an Image component, you can choose one or more image assets on the next page.

  1. Use the Browse button to locate the root folder for your project assets, and expand the directory tree to find an asset
  2. Choose a file type to filter for the asset type you want to select. The wizard displays a preview if it is available.
  3. Check the checkbox next to the filename to select it
  4. Click Finish to close the wizard

Tip: To select all the files in a folder and its sub folders, check the box next to the folder name or click Select all. Deselect all deselects all the files and folders.
