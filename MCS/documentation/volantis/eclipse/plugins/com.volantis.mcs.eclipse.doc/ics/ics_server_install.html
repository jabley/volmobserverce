Installing ICS to a web server

Creating a war file

  1. Change to the [path]/MCS/webapps/[install] directory
  2. Enter cd [install]/ics/webapps/ics
  3. In the directory above the [path]/MCS/webapps directory, create a .war file that contains the contents of the [path]/MCS/webapps/[context_root] directory by entering jar -cvf ../[filename].war

    Note: Ensure you put the space and the full stop after the filename at the end of the jar command to pick up the contents of the current directory.

Apache Tomcat version 4.1

To install the .war file on Tomcat.

  1. Stop Tomcat
  2. Copy the .war file $TOMCAT_HOME/webapps
  3. Start Tomcat to expand the .war file and create the ics directory, which also contains a sample image

WebSphere version 5

You can use the web-based WebSphere Administration Console to install your ICS application using the newly created .war file. For example:

http://localhost:9090/admin

  1. Log in as appropriate
  2. Expand the Applications option
  3. Choose the Install Applications option. You will now be guided through the steps to prepare and install a new application
  4. Select local path and click Browse to locate the .war file you created to hold the [path]/MCS/ics/webapps/ics directory
  5. Set the context root to ics. Click Next.
  6. Accept the default settings of 'do not override existing bindings' and default virtual host name of 'default_host'. Click Next.
  7. Accept the default settings of 'distribute applications' , your .war filename which will now have _war at the end instead of a .war extension and 'create Mbeans for resources'. Click Next.
  8. Choose the Image Conversion Service web module. Click Next. The installer displays a summary of your installation
  9. If you are satisfied with the entries scroll down and click Finish.
  10. When the installation has successfully completed you will need to save your Master configuration changes

Testing your installation

Now test that ICS has been properly installed, using the following URLs:

http://[machine-name]:[port]/ics/ics/ics/images/gp8/test.jpg

http://[machine-name]:[port]/ics/ics/ics/images/gp8/test.jpg?v.ditherMode=ordered-dither
