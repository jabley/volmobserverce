Specifying plugin class names

Plugins are application extensions that can be used for a wide variety of purposes. They are Java classes that can be provided by MCS or a user.

In the plugins section of the mcs-config.xml file, you can define the names of custom Java classes that implement the following interfaces.

  • com.volantis.mcs.integration.AssetURLRewriter
  • com.volantis.mcs.integration.PageURLRewriter
  • com.volantis.mcs.integration.PluggableAssetTranscoder
  • com.volantis.mcs.integration.URLRewriter

These contain default names and are commented out in the disctributed configuration.

<plugins asset-transcoder="ASSET_TRANSCODER_CLASS"
   asset-url-rewriter="ASSET_URL_REWRITER_CLASS"
   page-url-rewriter="PAGE_URL_REWRITER_CLASS"
   url-rewriter="URL_REWRITER_CLASS"/>
