debug-output

Purpose

Defines a directory to contain the pipeline debug output that is specified in the pipeline:serialize element.

Contained by

mcs-config

Attribute

NameDescriptionTypeDefaultOptionsUse
directoryThe directory path. If it is a relative path, it is relative to the mcs-config.xml file.xs:anyURInone   optional 

Example

<debug-output
   directory="/pipeline_debug"/>
