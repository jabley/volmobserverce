Local repository configuration

The local-repository element in the mcs-config.xml file contains the values you entered during installation. uses an XML repository.

<local-repository>
   <xml-repository/>
</local-repository>

Alternatively you can use a JDBC repository, as the next example shows.

<local-repository>
   <jdbc-repository>
      <jndi-data-source
         initial-context="myContext"
         name="datasourceName"/>
   </jdbc-repository>
<local-repository>
