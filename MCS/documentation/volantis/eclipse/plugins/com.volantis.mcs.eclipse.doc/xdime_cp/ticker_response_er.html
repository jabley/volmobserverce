BuzzCast response elements

XDIME 2 makes use of several elements from the BuzzCast and Widget vocabularies, used to provide dynamic capabilities for widgets. The namespace prefix used for the BuzzCast response elements is tr: and the namespace declaration is http://www.volantis.com/xmlns/2006/09/ticker/response.

NamePurpose
tr:add-item

Adds a new item to the Feed Poller widget.

tr:description

The mandatory child of the tr:add-item element. It contains item's description, and may include arbitrary XDIME markup.

tr:feed-poller

The main element of the Feed Poller widget response.

tr:icon

The optional child of the tr:add-item element. It contains an XDIME markup including item's icon.

tr:remove-item

Removes an item from the Feed Poller.

tr:set-polling-interval

Sets new polling interval, in seconds.

tr:set-skip-times

Changes the list of skip times, i.e. the time periods, during which Feed Poller is idle. The times are represented in minutes since midnight in client's time zone.

tr:set-url

Changes the URL of the BuzzCast Server. Further Feed Poller requests will be executed using changed URL.

tr:skip-time

The skip time, given as a number of minutes since midnight in client's time zone. The possible values are 0-1439. Up to two skip times can be defined for a single Feed Poller widget.

tr:title

The required child of tr:add-item element. It contains a title of an item, and may include arbitrary, styled inline XDIME markup.
