widget:dismiss

Purpose

Provides a method for dismissing a widget.

Multiple instances may be present within a widget:date-picker or widget:popup element; one for each dismiss action.

Note: If a widget:dismiss element is not provided then it is assumed that the widget will be hidden by some external trigger.

Contained by

widget:date-picker, widget:popup

Attribute

AttributeDescriptionTypeDefaultOptionsUse
typeDefines a label for a dismiss buttonxs:string  optional 

Attribute group

Core attributes

Example

<widget:popup>
   <div>Hello world!</div>
   <widget:dismiss>close</widget:dismiss>
</widget:popup>
