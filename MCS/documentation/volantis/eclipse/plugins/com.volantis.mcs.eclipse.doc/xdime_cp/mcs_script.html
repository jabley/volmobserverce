mcs:script

Purpose

Refers to an external script or to a script component, or contains a script.

Contained by

head

Attributes

AttributeDescriptionTypeDefaultOptionsUse
id

Unique identifier for the element

xs:ID  optional 
src

Name of the script component or the location of an external script file

xs:anyURI  optional 
srctype

Content type of an external script file

xs:string  optional 
type

Content type of the element's content

xs:string  optional 

Example

<html 
   xmlns="http://www.w3.org/2002/06/xhtml2" 
   xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs" 
   <head>
      <title>Using a script component</title>
      <mcs:script id="getResult" src="/scripts/myscript.mscr" srctype="text/javascript"/>
   </head>
   ...
</html>
