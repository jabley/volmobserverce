ticker:feed-poller

Purpose

The main element for the Feed Poller widget.

Note: There may only be one ticker:feed-poller element on the page.

Contained by

body

Attributes

AttributeDescriptionTypeDefaultOptionsUse
max-polling-intervalSpecifies maximum value of the polling interval in seconds. If not specified, it means that there is no limit.xs:intiger  optional 
min-polling-intervalSpecifies minimum value of the polling interval in seconds. If not specified, it means that there is no limit.xs:intiger  optional 
urlSpecifies the base URL of the BuzzCast Server. If the BuzzCast Server requests a change of the URL, the Feed Poller widget immediately makes a request for the changed URL. In case three subsequent requests using new URL fail, the original URL will be restored.xs:anyURl  required 

Attribute group

Core attributes

Example

<ticker:feed-poller url="http://www.myfeed.com/vmds"/>
