dt

Purpose

The name of a term defined within a definition list.

Contained by

dl

Attribute groups

Core attributes, Selection attributes

Example

<dl>
  <dt>Asset</dt>
  <dd>A file containing partial content for a web page,
     including images, audio, scripts and text.</dd>
  <dt>Asset group</dt>
  <dd>The URL of a group of assets on a server or a device.</dd>
  <dt>Audio component</dt>
  <dd>MCS component containing references to audio assets.</dd>
   ...
</dl>
