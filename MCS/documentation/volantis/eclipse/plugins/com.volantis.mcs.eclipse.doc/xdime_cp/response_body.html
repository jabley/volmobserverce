response:body

Purpose

Container for widget response elements.

Contains

response:autocomplete, response:carousel, response:clock, response:date-picker, response:deck, response:error, response:folding-item, response:progress, response:tab, response:tbody, response:ticker-tape, response:timer, response:validation, tr:feed-poller

Contained by

response:response

Attribute group

Core attributes

Example

Replaces existing content of a Carousel widget with another set of news information. Styles for the list items are specified in the main.mthm theme policy.

<response:response xmlns="http://www.w3.org/2002/06/xhtml2"
   xmlns:mcs="http://www.volantis.com/xmlns/2006/01/xdime/mcs"
   xmlns:response="http://www.volantis.com/xmlns/2006/05/widget/response">
   <response:head>
      <response:link rel="mcs:theme" href="/main.mthm"/>
   </response:head>
   <response:body>
      <response:carousel id="myWeatherCarousel">
         <li><a href="http://www.budapest.hu">Budapest</a>: 22°C, rain and wind</li>
         <li><a href="http://www.prague-city.cz">Prague</a>: 8°C, rain</li>
      </response:carousel>
   </response:body>
</response:response>
