widget:table

Purpose

Defines a Table widget.

Contains

widget:tbody, caption, tbody, tfoot, thead, tr

Attribute group

Core attributes

Example

<widget:table>
   <thead>
      <tr>
         <th>Code</th>
         <th>Country name</th>
      </tr>
   </thead>
   <widget:tbody id="tbody" cached-pages-count="10" style="mcs-table-rows-per-page:10">
      <widget:load src="service/table" when="onload"/>
      <tr>
         <td>--</td>
         <td>Loading, please wait...</td>
      </tr>
   </widget:tbody>
</widget:table>
