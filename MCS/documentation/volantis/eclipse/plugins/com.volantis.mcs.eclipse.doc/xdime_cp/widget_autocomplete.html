widget:autocomplete

Purpose

Defines a Autocompleter widget.

Contained by

xf:input

Attribute

AttributeDescriptionTypeDefaultOptionsUse
srcSpecifies the URL address of the autocompletion service responsible for returning a list of matching itemsxs:anyURI  required

Example

   <xf:input ref="f1">
      <xf:label>Country</xf:label>
      <widget:autocomplete id="myAutocompleter" src="service/autocomplete"/>
   </xf:input>
</xf:group>
