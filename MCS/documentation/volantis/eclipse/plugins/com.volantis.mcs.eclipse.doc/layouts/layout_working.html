Working with layouts

Layout policies allow you to deliver the same content to users of different devices by taking account of their characteristics, keeping content and layout completely separate.

To develop a layout in MCS you have to compete the following tasks:

  1. Add a layout policy to your project using the New Layout wizard
  2. Using the Layout editor, choose a device or devices to associate with your layout
  3. Add formats to the layout and assign attributes to them in the Format Attribute view
