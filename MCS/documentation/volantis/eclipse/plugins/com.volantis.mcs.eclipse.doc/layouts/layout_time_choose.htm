Setting iterator attributes

Spatial and temporal iterators have specific values that are not used in other format attributes.

Spatial iterators

Because spatial iterators can display data series in both rows and columns, you may need to specify the way that MCS should index the generated cells. MCS can number cells either left to right first, or top to bottom first. You may also want to set limits to the number of data sets that a user can display.

  1. In the 2D Indexing Direction control choose Across Down to number from left to right (columns first). Down Across numbers top to bottom (rows first)
  2. In Row Iterations or Column Iterations, specify either Up To or Exactly and enter a number

Temporal iterators

When you set the attributes of a temporal iterator, you can choose time values for the iterator cells.

  1. In the Time Values control click Browse to open the Time Selection dialog
  2. Set values for Hours Minutes and Seconds of a time value
  3. Click Add to add a time to the list
  4. Select a value in the list and click Up or Down to change the order to match the sequence of cells.

The values are added to the control in the sequence you chose.

To remove a value from the list, open the dialog, select a list item, and click Remove.
