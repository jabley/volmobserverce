Common format attributes

Purpose

Attributes that appear in the Format Attributes view for several formats.

ControlDescriptionOptions
Name

Format name, used to associate it with XDIME markup

  
Background Color

Color of the format background

 
Background Component

Name of background component for the format. Browse opens the Component Selection dialog, which displays existing image and dynamic visual component names.

 
Width

Format width in specified units

 
Width Units

Unit of width

Pixel, % 
Height

Format height in pixels

 
Border Width

Width of the format border in pixels

 
Cell Padding

Cell padding for format

 
Cell spacing

Cell spacing for format

 
Horizontal Alignment

Horizontal alignment of the format

Left, Center, Right 
Vertical Alignment

Vertical alignment of the format

Top, Center, Bottom 
Destination Area

Protocol-specific key value that targets a defined area. Examples include regions, the 'banner' area for WapTV, and 'Text' and 'Image' areas for MMS messages.

  
Markup Optimization

Whether any specific markup should be retained, integrated with containing markup without loss of style, or integrated regardless of loss of style

Never, Little Impact, Always 
Filter Usability Below

If the number entered here (1-100) is lower than the value in Keyboard Usability for the device, the fragment is filtered out.
