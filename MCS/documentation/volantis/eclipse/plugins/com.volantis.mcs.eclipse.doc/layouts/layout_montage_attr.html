Montage and segment attributes

Purpose

Montage and segment layout attributes define values for the output from multiple canvasses on a device.

ControlDescriptionOptions
Border Color

Color of the montage grid and/or the individual segment borders

  
Frame spacing

Spacing between the segments

  
Frame Border

Whether or not the montage grid and/or the individual segments have borders

  
Scrolling

Whether the segment content scrolls or not

 
Margin Height

Segment margin height in pixels

 
Margin Width

Segment margin width in pixels

  
Resize

Whether segment can be resized
