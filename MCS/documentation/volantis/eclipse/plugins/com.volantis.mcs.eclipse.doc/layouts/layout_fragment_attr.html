Fragment attributes

Purpose

Attributes of fragment formats.

Tip: You can override fragment link text values in XDIME markup using the layout element, and in XDIME 2 using meta element attributes. See Overriding fragment links and Fragment links in XDIME 2 for details.

ControlDescriptionOptions
Generate peer fragment links

Whether MCS should generate peer links

 
Link Order

Order of the links

Parent Peers, Peers Parent 
Link Styleclass

The name of a style class. The class must be from the theme that is bound to the same canvas as the layout.

 
Next/Previous Link Text

Link text to be used. A literal value or the name of a text component. If text component is selected, Browse is enabled and displays the Text Component Selection dialog.
