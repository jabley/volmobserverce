Update status

The Update Status widget is used to visualize the current update status of the Feed Poller widget. It provides a small (~16 x 16 pixel) status icons which change according to the connectivity status of the client. It indicates one of the following states:

  • normal - the last request has been successful
  • busy - the Feed Poller widget is performing the request
  • suspended - the requests are suspended (no items are updated)
  • failed - the last request failed

XDIME 2 elements

In the following example an Update Status widget is displayed in the top-left corner of the screen. When user clicks on the widget, the popup appears displaying detailed status information, including number of channels, and number of unread items.

<head>
   <mcs:handler id="click-handler" type="text/javascript">
      Widget.getInstance('myPopup').show();</mcs:handler>
   <event:listener observer="status-icon" handler="#click-handler" event="click"/>
</head>
<body>
   <ticker:update-status id="status-icon"/>
   <widget:popup id="myPopup">
      <ticker:update-status id="status-description"/>
         Channels with unread messages: <ticker:channels-count read="no"/>
               of <ticker:channels-count/>
         Items unread: <ticker:items-count read="no"/> of <ticker:items-count/>
      <widget:dismiss>OK</widget:dismiss>
   </widget:popup>  
</body>

where the styles are defined in the following way:

#status-icon {position: fixed; top: 0px; left: 0px}
#status-description:mcs-busy {content: "Busy..."}
#status-description:mcs-failed {content: "The last update did not succeed: "; font: red}
#status-description:mcs-normal {content: "The last poll updated OK"}
#status-description:mcs-suspended {content: "Updates are suspended"}

It is possible to use the custom icons instead of default once. To use custom icons, define styles as follows:

#status-icon:mcs-busy {content: mariner-component-url('busy.mimg')}
#status-icon:mcs-failed {content: mariner-component-url('failed.mimg')}
#status-icon:mcs-normal {content: mariner-component-url('normal.mimg')}
#status-icon:mcs-suspended {content: mariner-component-url('suspended.mimg')}

Supported style properties

Styles and content applied to the :mcs-busy, :mcs-failed, :mcs-normal, and :mcs-suspended pseudo-classes are used to denote related widget states.
