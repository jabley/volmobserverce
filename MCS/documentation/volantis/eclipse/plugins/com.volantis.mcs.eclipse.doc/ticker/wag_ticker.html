BuzzCast widget library

The BuzzCast Client, along with Volantis Client Framework, provides a library of components which can be modified in order to develop a custom BuzzCast Client user interface for each customer.

The BuzzCast Client consists of four main elements.

Feed Poller widget
A Feed Poller is a utility widget that enables the BuzzCast client to poll the server for feed updates at regular intervals. It handles communications between the BuzzCast client and the BuzzCast server, including feed content updates and usage statistics.
Update Status widget
The Update Status widget provides an indication of the client status with respect to updates.
Feed display
The Carousel and Ticker Tape widgets, known from Volantis Client Framework, are used to display item's title obtained by the Feed Poller from the BuzzCast Server. Clicking on an item in Carousel or Ticker Tape widget will show item details in the associated Item Display widget.
Item Display widget
The Item Display widget displays the content items received via the polled feed. This widget is called for a specific item which is identified by its ID, by a Carousel or Ticker Tape widget.
