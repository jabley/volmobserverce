Item display widget

The Item Display widget is used to display content of the single item. By default it is initially hidden and appears when user requests more information about a feed item from Carousel or Ticker Tape widgets.

XDIME 2 elements

<ticker:item-display>
   <strong>Title:</strong> <ticker:item-title/>
   <em>Details:</em>
   <p><ticker:item-description/></p>
      Used channel: <ticker:item-channel/>
   <widget:dismiss>Close</widget:dismiss>
</ticker:item-display>

Supported style properties

The Item Display widget can operate in two modes, 'popup' and 'in-place', configured by the Position property.

Popup mode, in which the widget appears in front of the currently displayed content is achieved by setting the Position property to 'relative' or 'absolute'.

In-place mode, in which the widget appears within the existing page, such that the surrounding content is moved out of the way when the widget is visible, is achieved by setting the Position property to 'static'.

Effects

This widget may use all the effects defined for the Client Framework. Refer to Client Framework documentation for details.

Java Script API

The hide() method hides the widget. It has no effect if the widget is already hidden.
