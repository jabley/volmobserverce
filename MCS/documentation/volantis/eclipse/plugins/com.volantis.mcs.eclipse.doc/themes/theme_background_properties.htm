Background properties

Property Description
Attachment Whether background is fixed or scrolling
Color Choice of 16 system colors, transparent, and values set on the user's machine. Button on left of control opens color picker.
Dynamic visual Drop-down options of inherit | none. Browse opens Dynamic Visual Component Selection dialog.
Image Drop-down options of inherit | none. Browse opens Image Component Selection dialog.
Repeat Whether background is repeated, and if so whether repeated or tiled
Position: Horizontal Number and units or percentage, or relative positions for a repeat
Position: Vertical
