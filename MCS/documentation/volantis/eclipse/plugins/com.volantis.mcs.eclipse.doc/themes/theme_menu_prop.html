Menu properties

Purpose

Specifies the style, order, orientation, and use of text an images of menus and menuitems.

Note: The Menu Item Iterator Allocation property is not supported for the deprecated row and column iterator formats.

ControlDescriptionOptions
DTMF Allocation

How MCS assigns DTMF access keys to menuitems. Automatic overrides any specified key; manual allows its use.

automatic, manual 
Scope

Whether menu content applies for the duration of this menu or during processing of the whole page. If this is the root page the menu applies the whole session.

dialog, document 
Style

The style of the menu

static, dynamic 
Link Style

Specifies the style of menu to use. If the value is set to numeric-shortcut, MCS renders the menu with numeric identifiers for each menuitem. If the device does not support this feature. Otherwise the default style for the device applies.

default, numeric-shortcut 
Image Style

How images are used in menuitems

none, plain, rollover 
Text Style

How text is used in menuitems

none, plain 
Orientation

Orientation of menuitems and shard links. Horizontal allows optional separators between items.

vertical, horizontal 
Menu Item Iterator Allocation

How the menuitems will be bound to instances of a pane in a spatial iterator. Automatic writes a menuitem to each instance of pane; none writes only to the first instance of the pane.

automatic, none 
Menu Item Order

Order in which image and text menuitems are used when both are present

image-first, text-first 
Menu Item Orientation

Orientation of the contents of a menuitem when more than one item is present. Some combinations of menu orientation and this property may not be supported on every protocol.

horizontal, vertical 
Menu Item Active Area

Whether image, text or both are active when both are present. Only one image style and one text style can be shown in a single menuitem.

text-only, image-only, both 
Menu Item Shortcut Active

Whether or not shortcuts are active for the menuitems in a menu

active, inactive
