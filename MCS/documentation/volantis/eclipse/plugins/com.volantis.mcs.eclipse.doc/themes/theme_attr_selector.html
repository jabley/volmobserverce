Attribute selectors

In the New Style Rule wizard you can click the Browse button next to the Attribute control, and enter an attribute name and choose an Action. You can also order the selectors and remove them. The Action options have the following meanings for attribute matching.

Note: Strings, as distinct from identifiers (elements, id and class attributes) must be quoted in attribute selectors.

Action Description Example
Set Any list item with the title attribute set  li[title] {font-weight: bold;}  
Equal A paragraph with the class attribute 'relinfo'  p[class=relinfo]{color: blue;}  
Contains word Menu labels with a class that is either 'sport', 'music' or 'games'. The list of words must be a space separated.  label[class~=sport music games]{background-color=silver;}  
Matches language A paragraph with a lang attribute of 'fr' or beginning with 'fr' and immediately followed by '-'.  p[lang|="fr"] {border: thin;}  
Starts with Menuitem links to a 'mycorp' URI  menuitem[href^="http://mycorp.com"]{color: red;}  
Ends with Image with a src attribute ending in '.png'  img[src$=".png"] {clear: both;}  
Contains The title attribute contains 'copy', which will select 'copyright' and 'photocopy'   p[title="copy"] {font-size: 9pt;}
