Adding a selector

On the Theme editor Design page you can add or edit selectors that are associated with element names, class attributes or id attributes in the XML data the device will display. For example, you'll want to create selectors for headings, paragraphs, links and so on.

The Selector list has two columns. The Selector column contains the names of one or more selectors. The Order column indicates the order the style rules.

You can use the Style Rule wizard to create a sequence of style selectors for the selector column or simply edit the text. In the wizard you can set values for the following selectors:

  • Element
  • Psuedo-class
  • Pseudo-element
  • Class
  • Id

To add a selector:

  1. Click Browse to start the Style Rule wizard.
  2. Set values for an Element, and its Class or Id attributes, or for a Pseudo Class or Pseudo Element
  3. Click Add to add the selection to the Selector Group
  4. Click Up or Down to change the order of the selectors in the rule
  5. Click Finish to close the wizard

You can also click Add to add a blank item to the list, and enter the selector text. To remove a selector select an item in the list and click Remove.

Editing selectors

To modify style selectors, you have to edit them in place, using commas to separated the selectors. You can also cut, copy and paste text, to change the order of selectors, or add them to another theme.
