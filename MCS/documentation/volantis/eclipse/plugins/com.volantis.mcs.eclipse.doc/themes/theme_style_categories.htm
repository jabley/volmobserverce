Style property categories

You can select each of the categories of style properties in the properties tree. The structure of the categories is defined in the table.

All categories contain the value 'inherit' as an option, and can be set as !Important.

See the Cascading Style Sheet, level 2 CSS2 Specification for further details of CSS style properties.

Category Description
Background Specifies the background of an element; colors, images, and dynamic visuals. You can also position the background, repeat it, and declare whether it should be fixed or scrolled
Border Specifies the width, color, and style of the border area of a box and the spacing between adjacent cell borders
Box Defines properties of the box, including the box type, inline box height, box content height and width
Chart Sets thematic information for chart properties including chart adornment and labels, colors, percentage size, and axis angles
Dynamic Visual Sets attributes for Dynamic Visual content, including positioning, level of quality, scaleability, and Windows mode
Font Sets a system font and defines the font family or font family set and font style settings such as size, color, weight, stretch, and the variant
Form Controls the appearance and positioning of end-user input objects or objects that might be used in forms
Format Defines general visual format rules and thematic information on the user interface elements
Frame Sets properties of multiple image frames (videos) such as the display rate, an index, and the looping behavior
Layout Controls several layout attributes of different objects, including the display of fragments and links
List Defines basic visual formatting of lists and the list item markers, including the positioning, the appearance, and any image file for the list item marker
Menus Specifies the order, orientation, text and separator properties
Position Defines a layout positioning scheme of generated boxes. These effects the positioning properties of any floating element.
Text Sets the visual presentation of characters, spaces, words, and paragraphs
