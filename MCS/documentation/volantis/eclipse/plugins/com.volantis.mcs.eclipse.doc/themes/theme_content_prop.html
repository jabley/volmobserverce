Content properties

Purpose

Defines generated content values.

ControlDescriptionOptions
Content

The text string to be generated as content for an element qualified with a pseudo-element such as :before or :element.

  
Increment Counter

The name of the element used to increment the counter. An optional integer defines the amount to increment for every occurrence of the element (default 1). Zero and negative integers are allowed.

  
Reset Counter

One or more names of counters, each one optionally followed by an integer, which gives the value that the counter is set to (default 0)

  

Example

The example adds the text 'Chapter' to each h1 and increments the count from 1, sets the section counter to 0 at each h1 and increments section numbers at each h2.

h1:before {
   content: "Chapter " counter(chapter) ". ";
   counter-increment: chapter;
}

h1 {
   counter-reset: section;
}

h2:before {
   content: counter(chapter) "." counter(section) " ";
   counter-increment: section;
}
