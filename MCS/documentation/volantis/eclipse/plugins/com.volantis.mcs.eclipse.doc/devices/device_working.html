Working with devices

MCS uses the attributes of devices in the repository to match the layout and theme policies that you have developed for them, and to select component variants that best match the device capabilities.

You can work with devices on the following tasks:

  1. Choose or change a device repository for your project
  2. Associate devices with layouts and themes by choosing a device
  3. View a device repository attributes
  4. Define identification patterns
  5. Add Type Allocation Codes
  6. Edit policy values
  7. Search a device repository
  8. Add custom policies
  9. Add custom devices
  10. Update the repository
