Adding a device

You can add new devices to the device repository using the Device Repository editor. For each device you add you should include at least one primary identification pattern.

  1. In the editor select a fallback device in the tree
  2. Click the New button under the device list to open the New Device wizard
  3. Enter a Name for the device
  4. Click Finish to close the wizard
  5. Define an identification pattern

To remove a device from the repository, select it in the tree and click Delete.
