Viewing a device repository

You can view the attributes of devices in the repository associated with a project.

  1. Choose Window | Show View | MCS Views | Device Repository to open a workbench view on the repository
  2. Expand the device tree and select a device. The Identifiers and Policies sections in the view show the identification patterns and attributes of the device.
  3. Click a header in the Policies section to sort them alphabetically using the values in the column
