policy-values

Purpose

Lists the specified policy values for a device.

If the body of the request is empty, a document with an empty policy-values element is returned. If the device name is missing or invalid, '404 Not Found' is returned.

Method

GET

Signature

/device/policy/values/[device-name]/

Parameter

NameDescriptionType
device_name

The device name

xs:string

Examples

Input with the policies 'aggregation' and 'UAProf.Push-MsgSize' in the body

/device/policy/values/Nokia-9110i

Returns

<policy-values
   xmlns="http://www.volantis.com/xmlns/2006/01/device-ws"
   device-name="Nokia-9110i">  
   <policy policy-name="aggregation">
      <boolean>false</boolean>
   </policy>   
   <policy policy-name="UAProf.Push-MsgSize">
      <number>
         <int>-1</int>
      </number>
   </policy>
</policy-values>
