Inline plugin elements

The XDIME Inline Plugin Module namespace prefix is ipim: and the namespace declaration is "http://www.volantis.com/xmlns/marlin-im".

Note: This approach to generating markup is deprecated in favor of Dynamic Content Integration methods.
NamePurpose
ipim:argument

An argument specified as name-value pairs. Both attributes are treated as Java Strings.

ipim:arguments

Contains one argument element for each argument to the method being invoked.

ipim:invoke

Used to call methods on user-written classes from XDIME markup.
