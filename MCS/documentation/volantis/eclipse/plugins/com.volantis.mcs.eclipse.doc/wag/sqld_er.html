SQL driver elements

The XDIME SQL driver namespace prefix is sqld: and the namespace declaration is http://www.volantis.com/xmlns/marlin-sql-driver.

NamePurpose
sqld:column

Defines a column in a database table.

sqld:modify

Container for SQL modify statements that can insert, update, or delete entries in database tables.

sqld:query

Container for a SQL query. The query can be a normal query, or be configured to take parameters. If a parameter is used, the query element must contain a nested string parameter.

sqld:transaction

Container for multiple SQL modifications (and possibly queries).

SQL driver datatype elements

There are several elements for datatypes defined in the sqld: namespace. For convenience they are grouped here rather than under individual element names.
