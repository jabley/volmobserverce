layout:getPaneInstance

Purpose

Returns a reference to one instance of a pane inside an iterating format. The reference can be used wherever a pane name can be used.

Note: The special type layout:paneReference may be used only with functions that return it for use in attributes that reference panes

Signature

layout:getPaneInstance($paneStem as xs:string, $index as xs:integer, ...) as layout:paneReference

Parameters

NameDescriptionType
index

The index of the instance of the pane required. There must be at least one of these values. There may be more if the reference requires it.

xs:integer
paneStem

The name of the pane as it appears in the layout without any index dependent information

xs:string

Example

<pane name="{layout:getPaneInstance('title',2)}">                    
    <p>Practical Digital Image Processing</p>
</pane>
