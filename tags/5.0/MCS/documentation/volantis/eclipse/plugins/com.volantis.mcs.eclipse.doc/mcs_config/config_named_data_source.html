named-data-source

Purpose

Defines a named datasource.

Contains

anonymous-data-source, connection-pool, data-source, jdbc-driver, jndi-data-source, mcs-database

Contained by

data-sources

Attributes

Name Description Type Default Options Use
connect-on-start-up Whether or not MCS should connect to the datasource on startup xs:boolean false optional
name A unique datasource name xs:string none required

Example

<data-sources>
   <named-data-source name="MCS"
      connect-on-start-up="false">
      <mcs-database
         host="HOST_NAME"
         port="1526"
         source="DBNAME"
         vendor="oracle"
         password="pwd"
         user="user" />
   </named-data-source>
</data-sources>
