policy-names

Purpose

Lists the names of all the device policies.

If the repository contains no policies, a document with an empty policy-names element is returned.

Method

GET

Signature

/policy/names

Examples

Input

/policy/names

Returns

<policy-names xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">
   <policy-name>pjpeginpage</policy-name>
   <policy-name>aggregation</policy-name>
   <policy-name>rmfinpage</policy-name>
   <policy-name>ggif2rule</policy-name>
</policy-names>
