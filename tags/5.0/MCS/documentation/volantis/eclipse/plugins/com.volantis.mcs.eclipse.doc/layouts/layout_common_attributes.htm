Common layout attributes

The layout attributes listed here are those on the Layout editor Overview page, and the format attributes that appear in the Format Attributes view for several formats. For other layout attributes see Related information

Overview page

Attribute Description
Default Fragment Identifies the default fragment (canvas)
Default Segment Identifies the default segment (montage)
Destination Layout Name of the target area in the Layout Group Name modeset
Device Name Name chosen from the device tree
Layout Group Name Name of a modeset in WapTV

Common format attributes

Attribute Description
Name Format name, used to associate it with XDIME markup
Background Color Color of the format background
Background Component Name of background component for the format
Width Format width in specified units
Width Units Drop-down with options for % or pixels
Height Format height
Border Width Width of format border
Cell Padding Cell padding for format
Cell Spacing Cell spacing for format
Horizontal Alignment Left, center or right
Vertical Alignment Top, center or bottom
Destination Area WML devices such as WapTV have predefined areas which can be targeted with this attribute
Markup Optimization Whether any specific markup should be retained, integrated with containing markup without loss of style, or integrated regardless of loss of style
Filter Usability Below If the number entered here (1-100) is lower than the value in Keyboard Usability for the device, the fragment is filtered out.
