xml-policies

Purpose

Indicates that the project's policies are stored as XML.

Contained by

default, project

Attribute

Name Description Type Default Options Use
directory The root directory containing the policies. xs:string none required

Example

<projects>
   <default>
      <xml-policies directory="[home]/runtimes/repository/xml-repository" />
      <assets base-url="/wps/">
         <audio-assets prefix-url="/audio/"/>
         <dynamic-visual-assets prefix-url="/dynvis/"/>
         <image-assets prefix-url="/images/"/>
         <script-assets prefix-url="/scripts/"/>
         <text-assets prefix-url="/text/"/>
      </assets>
   </default>
   ...
</projects>
