ticker:update-status

Purpose

Displays status of the Feed Poller widget.

Contained by

body

Attribute group

Core attributes

Examples

Create an Update Status widget displayed in the top-left corner of the screen. When user clicks on the widget, the popup appears displaying textual status description, together with advanced status information including number of channels, number of unread items.

<head>
   <mcs:handler id="click-handler" type="text/javascript">
      Widget.getInstance('myPopup').show();</mcs:handler>
   <event:listener observer="status-icon" handler="#click-handler" event="click"/>
</head>
<body>
   <ticker:update-status id="status-icon"/>
   <widget:popup id="myPopup">
      <ticker:update-status id="status-description"/>
         Channels with unread messages: <ticker:channels-count read="no"/> of <ticker:channels-count/>
         Items unread: <ticker:items-count read="no"/> of <ticker:items-count/>
      <widget:dismiss>OK</widget:dismiss>
   </widget:popup>  
</body>

where the styles are defined in the following way:

#status-icon {position: fixed; top: 0px; left: 0px }
#status-description:mcs-busy {content: "Busy..."}
#status-description:mcs-failed {content: "The last update did not succeed: "; font: red}
#status-description:mcs-normal {content: "The last poll updated OK"}
#status-description:mcs-suspended {content: "Updates are suspended"}
