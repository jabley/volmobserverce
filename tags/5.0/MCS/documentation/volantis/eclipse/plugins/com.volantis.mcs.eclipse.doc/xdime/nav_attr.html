Navigation attributes

Purpose

Attributes that define navigation features. The shortcut attribute can contain literal text, or an MCS expression or a JSP expression that evaluates to a string or a text component name.

Attributes

AttributeDescriptionTypeDefaultOptionsUse
shortcutThe shortcut key to set focus on the elementxs:stringnone  optional 
tabindexDefines the order of moving focus from element to element using the TAB key or equivalent.xs:intnone  optional
