widget:deck

Purpose

Defines a Deck widget.

Contains

widget:deck-page, widget:load

Attribute group

Core attributes

Actions

ActionDescription
first-pageDisplays the first page of a deck, unless deck is displaying the first page
last-pageDisplays the last page of a deck, unless deck is displaying the last page
next-pageDisplays the next page of a deck, unless deck is displaying the last page
previous-pageDisplays the previous page of a deck, unless deck is displaying the first page

Properties

PropertyDescriptionTypeAccess
current-page-numberThe number of the page being displayedxs:integerread 
pages-countThe total number of available pagesxs:integerread 

Example

<widget:deck id="deck" style="mcs-deck-mode:switch">
   <widget:deck-page>Page 1</widget:deck-page>
   <widget:deck-page>Page 2</widget:deck-page>
   <widget:deck-page>Page 3</widget:deck-page>
</widget:deck>

<widget:display property="deck#displayed-page-number"/>
            of <widget:display property="deck#pages-count"/>

<widget:button action="deck#first-page">&lt;&lt;</widget:button>
<widget:button action="deck#previous-page">&lt;</widget:button>
<widget:button action="deck#next-page">&gt;</widget:button>
<widget:button action="deck#last-page">&gt;&gt;</widget:button>
