widget:date-picker

Purpose

Defines a Date Picker widget that is bound to a form field.

Contains

widget:calendar-display, widget:dismiss, widget:load, widget:month-display, widget:next-month, widget:next-year, widget:previous-month, widget:previous-year, widget:set-today, widget:year-display

Contained by

body

Attributes

AttributeDescriptionTypeDefaultOptionsUse
dayInputFieldReferences a day form fieldxs:string  optional 
inputField References a date input form fieldxs:string  optional 
monthInputFieldReferences a month form fieldxs:string  optional 
yearInputFieldReferences a year form fieldxs:string  optional 

Attribute group

Core attributes

Actions

ActionDescription
dismissHides a Date Picker widget
next-monthSwitches a calendar display to the next month
next-yearSwitches a calendar display to the next month
previous-monthSwitches a calendar display to the previous month
previous-yearSwitches a calendar display to the previous year
set-todaySets a current date in the field associated with a Date Picker widget

Properties

PropertyDescriptionTypeAccess
monthReturns a month displayed by the Date Picker widgetxs:stringread 
yearReturns a year displayed by the Date Picker widgetxs:stringread 

Example

<xf:input id="date-input">
   <xf:label>Enter your birth date </xf:label>
</xf:input>

<widget:date-picker inputField="date-input">
   <widget:previous-year> Previous Year </widget:previous-year>  
   <widget:previous-month> Previous Month </widget:previous-month>
   <widget:month-display/>
   <widget:year-display/>
   <widget:next-month> Next Month </widget:next-month>
   <widget:next-year> Next Year </widget:next-year>
   <widget:set-today> Set today </widget:set-today>
   <widget:calendar-display/>  
   <widget:dismiss> [close] </widget:dismiss>
</widget:date-picker>
