device:isInstanceOf

Purpose

Checks whether the current device is an instance of the named device.

Signature

device:isInstanceOf($device as xs:string) as xs:boolean?

Parameters

NameDescriptionType
device

The name of the device to compare against

xs:string
false

The device is not an instance of the named device

xs:boolean
true

The device is an instance of the named device

xs:string

Example

<p expr="device:isInstanceOf('PC')">
  This message is only for those PC users out there.
</p>
