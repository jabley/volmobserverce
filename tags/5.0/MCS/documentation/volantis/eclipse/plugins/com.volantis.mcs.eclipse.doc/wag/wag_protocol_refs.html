Protocol support

MCS supports the following Internet protocols.

  • AvantGo
  • HDML 3.0
  • HTML, Versions 3.2 and 4.0
  • i-mode HTML (cHTML)
  • MML (J-Phone Mobile Markup Language)
  • Palm WCA (Web Clipping Applications)
  • Sky WTVML
  • WML Versions 1.1, 1.2 and 1.3
  • XHTML 1.0 Basic, Transitional and Full
  • VoiceXML 1.0

See the protocol references below for further information.
