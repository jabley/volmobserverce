Container reference functions

XDIME-CP provides function expressions to that you can use to apply styles to content in iterator layouts. These functions are not defined to have a namespace. See XDIME-CP in layouts for examples using these functions.

Name Description
counter Defines content instances in an iterator layout defined by the mcs-container-instance function  
mcs-container-instance Selects a specific instance of a layout container
