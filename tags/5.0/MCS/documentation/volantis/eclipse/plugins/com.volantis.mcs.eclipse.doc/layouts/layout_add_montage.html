Adding montage formats

In a montage you construct a grid structure and add segments to the grid to contain several canvasses.

To add montage formats:

  1. Choose a montage device in the Outline view
  2. Add a grid with a cell for each canvas you want to associate with the montage
  3. Right-click on each grid cell and choose Add | Segment
  4. Set attributes for each format
