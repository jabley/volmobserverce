xftextinput

Purpose

Defines a text input control that can represent a single line, multi-line or voice input text entry.

Contained by

xfform

Attributes

AttributeDescriptionTypeDefaultOptionsUse
errmsgThe error message to be issued if the control fails validationxs:stringnone  optional 
initialThe initial value of the controlxs:stringnone  optional 
inputModeSets the default character input mode for iMode and MML devicesxs:stringnone katakana, hiragana, latin, latin digits optional 
maxLengthThe maximum number of characters in the contentxs:stringnone  optional 
nameThe name of the control and the form variable return to the serverxs:stringnone  required 
typeThe type of text input controlxs:stringtext text, password optional 
validateRules for validating the contentxs:stringtext  optional 

Attribute groups

Event attributes, Form attributes, Focus attributes, Navigation attributes, Core attributes

Example

<xfform name="address"
   method="post" action="address.xml">  
      <xftextinput name="name" caption="Name"/>
      <xftextinput name="street" caption="Street address"/>
      <xftextinput name="city" caption="City"/>
      <xftextinput name="postcode" caption="Postcode"/>
      <xfsiselect name="country" caption="Country">
         <xfoption caption="Austria" value="at"/>
         <xfoption caption="Belgium" value="be"/>
         <xfoption caption="Denmark" value="da"/>
         <xfoption caption="Finland" value="fi"/>
         <xfoption caption="France" value="fr"/>
         ...
      </xfsiselect>
      <xftextinput name="email" caption="Email"/>     
      <xftextinput name="phone" caption="Telephone"/>
    <xfaction type="submit" caption="Submit"/>
    <xfaction type="reset" caption="Reset"/>
</xfform>
