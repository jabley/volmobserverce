pipeline:alternative

Purpose

Contains alternative content inside a try element.

Contained by

pipeline:try

Example

<pipeline:try> 
   <pipeline:preferred> 
      <pipeline:transform href="title.xsl" compilable="true">
         <urid:fetch href="title.xml" />
      </pipeline:transform>  
  </pipeline:preferred> 
  <pipeline:alternative> 
      <pipeline:content>
         <p>There was a problem retrieving the information requested.</p>
      </pipeline:content> 
  </pipeline:alternative> 
</pipeline:try>
