device-names

Purpose

Lists all the devices in the repository.

If the repository contains no devices, a document with an empty device-names element is returned.

Method

GET

Signature

/device/names

Examples

Input

/device/names

Returns

<device-names xmlns="http://www.volantis.com/xmlns/2006/01/device-ws">
   <device-name>Master</device-name>
   <device-name>Voice</device-name>
   <device-name>BeVocal</device-name>
   <device-name>Internet-Appliance</device-name>
   <device-name>Clamshell</device-name>
   <device-name>Nokia-9110i</device-name>
</device-names>
