webd:script

Purpose

Specifies a script module to handle web page scripts, with a reference to a mcs-config.xml file.

Contained by

webd:get, webd:post

Attribute

AttributeDescriptionTypeDefaultOptionsUse
refA reference to a module element in the mcs-config.xml file.xs:stringnone  required 

Example

<webd:get
   url="http://newsline.net/2004-12-18/index.xml">
   ...
   <webd:script ref="js_module"/>
   ...
</webd:get>
