widget:popup

Purpose

The main element for a Popup widget.

Contains

gallery:slideshow, widget:dismiss

Contained by

body

Attribute group

Core attributes

Actions

ActionDescription
showCauses the popup to be displayed. If the popup is already displayed it has no effect.
hideCauses the popup to be hidden. If the popup is already hidden it has no effect.

Example

<widget:popup>
   <div>Hello world!</div>
   <widget:dismiss>close</widget:dismiss>
</widget:popup>
