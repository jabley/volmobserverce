select

Purpose

Container for conditional markup elements. The content consists of one or more when elements and an optional otherwise element.

Contains

otherwise, when

Contained by

canvas

Attributes

AttributeDescriptionTypeDefaultOptionsUse
exprAn MCS expressionxs:stringnone  optional 
preceptDetermines whether only the first, or every matching when element is selected for processingxs:stringmatchfirst matchfirst, matchevery optional 

Example

<select precept="matchevery">
   <when expr="device:getPolicyValue('preferredimagetype')='GIF'">
      <img src="/logogif.mimg"/>
   </when>
   <when expr="device:getPolicyValue('preferredimagetype')='PNG'">
      <img src="/logopng.mimg"/>
   </when>
   <otherwise>
      <p>MyCorp logo</p>
   </otherwise>
</select>
