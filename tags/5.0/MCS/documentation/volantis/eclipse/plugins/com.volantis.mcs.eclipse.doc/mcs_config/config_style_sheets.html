style-sheets

Purpose

Controls the caching of generated stylesheets.

Note: From MCS version 3.5, the max-age attribute on the style-sheets element has no effect and is deprecated. You should use the max-age attribute on the contained page-level-generation element to set the cache age.

Contains

external-generation, internal-generation, page-level-generation

Contained by

mcs-config

Attribute

NameDescriptionTypeDefaultOptionsUse
max-ageThe maximum period in seconds that an entry may remain in the cachexs:nonNegativeInteger or 'unlimited'unlimited  deprecated 

Example

<style-sheets>
   <external-generation
      base-directory="[path]/mcs/css"
      base-url="[context_root]/MCSCSS"/>
   <page-level-generation max-age="120000"/>
</style-sheets>
