template:documentation

Purpose

Container for user documentation. A vocabulary such as XHTML is recommended. A custom XSL stylesheet may be used to extract the content for a user guide.

Contained by

template:definition, template:parameter

Example

<template:documentation>
   <div>
      <h2>wml2xdime</h2>
         <p>This template generates XDIME XML from WML content.</p>
   </div>
</template:documentation>
