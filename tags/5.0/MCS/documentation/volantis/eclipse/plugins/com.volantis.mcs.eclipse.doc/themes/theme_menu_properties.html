Menu properties

Property Description
DTMF Allocation Whether DTMK access keys are assigned automatically or manually to child menuitem elements
Scope
 Whether the menu applies to the whole document or just to the current dialog
Style Specifies whether the menu is to be static or dynamic
Link Style Whether shard links are default or numeric-shortcut
Image Style Whether menuitem images are plain or rollover
Text Style How text is used in creating the menuitems
Orientation Orientation of menu items and shard links
Menu Item Iterator Allocation Whether menuitems inside a spatial iterator will be allocated automatically or not
Menu Item Order Order in which image and text menuitems are used when both are present
Menu Item Orientation Orientation of the contents of a menuitem when more than one item is present
Menu Item Active Area Whether image, text or both are active when both are present

Separators properties

If you specify that menus are oriented horizontally, MCS applies the separator properties between groups of menu items.

Property Description
Characters Characters to be used
Position
 Whether the separator is located before or after groups or both
Repeat Number of times to repeat the separator
Image Image component to be used as the separator
Horizontal Character Spacing between groups
Type Whether to use the characters or image component between groups
