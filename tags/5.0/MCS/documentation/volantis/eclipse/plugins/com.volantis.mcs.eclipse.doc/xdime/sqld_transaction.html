sqld:transaction

Purpose

Container for multiple SQL modifications (and possibly queries).

Contains

sqld:modify, sqld:query

Attributes

AttributeDescriptionTypeDefaultOptionsUse
datasourceName of a valid datasource as specified in the mcs-config.xml file.xs:stringnone  required 
idIdentifies the transaction set.xs:stringnone  required 

Example

<sqld:transaction
   dataSource="address"
   id="update-address">
   <sqld:modify
      transaction="update-address"
      sql="insert into ADDRESS (FORENAME, SURNAME, ADDRESS) values (?, ?, ?)">
     ...
    </sqld:modify>
    <sqld:modify
       transaction="update-address"
       sql="insert into PREFERENCES (FORENAME, SURNAME, PREFERENCES) values (?, ?, ?)">
       ...
    </sqld:modify>
</sqld:transaction>
