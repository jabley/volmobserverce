Using the MCS perspective

When you open the MCS perspective, MCS initially displays the set of features you are most likely to need for policy development. You can modify the layout of a perspective at any time, and Eclipse will save the current set up.

You select the MCS perspective in different ways, depending on how you have customized the Eclipse environment. For example, you may have customized an XML editor perspective to include MCS on a menu.

To open the MCS perspective when no menus are customized:

Choose Window | Open Perspective | Other.

To customize another perspective menu for MCS:

  1. Choose Window | Customize Perspective
  2. On the Shortcuts tab select 'Open Perspective' in the Submenu dropdown list
  3. Check the MCS checkbox in the Shortcut Categories list and click OK

The next time you choose Open Perspective from the one you have customized, MCS appears on the menu.

To customize the MCS perspective:

  1. Choose Window | Customize Perspective
  2. On the Shortcuts tab select 'New' in the Submenu dropdown list
  3. Check the MCS checkbox in the Shortcut Categories list and select the item
  4. Check the shortcuts you would like in the New menu and click OK

The next time you choose New in the MCS perspective the shortcuts will appear.

Tip: If you have more than one perspective open you can switch perspective by clicking on the tab.
