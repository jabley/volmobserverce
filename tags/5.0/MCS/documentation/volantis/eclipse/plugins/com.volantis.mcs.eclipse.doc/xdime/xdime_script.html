script

Purpose

Used to contain or refer to a script.

Note: Contained scripts are deprecated, but are supported for backward compatibility. You should use the src attribute to refer to a script component or file.

Attributes

AttributeDescriptionTypeDefaultOptionsUse
   
charSetCharacter encoding of the script if a script filexs:stringnone  optional 
deferA hint that the script will not generate output. If set to 'true', execution can be deferred.xs:booleannone  optional

srcName of script component or location of an external script filexs:stringnone  optional 
typeThe content type of the script if a script filexs:stringnone  optional 

Attribute group

Core attributes

Examples

The first example refers to a script file and uses the type attribute. The second example refers to a script component.

<script src="../scripts/release.js" type="text/javascript"/> 
<script
   id="ro1"
   src="{/ReleaseOpts.mscr}"
   defer="true"/>
