Dynamic visual properties

Purpose

Sets attributes for dynamic visual content, including positioning, level of quality, scaleability, and Windows mode.

ControlDescriptionOptions
Flash Align

How the movie window is positioned in the browser window

bottom, center, left, right, top 
Flash Quality

Level of anti-aliasing to be used during playback

auto-high, auto-low, best, high, low, medium 
Flash Scale

How the movie is placed in the browser window when width and height values are percentages

exact-fit, no-border, show-all 
Flash Window Mode

Windows ActiveX display mode. Requires Internet Explorer 4.0 or later. There is no adverse effect when viewing the page on another system.

opague, transparent, window 
Horizontal

Position within the area defined by the width settings

left, center, right 
Vertical

Position within the area defined by the height settings

top, center, bottom
