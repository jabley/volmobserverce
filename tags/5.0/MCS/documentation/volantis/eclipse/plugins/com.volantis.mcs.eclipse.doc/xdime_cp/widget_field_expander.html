widget:field-expander

Purpose

Defines a Field Expander widget that includes initially hidden form fields.

Contains

xf:group, xf:input, xf:secret, xf:select, xf:select1, xf:textarea

Contained by

body

Attribute group

Core attributes

Example

<widget:field-expander>
    <xf:input ref="name" >
      <xf:label>Name </xf:label>
   </xf:input>
    <xf:group>
      <table>
         <tr>
            <td>Address</td>
            <td>
               <xf:input ref="address" />
            </td>
         </tr>
         <tr>
            <td>E-mail</td>
            <td>
               <xf:input ref="email"/>
            </td>
         </tr>
      </table>
   </xf:group>
</widget:field-expander>
