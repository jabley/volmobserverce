template:simpleValue

Purpose

Defines a simple parameter value containing character data. See Using template parameters for more information on the possible values for the evaluationMode attribute.

Contained by

template:binding, template:parameter

Attribute

AttributeDescriptionTypeDefaultOptionsUse
evaluationModeThe method of evaluating the value.xs:NCNameimmediate immediate, deferred, repeated optional 

Example

<template:simpleValue>
   <a href="%{$varName}">Link text here</a>	
</template:simpleValue>
