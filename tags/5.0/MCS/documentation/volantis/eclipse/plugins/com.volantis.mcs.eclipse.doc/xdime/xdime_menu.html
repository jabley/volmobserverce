menu

Purpose

Used to create a menu incorporating image and text effects. Menus may be nested. You use a theme to set the orientation of the contained menuitem elements.

Note: The menu element should appear at top-level, inside the canvas element. If it is used inside a pane element, the output order of the menu and other elements in the pane may not be correct.

Contains

label, menu, menuitem, menuitemgroup

Contained by

canvas

Example

<menu pane="p1" class="newmenu">
   <label>Main Menu</label>
   <menuitem href="home.xml" text="Home"/>
   <menuitem href="specials.xml" text="Special Offers"/>
</menu>
<menu>
